## PROBLEM SETTING
The problem is coherent diffractive imaging (CDI) phase retrieval for metrology of copper pad topography in hybrid bonding wafer inspection. The input is a 2D far-field diffraction intensity pattern I(k), which is real-valued and non-negative, captured on a 2048×2048 pixel EUV CCD detector at 13.5 nm wavelength. The output is a complex-valued 2D object reflection function O(r) defined over a grid representing a 15×15 array of copper pads (each pad occupying 22 pixels, pitch ~800 nm, CD ~600 nm, pixel size 27 nm). The object magnitude encodes material reflectivity (determined by n and k optical constants of Cu, SiCN, SiO2, Si at EUV wavelengths), and the object phase encodes pad surface topography via the relation ΔΦ = 4πh/λ, where pad recession h ranges from 1 to 3 nm, yielding phase modulations up to ~2.8 radians (no wrapping). The problem operates across both domains: constraints are enforced in real space (support, reflectivity prior) and Fourier/frequency space (measured intensity magnitude replacement). Gaussian noise is added to diffraction patterns. The data are complex-valued intermediate quantities with real-valued intensity measurements as input.

## FORWARD MODEL & DEGRADATION (CRITICAL)
## Forward Model

This paper simulates coherent diffractive imaging (CDI) in the Fraunhofer (far-field) regime. The forward model generates the measured diffraction intensity as:

$$I(\mathbf{k}) = |\mathcal{F}\{P(\mathbf{r} - \mathbf{R}) \cdot O(\mathbf{r})\}|^2 + n$$

where $O(\mathbf{r})$ is the complex-valued reflective object function, $P(\mathbf{r})$ is the illumination probe (either an Airy spot or a structured speckle field), $\mathcal{F}$ denotes the Fourier transform (valid in the Fraunhofer far-field regime per Eq. 3), and $n$ is additive Gaussian noise. The object's reflection function is computed via the transmission matrix method using tabulated optical constants (n, k) for SiCN, SiO₂, Si, and Cu at λ = 13.5 nm (EUV), with a phase term $\Delta\Phi = 4\pi h(\xi,\eta)/\lambda$ encoding the copper pad recession depth $h \in [1, 3]$ nm (yielding up to 2.8 rad phase modulation). The object is a 15×15 periodic array of pads with ~600 nm CD, ~800 nm pitch, duty cycle 0.6, simulated on a grid with 27 nm pixel size. For ptychography, the probe is scanned with ~75% overlap (step size of 40 pixels). **The paper states only that "Gaussian noise is added to the diffraction patterns" but does not specify the exact noise standard deviation, SNR level, or any quantitative noise parameter.** This is a critical omission — no numerical value for the noise variance or signal-to-noise ratio is provided anywhere in the paper.

## ALGORITHMIC BLUEPRINT
## Algorithm Details

The paper presents two iterative phase retrieval algorithms (rpIO and rpPIE) for coherent diffractive imaging of periodic copper pad arrays. **rpIO (Reflective-Prior Input–Output):** At each iteration $n$, the object is updated as: for $\mathbf{r} \in S$ (support/pad region), $O_{n+1}(\mathbf{r}) = \mathcal{H}^{-1}\!\left[\sqrt{I(\mathbf{k})}\frac{\mathcal{H}(O_n(\mathbf{r}))}{|\mathcal{H}(O_n(\mathbf{r}))|}\right]$, enforcing the measured Fourier magnitude constraint via modulus replacement; for $\mathbf{r} \notin S$ (surrounding planar layer), $O_{n+1}(\mathbf{r}) = \beta\, O_n(\mathbf{r}) + (1-\beta)\,R_p$, where $R_p$ is the known reflectivity of the substrate material and $\beta$ is a feedback constant. The propagator $\mathcal{H}$ is the Fourier transform (Fraunhofer/far-field regime, valid when $z \gg 2d^2/\lambda$). **rpPIE (Reflective-Prior Ptychographic Iterative Engine):** The exit wave is $\Psi(\mathbf{r}, \mathbf{R}) = P(\mathbf{r}-\mathbf{R}_j)\,O(\mathbf{r})$ where $P$ is the probe function. The object update consists of two steps: (1) $O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \frac{|P(\mathbf{r}-\mathbf{R})|}{|P_{\max}(\mathbf{r}-\mathbf{R})|}\frac{P^*(\mathbf{r}-\mathbf{R})}{(|P(\mathbf{r}-\mathbf{R})|^2+\alpha)}\times\beta\,(\Psi_{c,n}(\mathbf{r},\mathbf{R})-\Psi_n(\mathbf{r},\mathbf{R}))$, which is the standard PIE update with regularization parameter $\alpha$; (2) for $\mathbf{r} \notin S$, the reflectivity prior is enforced: $O_{n+1}(\mathbf{r}) = \beta\, O_{n+1}(\mathbf{r}) + (1-\beta)\,R_p$. The corrected exit wave $\Psi_{c,n}$ is obtained by replacing the Fourier modulus with $\sqrt{I(\mathbf{k})}$ while preserving the phase. The phase difference introduced by Cu pad topography (recession depth $h$) is modeled as $\Delta\Phi = 4\pi h(\xi,\eta)/\lambda$. **Loss/Error Function:** The error metric used to evaluate reconstruction quality is the discrepancy between the reconstructed object and the ground truth (shown decreasing up to 2 orders of magnitude for rpPIE vs. standard PIE); the Fourier-domain constraint enforces consistency with measured intensity $I(\mathbf{k})$. **Hyperparameters:** Wavelength $\lambda = 13.5$ nm; pixel size = 27 nm; pad CD ≈ 600 nm; pitch ≈ 800 nm; duty cycle = 0.6; pad array size = $15\times15$; each pad = 22 pixels; Cu pad recession depth $h \in [1, 3]$ nm (maximum phase modulation = 2.8 rad); ptychographic scan step = 40 pixels with ~75% overlap; Gaussian noise added to diffraction patterns. There is no learning rate, optimizer, or scheduler as these are not gradient-based/neural-network methods but iterative projection algorithms with the feedback parameter $\beta$ controlling convergence speed and the regularization constant $\alpha$ preventing division by zero in the PIE denominator.

## EXPERIMENTAL PROTOCOL
This paper is a **simulation-only study** with no experimental datasets, training/testing splits, or standard image-quality metrics such as PSNR or SSIM. The "dataset" consists of synthetically generated coherent diffraction patterns from a simulated 15×15 array of copper pads (each pad occupying 22 pixels, duty cycle 0.6, pitch ~800 nm, CD ~600 nm, pixel size 27 nm, wavelength 13.5 nm). The sample's reflection function is computed via the transmission matrix method, with pad recession depths randomly drawn from the interval [1, 3] nm, introducing a phase modulation up to ~2.8 rad via ΔΦ = 4πh/λ. Material optical constants (n, k) for SiCN, SiO₂, Si, and Cu are taken from the CXRO database (Henke et al.). Gaussian noise is added to the simulated far-field diffraction patterns. Two illumination types are tested: an Airy spot and a structured speckle field, with ptychographic scans using 40-pixel steps ensuring ~75% overlap. No normalization to [0,1] or [-1,1] is described. The primary evaluation metric is a **reconstruction error (loss function)** measuring the discrepancy between the reconstructed complex object and the ground truth (reported on a log scale, with improvements of up to 2 orders of magnitude for the proposed rpIO/rpPIE over standard HIO/PIE), along with cross-sectional phase comparisons and average phase error over a region of interest (e.g., ~2.6×10⁻² rad for Airy-spot rpPIE, ~4.1×10⁻⁴ rad for speckle-field rpPIE). Baselines are the standard HIO algorithm (which did not converge) and the standard PIE algorithm.
