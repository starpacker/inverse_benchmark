

# Problem Formulation, Proposed Strategy, and Step-by-Step Plan

---

## 1. [Problem Formulation]

### Forward Model

The imaging system is a coherent diffractive imaging (CDI) / ptychographic setup operating in reflection mode at EUV wavelength λ = 13.5 nm.

**Object Model:**
The complex-valued reflection function of the sample is:

$$O(\mathbf{r}) = R(\mathbf{r}) \cdot \exp\left(i \frac{4\pi h(\xi, \eta)}{\lambda}\right)$$

where:
- $\mathbf{r} = [\xi, \eta]$ is the 2D spatial coordinate
- $R(\mathbf{r})$ is the amplitude reflectivity computed from optical constants $(n, k)$ via the transmission matrix method (Fresnel coefficients for the multilayer SiCN/SiO₂/Si/Cu stack)
- $h(\xi, \eta) \in [1, 3]$ nm is the pad recession depth (randomly assigned per pad)
- Maximum phase modulation: $\Delta\Phi_{\max} = 4\pi \cdot 3 / 13.5 \approx 2.8$ rad (no phase wrapping)

**Exit Wave (per scan position $\mathbf{R}_j$):**

$$\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})$$

where $P(\mathbf{r})$ is the known probe function (Airy spot or speckle field).

**Fraunhofer Propagation (Eq. 3):**

$$\hat{\Psi}_j(\mathbf{k}) = \mathcal{F}\{\Psi_j(\mathbf{r})\}$$

valid when $z \gg 2d^2/\lambda$ (Eq. 2), with detector coordinates mapping to spatial frequencies $f_x = x/(\lambda z)$, $f_y = y/(\lambda z)$ (Eq. 4).

**Measurement Model:**

$$I_j(\mathbf{k}) = \left|\hat{\Psi}_j(\mathbf{k})\right|^2 + n_j(\mathbf{k})$$

where $n_j(\mathbf{k}) \sim \mathcal{N}(0, \sigma^2)$ is additive Gaussian noise.

**Inverse Problem:**
Given $\{I_j(\mathbf{k})\}_{j=1}^{J}$, the known probe $P(\mathbf{r})$, support mask $S(\mathbf{r})$, and substrate reflectivity prior $R_p$, recover $O(\mathbf{r})$ — specifically the phase map encoding $h(\xi, \eta)$.

## 2. [Proposed Strategy]

**Algorithm Suite:** Implement and compare four algorithms:

1. **Standard HIO** (Hybrid Input-Output) — baseline, expected non-convergence for periodic structures
2. **rpIO** (Reflectivity-Prior Input-Output, Eq. 5) — HIO with substrate reflectivity enforcement outside support
3. **Standard PIE** (Ptychographic Iterative Engine) — ePIE-style object update only
4. **rpPIE** (Reflectivity-Prior PIE, Eq. 6) — ePIE with two-step nested update: object update + reflectivity prior enforcement

**Rationale:** The periodic structure creates spatially separated diffraction orders at the detector, causing standard HIO/PIE to stagnate. The reflectivity prior $R_p$ acts as a physics-informed constraint that regularizes the solution outside the pad support, breaking the ambiguity. rpPIE with ptychographic diversity (overlapping scans) provides the most robust reconstruction, achieving ~2 orders of magnitude error reduction over standard PIE.

## 3. [Step-by-Step Plan]

### Step 1: Data Preprocessing & Ground Truth Generation

**1.1 Compute Reflectivities via Transmission Matrix Method**

- Build the multilayer stack model: vacuum / SiCN / SiO₂ / Si substrate (for regions between pads) and vacuum / Cu / SiCN / SiO₂ / Si (for pad regions).
- For each layer $l$ with complex refractive index $\tilde{n}_l = n_l - i k_l$, compute the Fresnel reflection/transmission coefficients at each interface at normal incidence:

$$r_{l,l+1} = \frac{\tilde{n}_l - \tilde{n}_{l+1}}{\tilde{n}_l + \tilde{n}_{l+1}}, \quad t_{l,l+1} = \frac{2\tilde{n}_l}{\tilde{n}_l + \tilde{n}_{l+1}}$$

- Propagate through each layer using transfer matrices:

$$M_l = \begin{pmatrix} \cos(\tilde{n}_l k_0 d_l) & -\frac{i}{\tilde{n}_l}\sin(\tilde{n}_l k_0 d_l) \\ -i\tilde{n}_l \sin(\tilde{n}_l k_0 d_l) & \cos(\tilde{n}_l k_0 d_l) \end{pmatrix}$$

where $k_0 = 2\pi/\lambda$ and $d_l$ is layer thickness.

- Extract overall complex reflectivity $R_{\text{pad}}$ (on pads) and $R_{\text{sub}} = R_p$ (substrate, between pads).
- **Note:** If exact layer thicknesses are not specified, use the optical constants to compute surface Fresnel reflectivities as a first-order approximation, then modulate phase by $\Delta\Phi = 4\pi h/\lambda$.

**1.2 Construct the Object $O(\mathbf{r})$**

- Define a grid of size $N \times N$ where $N$ accommodates the 15×15 pad array plus border. Pad pitch ≈ 30 pixels, so the array spans $15 \times 30 = 450$ pixels. Add margins for probe extent → total grid ~512×512 or 600×600.
- Create binary support mask $S(\mathbf{r})$: value 1 on pad locations (22×22 pixel squares at 30-pixel pitch), value 0 elsewhere.
- Assign each of the 225 pads a random recession depth $h_m \sim \mathcal{U}[1, 3]$ nm.
- Compute:

$$O(\mathbf{r}) = \begin{cases} R_{\text{pad}} \cdot \exp\left(i \frac{4\pi h_m}{\lambda}\right) & \text{if } \mathbf{r} \in \text{pad } m \\ R_p & \text{if } \mathbf{r} \notin S \end{cases}$$

**1.3 Construct the Probe $P(\mathbf{r})$**

- **Airy spot:** $P(\mathbf{r}) = \left[\frac{2 J_1(\pi |\mathbf{r}| / r_0)}{\pi |\mathbf{r}| / r_0}\right]$ where $r_0$ is chosen so the probe covers multiple pads (ensuring ~75% overlap at 40-pixel step). Normalize to unit max amplitude.
- **Speckle field:** Generate a random complex field, low-pass filter to appropriate bandwidth, normalize. Alternatively, propagate a random phase screen through a lens model.

**1.4 Generate Scan Positions**

- Raster scan with step size $\Delta s = 40$ pixels in both $\xi$ and $\eta$.
- Number of positions: ensure full coverage of the 15×15 array. With ~450-pixel span and 40-pixel steps: approximately $J \approx 12 \times 12 = 144$ positions (adjust based on probe diameter to ensure edge coverage).
- Store positions $\{\mathbf{R}_j\}_{j=1}^{J}$.

**1.5 Simulate Diffraction Data**

For each scan position $j$:
1. Extract the windowed exit wave: $\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})$
2. Compute far-field: $\hat{\Psi}_j(\mathbf{k}) = \mathcal{F}\{\Psi_j(\mathbf{r})\}$
3. Compute noise-free intensity: $I_j^{(0)}(\mathbf{k}) = |\hat{\Psi}_j(\mathbf{k})|^2$
4. Add Gaussian noise: $I_j(\mathbf{k}) = I_j^{(0)}(\mathbf{k}) + \sigma \cdot \epsilon_j(\mathbf{k})$, where $\epsilon_j \sim \mathcal{N}(0, 1)$
5. Clip negative values: $I_j(\mathbf{k}) = \max(I_j(\mathbf{k}), 0)$

Choose $\sigma$ to match a realistic SNR (e.g., 30–40 dB, or as specified).

### Step 2: Forward Operator Implementation

**2.1 Fraunhofer Propagation**

$$\mathcal{A}_j: O(\mathbf{r}) \mapsto \mathcal{F}\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\}$$

Implemented as 2D FFT of the exit wave patch. The patch size equals the probe support (e.g., $M \times M$ pixels where $M$ is chosen to contain the probe).

**2.2 Fourier Magnitude Constraint (Modulus Replacement)**

Given current estimate $\hat{\Psi}_j(\mathbf{k})$ and measured intensity $I_j(\mathbf{k})$:

$$\hat{\Psi}_{c,j}(\mathbf{k}) = \sqrt{I_j(\mathbf{k})} \cdot \frac{\hat{\Psi}_j(\mathbf{k})}{|\hat{\Psi}_j(\mathbf{k})| + \epsilon}$$

where $\epsilon$ is a small constant to avoid division by zero.

**2.3 Inverse Propagation**

$$\Psi_{c,j}(\mathbf{r}) = \mathcal{F}^{-1}\{\hat{\Psi}_{c,j}(\mathbf{k})\}$$

### Step 3: Solver / Algorithm Architecture Details

#### Algorithm 1: Standard HIO

Initialize $O_0(\mathbf{r})$ (e.g., random phase with known amplitude, or uniform $R_p$).

At iteration $n$, for a single (non-ptychographic) measurement:
1. $\hat{\Psi}_n = \mathcal{F}\{O_n\}$
2. $\hat{\Psi}_{c,n} = \sqrt{I} \cdot \hat{\Psi}_n / |\hat{\Psi}_n|$ (modulus replacement)
3. $O'_n = \mathcal{F}^{-1}\{\hat{\Psi}_{c,n}\}$
4. Update:

$$O_{n+1}(\mathbf{r}) = \begin{cases} O'_n(\mathbf{r}) & \text{if } \mathbf{r} \in S \\ O_n(\mathbf{r}) - \beta \cdot O'_n(\mathbf{r}) & \text{if } \mathbf{r} \notin S \end{cases}$$

**Expected:** Does not converge for periodic structures (diffraction orders are delta-like, leaving large gaps in Fourier space).

#### Algorithm 2: rpIO (Eq. 5)

Same as HIO but with reflectivity prior outside support:

$$O_{n+1}(\mathbf{r}) = \begin{cases} O'_n(\mathbf{r}) & \text{if } \mathbf{r} \in S \\ \beta \cdot O_n(\mathbf{r}) + (1 - \beta) \cdot R_p & \text{if } \mathbf{r} \notin S \end{cases}$$

This replaces the HIO negativity/zero constraint with a convex combination pushing toward the known substrate reflectivity $R_p$.

#### Algorithm 3: Standard PIE (ePIE object update only)

Initialize $O_0(\mathbf{r}) = R_p$ everywhere (or random).

For each iteration, loop over scan positions $j = 1, \ldots, J$ (optionally randomized order):

1. Form exit wave: $\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O_n(\mathbf{r})$
2. Propagate: $\hat{\Psi}_j = \mathcal{F}\{\Psi_j\}$
3. Modulus replacement: $\hat{\Psi}_{c,j} = \sqrt{I_j} \cdot \hat{\Psi}_j / |\hat{\Psi}_j|$
4. Back-propagate: $\Psi_{c,j} = \mathcal{F}^{-1}\{\hat{\Psi}_{c,j}\}$
5. Object update (ePIE rule):

$$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \frac{|P(\mathbf{r} - \mathbf{R}_j)|}{|P_{\max}(\mathbf{r} - \mathbf{R}_j)|} \cdot \frac{P^*(\mathbf{r} - \mathbf{R}_j)}{|P(\mathbf{r} - \mathbf{R}_j)|^2 + \alpha} \cdot \beta \left(\Psi_{c,j}(\mathbf{r}) - \Psi_j(\mathbf{r})\right)$$

where:
- $|P_{\max}|$ = maximum absolute value of the probe
- $\alpha$ = Tikhonov regularization (e.g., $10^{-3} \cdot |P_{\max}|^2$)
- $\beta$ = step size / feedback parameter

#### Algorithm 4: rpPIE (Eq. 6) — Two-Step Nested Update

Same as Standard PIE, but after each object update (Step 1), apply the reflectivity prior (Step 2):

**Step 1 (Object Update):** Same as PIE Eq. above → yields intermediate $O_{n+1}^{(\text{int})}(\mathbf{r})$

**Step 2 (Reflectivity Prior Enforcement):**

$$O_{n+1}(\mathbf{r}) = \begin{cases} O_{n+1}^{(\text{int})}(\mathbf{r}) & \text{if } \mathbf{r} \in S \\ \beta \cdot O_{n+1}^{(\text{int})}(\mathbf{r}) + (1 - \beta) \cdot R_p & \text{if } \mathbf{r} \notin S \end{cases}$$

This enforces the known substrate reflectivity in the inter-pad regions after every scan-position update, providing a strong physics-informed regularization.

**Implementation Notes for rpPIE:**
- The prior enforcement happens **per scan position** (inside the inner loop), not just per epoch.
- The support mask $S$ is applied globally — only the region illuminated by the current probe position is effectively updated in Step 1, but Step 2 can be applied to the full object or just the illuminated patch (the paper's Eq. 6 suggests it is applied to the full update region).
- Use the **same** $\beta$ for both the ePIE update weight and the convex combination (or tune separately if the paper specifies distinct parameters).

### Step 4: Loss Function, Evaluation Metrics & Convergence

**4.1 Error Metric (per iteration)**

Fourier-space error (normalized):

$$E_n = \frac{\sum_j \sum_{\mathbf{k}} \left(\sqrt{I_j(\mathbf{k})} - |\hat{\Psi}_{j,n}(\mathbf{k})|\right)^2}{\sum_j \sum_{\mathbf{k}} I_j(\mathbf{k})}$$

This is the standard R-factor / Fourier error used in CDI.

**4.2 Real-Space Evaluation**

- Extract phase map: $\phi(\mathbf{r}) = \arg[O_{\text{recon}}(\mathbf{r})]$
- Convert to recession depth: $h_{\text{recon}}(\xi, \eta) = \phi(\mathbf{r}) \cdot \lambda / (4\pi)$
- Compare against ground truth via:
  - **PSNR** of the phase map (target: match within ±0.3 dB)
  - **RMSE** of recession depth (nm)
  - **Cross-sectional profiles** through pad rows/columns
  - **Per-pad recession depth error** (mean and max)

**4.3 Convergence Criteria**

- Run for a fixed number of iterations (e.g., 200–500 for HIO/rpIO, 50–200 epochs for PIE/rpPIE)
- Monitor $E_n$; stop when $|E_{n} - E_{n-1}|/E_n < 10^{-6}$ or iteration limit reached
- Record $E_n$ vs. iteration for all four algorithms to reproduce the ~2 orders of magnitude improvement plot

**4.4 Hyperparameter Settings**

| Parameter | Suggested Range | Tuning Strategy |
|---|---|---|
| $\beta$ (feedback) | 0.5 – 0.9 | Grid search; 0.7 typical |
| $\alpha$ (regularization) | $10^{-3}|P_{\max}|^2$ | Fixed |
| Noise $\sigma$ | Set for target SNR | Match paper's noise level |
| Iterations (HIO/rpIO) | 500 | Fixed |
| Epochs (PIE/rpPIE) | 100–200 | Until convergence |
| Scan randomization | Per-epoch shuffle | Standard practice |

**5.2 For each experiment, record:**
1. Convergence curve: $E_n$ vs. iteration number (log scale)
2. Final reconstructed amplitude $|O(\mathbf{r})|$ and phase $\arg[O(\mathbf{r})]$ images
3. Cross-sectional phase profiles along selected rows
4. Per-pad recession depth estimates vs. ground truth
5. PSNR of phase reconstruction
6. Final Fourier error $E_{\text{final}}$

**5.3 Reproduction Validation:**
- rpPIE should achieve ~2 orders of magnitude lower Fourier error than standard PIE
- HIO should fail to converge (error plateaus at high value)
- rpIO should converge but slower/less accurately than rpPIE
- Phase profiles should show clear step-like transitions at pad boundaries with correct recession depths
- PSNR match within ±0.3 dB of reported values
