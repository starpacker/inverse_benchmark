## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function encoding amplitude and phase (topography)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe function (Airy or speckle)
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Phase-Topography Relation
$$\Delta\Phi(\xi,\eta) = \frac{4\pi h(\xi,\eta)}{\lambda}, \quad h \in [1,3]\text{ nm}, \quad \lambda = 13.5\text{ nm}$$

### Inverse Problem
$$\hat{O} = \arg\min_O \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|^2 + \lambda_{prior} \cdot \mathcal{R}(O)$$

where $\mathcal{R}(O)$ is the reflectivity prior enforcing known substrate reflectivity outside pad support.

## 2. [Proposed Strategy]

**Algorithm**: **Standard rpPIE (Eq. 6)** with simple magnitude projection, no momentum, no RAAR — returning to the proven stable formulation with careful hyperparameter tuning.

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 7 | 6.59 | Batched updates destroyed sequential ePIE convergence |
| 8 | 9.16 | Too slow convergence, weak prior, β=1.0 oscillation |
| 9 | 8.35 | **RAAR + momentum + aggressive prior = instability** |

**Key insight**: Every "clever" addition (RAAR, momentum, batching) has made things WORSE. The baseline at 10.97 PSNR likely uses a clean, standard ePIE/rPIE. We must go back to basics.

### Design Principles for This Iteration

1. **KISS**: Standard magnitude projection, standard ePIE update, no momentum
2. **Correct the support mask**: This is potentially the #1 issue — wrong geometry = wrong reconstruction
3. **Gentle prior**: Weak reflectivity enforcement, gradually strengthened
4. **Robust FFT detection**: Run both conventions for 3 full epochs, compare residuals
5. **More iterations**: Use the full time budget efficiently

### Key Changes from Iter 9

| Aspect | Iter 9 (PSNR=8.35) | This Iteration |
|--------|---------------------|----------------|
| **Fourier constraint** | RAAR (β_raar=0.9) | **Standard magnitude projection** |
| **Momentum** | Nesterov γ=0.3 | **None** |
| **Object update β** | 0.8 | **1.0** (standard ePIE) |
| **α regularization** | 1e-4 | **1e-3** (more stable) |
| **Prior strength** | β_prior=0.1 (90% R_p) | **β_prior=0.5** ramp from 0.8→0.3 |
| **FFT detection** | 20 positions, epoch 0 | **3 full epochs each convention** |
| **Support mask** | pitch=30 only | **Try multiple pitches**, pick best |
| **Probe init** | Gaussian σ=40 | **Data-driven + Gaussian, both tested** |
| **Probe update start** | Epoch 5 | **Epoch 3** |
| **TV smoothing** | Every 25 epochs | **None** (simplify) |

def make_support(obj_size, pad_sz, pitch_px, n_pads):
    arr_ext = (n_pads - 1) * pitch_px + pad_sz
    arr_off = (obj_size - arr_ext) // 2

### Step 6: Robust FFT Convention & Configuration Selection

**Run 3 full epochs for each configuration (convention × probe × pitch) and pick the one with lowest Fourier residual.** This is the most important step — getting the configuration right.

```python
def prepare_diff_amp(diff_amp_raw, use_fftshift):
    if use_fftshift:
        return np.fft.ifftshift(diff_amp_raw, axes=(-2, -1))
    else:
        return diff_amp_raw.copy()

def init_object(obj_size, S, R_p, R_Cu):
    O = np.full((obj_size, obj_size), R_p, dtype=np.complex128)
    O[S] = R_Cu
    return O

def run_rpPIE_epochs(diff_amp_w, P_init, O_init, S, positions, n_epochs, 
                      beta_obj=1.0, alpha=1e-3, beta_prior=0.5):
    """Run n_epochs of standard rpPIE and return final Fourier residual."""
    O = O_init.copy()
    P = P_init.copy()
    ps = P.shape[0]
    eps = 1e-12
    R_p_val = R_p  # from outer scope
    outside_S = ~S
    
    for epoch in range(n_epochs):
        order = np.random.permutation(len(positions))
        for idx in order:
            ry, rx = positions[idx]
            O_patch = O[ry:ry+ps, rx:rx+ps]
            
            psi = P * O_patch
            Psi = np.fft.fft2(psi)
            
            # Standard magnitude projection
            Psi_amp = np.abs(Psi) + eps
            Psi_c = diff_amp_w[idx] * Psi / Psi_amp
            
            psi_c = np.fft.ifft2(Psi_c)
            dpsi = psi_c - psi
            
            # ePIE object update (Eq. 6 Step 1)
            P_abs = np.abs(P)
            P_max = np.max(P_abs) + eps
            denom = P_abs**2 + alpha
            weight = (P_abs / P_max) * np.conj(P) / denom
            O[ry:ry+ps, rx:rx+ps] = O_patch + beta_obj * weight * dpsi
        
        # rpPIE prior (Eq. 6 Step 2)
        O[outside_S] = beta_prior * O[outside_S] + (1.0 - beta_prior) * R_p_val
    
    # Compute residual
    residual = 0.0
    for idx in range(len(positions)):
        ry, rx = positions[idx]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        residual += np.sum((np.abs(Psi) - diff_amp_w[idx])**2)
    
    return residual, O, P

# Test configurations: 2 conventions × 2 probes × best 2 pitches = up to 8
# But limit to save time: test 2 conventions × 2 probes × 1 pitch (30) = 4 configs
# Each config: 3 epochs × 81 positions ≈ 3 * 0.16s = 0.5s per config → 2s total

# First determine DC location
center_region = slice(ps//2 - 32, ps//2 + 32)
center_energy = np.mean(I[:, center_region, center_region])
corner_energy = (np.mean(I[:, :32, :32]) + np.mean(I[:, :32, -32:]) +
                 np.mean(I[:, -32:, :32]) + np.mean(I[:, -32:, -32:])) / 4
dc_centered = (center_energy > corner_energy)

# Prepare both conventions

diff_amp_B = prepare_diff_amp(diff_amp, not dc_centered)    # opposite

best_residual = np.inf
best_config = None

pitch_test = 30  # primary candidate
S_test, _ = supports[pitch_test]

for conv_idx, da in enumerate([diff_amp_A, diff_amp_B]):
    for p_idx, P_init in enumerate([P_gauss, P_data]):
        O_init = init_object(obj_size, S_test, R_p, R_Cu)
        res, _, _ = run_rpPIE_epochs(da, P_init, O_init, S_test, positions, 
                                      n_epochs=3, beta_obj=1.0, alpha=1e-3, beta_prior=0.5)
        if res < best_residual:
            best_residual = res
            best_config = (conv_idx, p_idx, pitch_test)

conv_idx, p_idx, best_pitch = best_config
diff_amp_work = [diff_amp_A, diff_amp_B][conv_idx]
P = [P_gauss.copy(), P_data.copy()][p_idx]
S = supports[best_pitch][0]
outside_S = ~S

# Now test other pitches with the best convention/probe
for p_cand in pitch_candidates:
    if p_cand == best_pitch or p_cand not in supports:
        continue
    S_cand, _ = supports[p_cand]
    O_init = init_object(obj_size, S_cand, R_p, R_Cu)
    P_test = [P_gauss.copy(), P_data.copy()][p_idx]
    res, _, _ = run_rpPIE_epochs(diff_amp_work, P_test, O_init, S_cand, positions,
                                  n_epochs=3, beta_obj=1.0, alpha=1e-3, beta_prior=0.5)
    if res < best_residual:
        best_residual = res
        best_pitch = p_cand
        S = S_cand
        outside_S = ~S
```

### Step 7: Main rpPIE Reconstruction Loop

This is the core algorithm — **standard rpPIE from Eq. 6, no modifications**.

```python
# Initialize object
O = np.full((obj_size, obj_size), R_p, dtype=np.complex128)
O[S] = R_Cu

# Hyperparameters — conservative, stable
beta_obj = 1.0      # standard ePIE
alpha = 1e-3         # regularization (larger = more stable)
beta_probe = 0.5     # probe update strength
alpha_probe = 1e-3   # probe regularization
eps = 1e-12

# Prior schedule: start weak, strengthen gradually
# beta_prior: fraction of current estimate retained (1-beta_prior = fraction of R_p)
# Start at 0.8 (weak prior), decrease to 0.3 (strong prior)
def get_beta_prior(epoch, n_epochs):
    # Linear ramp: 0.8 → 0.3 over first half, then constant 0.3
    if epoch < n_epochs // 2:
        return 0.8 - 0.5 * (epoch / (n_epochs // 2))
    return 0.3

# Estimate time budget
time_elapsed = time.time() - t_start
time_remaining = 55.0 - time_elapsed
# Each epoch ≈ 0.16s (81 positions × ~2ms each)
max_epochs = min(int(time_remaining / 0.18), 350)
n_epochs = max(max_epochs, 100)

for epoch in range(n_epochs):
    if time.time() - t_start > 54.0:
        break
    
    beta_prior = get_beta_prior(epoch, n_epochs)
    
    # Random scan order for better convergence
    order = np.random.permutation(n_pos)
    
    # Precompute probe quantities (constant within epoch if probe not updated yet)
    P_abs = np.abs(P)
    P_max = np.max(P_abs) + eps
    P_conj = np.conj(P)
    P_abs_sq = P_abs ** 2
    obj_weight = (P_abs / P_max) * P_conj / (P_abs_sq + alpha)
    
    for idx in order:
        ry, rx = positions[idx]
        
        O_patch = O[ry:ry+ps, rx:rx+ps]
        
        # Exit wave
        psi = P * O_patch
        
        # Forward propagation
        Psi = np.fft.fft2(psi)
        
        # Standard magnitude projection (Fourier constraint)
        Psi_amp = np.abs(Psi) + eps
        Psi_c = diff_amp_work[idx] * Psi / Psi_amp
        
        # Inverse propagation
        psi_c = np.fft.ifft2(Psi_c)
        
        # Difference
        dpsi = psi_c - psi
        
        # Object update (Eq. 6 Step 1)
        O[ry:ry+ps, rx:rx+ps] = O_patch + beta_obj * obj_weight * dpsi
        
        # Probe update (start after epoch 3 for object to stabilize)
        if epoch >= 3:
            O_abs = np.abs(O_patch)
            O_max = np.max(O_abs) + eps
            O_conj = np.conj(O_patch)
            O_abs_sq = O_abs ** 2
            probe_weight = (O_abs / O_max) * O_conj / (O_abs_sq + alpha_probe)
            P = P + beta_probe * probe_weight * dpsi
            
            # Update precomputed probe quantities periodically
            # (every 20 positions to balance speed vs accuracy)
            # Actually, for correctness, update after each position
            P_abs = np.abs(P)
            P_max = np.max(P_abs) + eps
            P_conj = np.conj(P)
            P_abs_sq = P_abs ** 2
            obj_weight = (P_abs / P_max) * P_conj / (P_abs_sq + alpha)
    
    # --- End-of-epoch constraints ---
    
    # rpPIE reflectivity prior (Eq. 6 Step 2)
    O[outside_S] = beta_prior * O[outside_S] + (1.0 - beta_prior) * R_p
    
    # Amplitude constraint: |O| should not exceed physical bounds
    amp = np.abs(O)
    mask_large = amp > 1.0
    if np.any(mask_large):
        O[mask_large] *= 1.0 / amp[mask_large]
    
    # Probe energy normalization (prevent divergence)
    if epoch == 0:
        P_energy_ref = np.sum(np.abs(P)**2)
    else:
        P_energy = np.sum(np.abs(P)**2)
        if P_energy > 2.0 * P_energy_ref:
            P *= np.sqrt(P_energy_ref / P_energy)
```

### Step 9: Performance Optimization — Avoid Redundant Probe Recomputation

The probe weight recomputation inside the inner loop is expensive. Optimize:

```python
# OPTIMIZATION: Only recompute probe-derived weights every N positions
# when probe is being updated. This saves ~50% of computation.
# In the inner loop, replace the per-position probe weight update with:

if epoch >= 3 and (pos_count % 10 == 0):
    # Recompute probe weights every 10 positions
    P_abs = np.abs(P)
    P_max = np.max(P_abs) + eps
    P_conj = np.conj(P)
    P_abs_sq = P_abs ** 2
    obj_weight = (P_abs / P_max) * P_conj / (P_abs_sq + alpha)
```

This allows ~300+ epochs within the time budget, giving much better convergence.

## Summary of Critical Changes from Iter 9

| Change | Rationale |
|--------|-----------|
| **Remove RAAR** → standard magnitude projection | RAAR extrapolation destabilized convergence |
| **Remove momentum** | Non-convex problem; momentum pushed away from good basins |
| **β_obj = 1.0** (standard) | Standard ePIE normalization; 0.8 was under-updating |
| **α = 1e-3** (10× larger) | More stable denominator regularization |
| **β_prior ramp 0.8→0.3** | Gradual prior enforcement; 0.1 was too aggressive |
| **Probe weight update every 10 positions** | 3× speedup → more epochs in time budget |
| **Multi-pitch support testing** | Wrong support mask causes catastrophic failure |
| **3 full epoch convention test** | More reliable than 20-position spot check |
| **No TV smoothing** | Simplification; was adding complexity without benefit |
