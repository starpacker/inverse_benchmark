## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function (amplitude encodes reflectivity, phase encodes topography via $\Delta\Phi = 4\pi h/\lambda$)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|_2^2$$

## 2. [Proposed Strategy]

**Algorithm**: **ePIE (alpha=1) with updated-object probe update, fast convention detection, NO probe re-normalization, and momentum acceleration.**

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 11 | 2.92 | Global-max normalization killed updates |
| 12 | 6.54 | Sequential ePIE, poor probe init, too few epochs |
| 13 | 1.84 | Nesterov momentum caused divergence |
| 14 | 2.51 | Wrong FFT convention; scalar max normalization; no amplitude clipping |
| 15 | 7.10 | **alpha=0.1 too conservative → slow convergence; probe re-normalization every epoch destroyed probe scale; 12s wasted on detection; stale O_patch in probe update** |

### Critical Fixes for This Iteration

1. **Use ePIE (alpha=1.0)** — Standard ePIE converges much faster than rPIE with small alpha. The denominator becomes $|P|^2 + \epsilon$, giving maximum step size.
2. **NO probe re-normalization after init** — The probe magnitude must converge to its physical value. Only normalize once at initialization.
3. **Updated object in probe update** — Use the UPDATED object patch (after object update) in the probe update denominator and numerator for faster convergence.
4. **Reduce detection time to 4s total** (2s per convention) — Maximize main reconstruction budget.
5. **Gentle momentum on object** — Use $\beta_{mom}=0.3$ damped momentum (NOT Nesterov, which diverged in iter 13). Apply only after first epoch stabilizes.
6. **Step size parameters** — Object step $\beta_O = 1.0$, Probe step $\beta_P = 1.0$ (standard ePIE).
7. **Object amplitude clipping at 1.0** — Physical reflectance constraint, applied gently.

### Step 3: Probe Initialization (One-time normalization)

```python
def init_probe(damp):
    """Initialize probe from average diffraction amplitude. Normalize ONCE."""
    avg_amp = np.mean(damp, axis=0)  # (256, 256)
    P = np.fft.ifft2(avg_amp)  # complex (256, 256)
    # Apply gentle Gaussian window for spatial compactness
    yy, xx = np.mgrid[:ps, :ps]
    cy, cx = ps / 2.0, ps / 2.0
    r2 = (yy - cy)**2 + (xx - cx)**2
    window = np.exp(-r2 / (2 * 80.0**2))  # σ=80 pixels, generous
    window_fft = np.fft.ifftshift(window)
    P *= window_fft
    # Normalize ONCE to reasonable scale — then NEVER again
    P = P / (np.max(np.abs(P)) + eps) * 1.0
    return P

def init_object():
    """Initialize object as uniform reflector with amplitude ~0.7 (reasonable Cu reflectance)."""
    return np.full((obj_size, obj_size), 0.7 + 0j, dtype=np.complex128)
```

### Step 4: Core ePIE Engine (alpha=1, updated-object probe update, optional momentum)

```python
def run_ePIE(O, P, damp, positions, time_budget, use_momentum=True, beta_mom=0.3):
    """
    Standard ePIE (Maiden & Rodenburg 2009) with alpha=1.
    
    Object update:
      O(r) += conj(P) * dpsi / (|P|^2_max + eps)   [ePIE standard]
    
    Probe update (uses UPDATED object):
      P(r) += conj(O_new) * dpsi / (|O_new|^2_max + eps)
    
    Key: NO probe re-normalization. Physical probe scale preserved.
    """
    t0 = time.time()
    n_pos = len(positions)
    epoch = 0
    
    # Momentum buffer for object
    O_prev = O.copy() if use_momentum else None
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        # Save for momentum
        if use_momentum and epoch >= 1:
            O_before_epoch = O.copy()
        
        # Randomize position ordering each epoch
        order = np.random.permutation(n_pos)
        
        P_max_sq = np.max(np.abs(P)**2)
        
        for idx in range(n_pos):
            if idx % 27 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            # Extract object patch
            O_patch = O[ry:ry+ps, rx:rx+ps]
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fft2(psi)
            
            # Fourier magnitude replacement
            Psi_corrected = damp[j] * np.exp(1j * np.angle(Psi))
            
            # Inverse propagate
            psi_c = np.fft.ifft2(Psi_corrected)
            
            # Difference
            dpsi = psi_c - psi
            
            # ePIE Object update (alpha=1 → divide by max|P|^2)
            O[ry:ry+ps, rx:rx+ps] = O_patch + np.conj(P) * dpsi / (P_max_sq + eps)
            
            # Object amplitude clipping (reflectance ≤ 1)
            O_patch_new = O[ry:ry+ps, rx:rx+ps]
            amp = np.abs(O_patch_new)
            clip_mask = amp > 1.0
            if np.any(clip_mask):
                O_patch_new[clip_mask] *= 1.0 / amp[clip_mask]
                O[ry:ry+ps, rx:rx+ps] = O_patch_new
            
            # ePIE Probe update using UPDATED object
            O_patch_updated = O[ry:ry+ps, rx:rx+ps]
            O_max_sq = np.max(np.abs(O_patch_updated)**2)
            P += np.conj(O_patch_updated) * dpsi / (O_max_sq + eps)
        
        # Update P_max_sq for next epoch
        P_max_sq = np.max(np.abs(P)**2)
        
        # Apply gentle momentum to object (NOT Nesterov — just simple momentum)
        if use_momentum and epoch >= 2:
            O = O + beta_mom * (O - O_before_epoch)
            # Re-clip after momentum
            amp_O = np.abs(O)
            clip_global = amp_O > 1.0
            if np.any(clip_global):
                O[clip_global] *= 1.0 / amp_O[clip_global]
        
        epoch += 1
    
    return O, P, epoch
```

### Step 5: Residual Computation for Convention Detection

```python
def compute_residual(O, P, damp, positions, n_sample=20):
    """Compute Fourier residual on a subset of positions."""
    rng = np.random.RandomState(42)
    subset = rng.choice(len(positions), min(n_sample, len(positions)), replace=False)
    res = 0.0
    norm = 0.0
    for j in subset:
        ry, rx = positions[j]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - damp[j])**2)
        norm += np.sum(damp[j]**2)
    return res / (norm + eps)
```

### Step 8: Post-Processing — Phase Alignment & Output

```python
# Build illuminated-region mask
scan_count = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(n_pos):
    ry, rx = positions[j]
    scan_count[ry:ry+ps, rx:rx+ps] += 1.0

illuminated = scan_count > 0
well_illuminated = scan_count >= 4  # well-covered region

amp_result = np.abs(best_O)
phase_result = np.angle(best_O)

# Phase reference: find dominant phase in well-illuminated region
valid = well_illuminated & (amp_result > 0.1)

if np.sum(valid) > 100:
    phase_valid = phase_result[valid]
    # Histogram-based mode finding
    n_bins = 720
    hist, bin_edges = np.histogram(phase_valid, bins=n_bins, range=(-np.pi, np.pi))
    # Smooth
    kernel = np.ones(21) / 21.0
    hist_smooth = np.convolve(hist, kernel, mode='same')
    mode_idx = np.argmax(hist_smooth)
    phase_ref = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2.0
else:
    phase_ref = 0.0

# Remove global phase offset (substrate → 0 phase)
best_O *= np.exp(-1j * phase_ref)

# Zero out non-illuminated regions
best_O[~illuminated] = 0.0

# Final output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(best_O).astype(np.float32)
output[1] = np.angle(best_O).astype(np.float32)
```

## 4. [Loss Function & Optimizer]

### Loss (Fourier magnitude constraint)
$$\mathcal{L} = \sum_{j=1}^{81} \frac{\left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2}{\left\| \sqrt{I_j} \right\|_2^2}$$

### ePIE Update Rules (alpha=1, Maiden & Rodenburg 2009)

**Object update** at position $j$:
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{\max_{\mathbf{r}'}|P(\mathbf{r}')|^2 + \epsilon} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$ (using **updated** $O^{(n+1)}$):
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \frac{O^{(n+1)*}(\mathbf{r}+\mathbf{R}_j)}{\max_{\mathbf{r}'}|O^{(n+1)}(\mathbf{r}')|^2 + \epsilon} \cdot \Delta\psi_j(\mathbf{r})$$

**Momentum** (applied after epoch $\geq 2$, gentle):
$$O^{(n+1)} \leftarrow O^{(n+1)} + 0.3 \cdot (O^{(n+1)} - O^{(n)})$$
