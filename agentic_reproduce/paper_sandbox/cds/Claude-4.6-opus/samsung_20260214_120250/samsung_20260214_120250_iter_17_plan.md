## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|_2^2$$

---

## 2. [Proposed Strategy]

**Algorithm**: **Standard ePIE with PIXEL-WISE normalization, Gaussian probe initialization, and rpPIE reflectivity prior.**

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 14 | 2.51 | Wrong FFT convention; scalar max normalization |
| 15 | 7.10 | alpha=0.1 too conservative; probe re-normalization every epoch; stale O in probe update |
| 16 | 5.29 | **Global max normalization instead of pixel-wise; bad probe init (ifft2 of real amplitude); object init 0.7 too low; wasted time on convention detection** |

### Critical Fixes for This Iteration

1. **PIXEL-WISE normalization** — The standard ePIE formula: $O \mathrel{+}= \beta \frac{P^*}{|P|^2 + \epsilon} \Delta\psi$. NOT dividing by global max. This is the most critical fix.
2. **Gaussian disk probe initialization in REAL SPACE** — Initialize probe as a Gaussian-windowed disk, not from ifft2 of average amplitude. Physical probe is spatially compact.
3. **Object init at amplitude 1.0** — Fully reflective substrate assumption; let algorithm reduce amplitude where needed.
4. **Faster FFT convention detection** — Use DC-component heuristic (2s total) instead of running mini-reconstructions.
5. **Momentum from epoch 1** with small $\beta_{mom}=0.2$ — No two-phase split.
6. **rpPIE reflectivity prior** — Outside support, gently push toward known substrate reflectivity (as described in paper's Eq. 6 Step 2).
7. **Step sizes**: $\beta_O = 0.9$, $\beta_P = 0.5$ (probe updates more conservatively).

### Step 3: FFT Convention Detection (Fast Heuristic — 2s max)

The key insight: if the DC component (brightest pixel) of diffraction patterns is at the center, data is fftshifted and we need `ifftshift` before use. If DC is at corner (0,0), data is in standard FFT order.

```python
def detect_fft_convention(diff_amp):
    """Fast DC-location heuristic. Check a few patterns."""
    avg_pattern = np.mean(diff_amp[:10]**2, axis=0)  # average intensity
    h, w = avg_pattern.shape
    
    # Energy in center 32x32 vs corner 32x32
    c = 16
    center_energy = np.sum(avg_pattern[h//2-c:h//2+c, w//2-c:w//2+c])
    corner_energy = np.sum(avg_pattern[:2*c, :2*c]) + np.sum(avg_pattern[:2*c, -2*c:]) + \
                    np.sum(avg_pattern[-2*c:, :2*c]) + np.sum(avg_pattern[-2*c:, -2*c:])
    
    if center_energy > corner_energy:
        # DC is at center → need ifftshift to move to corner for fft2
        return True  # needs ifftshift
    else:
        return False  # already in FFT order

needs_shift = detect_fft_convention(diff_amp)

if needs_shift:
    diff_amp_use = np.fft.ifftshift(diff_amp, axes=(-2, -1))
else:
    diff_amp_use = diff_amp.copy()
```

```python
def quick_test(damp, O, P, positions, max_iters=20):
    """Run ~20 position updates and return residual."""
    rng = np.random.RandomState(0)
    subset = rng.choice(len(positions), min(max_iters, len(positions)), replace=False)
    O_t = O.copy()

        O_patch = O_t[ry:ry+ps, rx:rx+ps]

        Psi = np.fft.fft2(psi)
        Psi_c = damp[j] * np.exp(1j * np.angle(Psi))
        psi_c = np.fft.ifft2(Psi_c)
        dpsi = psi_c - psi
        denom = np.abs(P)**2 + eps
        O_t[ry:ry+ps, rx:rx+ps] = O_patch + 0.9 * np.conj(P) * dpsi / denom
    # Compute residual on same subset

        psi = P * O_t[ry:ry+ps, rx:rx+ps]
```

### Step 4: Probe Initialization (Gaussian Disk in Real Space)

```python
def init_probe(diff_amp_use):
    """
    Initialize probe as Gaussian disk in real space.
    Scale to match diffraction data energy.
    """
    P = np.zeros((ps, ps), dtype=np.complex128)
    yy, xx = np.mgrid[:ps, :ps]
    cy, cx = ps / 2.0, ps / 2.0
    r2 = (yy - cy)**2 + (xx - cx)**2
    
    # Gaussian with sigma ~ 50 pixels (reasonable Airy-like spot)
    sigma = 50.0
    P = np.exp(-r2 / (2 * sigma**2)).astype(np.complex128)
    
    # Scale: match average diffraction energy
    # |F{P*O}|^2 ~ I, with O~1 → |F{P}|^2 ~ I
    # Energy conservation: sum|P|^2 * N = sum|F{P}|^2
    mean_energy = np.mean(np.sum(diff_amp_use**2, axis=(-2, -1)))
    current_energy = np.sum(np.abs(P)**2)
    P *= np.sqrt(mean_energy / (current_energy + eps))
    
    return P

def init_object():
    """Initialize as uniform amplitude=1.0, phase=0."""
    return np.ones((obj_size, obj_size), dtype=np.complex128)

P = init_probe(diff_amp_use)
O = init_object()
```

### Step 5: Core ePIE Engine (PIXEL-WISE normalization)

This is the **critical difference** from all previous iterations. Standard ePIE uses pixel-wise $|P(\mathbf{r})|^2$ in the denominator, NOT the global maximum.

```python
def run_ePIE(O, P, damp, positions, time_budget, 
             beta_obj=0.9, beta_probe=0.5, use_momentum=True, beta_mom=0.2):
    """
    Standard ePIE (Maiden & Rodenburg 2009) with PIXEL-WISE normalization.
    
    Object update:
      O(r) += beta_obj * conj(P) / (|P|^2 + eps) * dpsi
    
    Probe update (using UPDATED object):
      P(r) += beta_probe * conj(O_new) / (|O_new|^2 + eps) * dpsi
    
    NO global max normalization. Pixel-wise division.
    """
    t0 = time.time()
    n_pos = len(positions)
    epoch = 0
    O_prev = O.copy()
    
    # Pre-compute |P|^2 (updated each epoch)
    P_abs2 = np.abs(P)**2
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        O_before = O.copy() if use_momentum else None
        
        # Randomize scan order
        order = np.random.permutation(n_pos)
        
        for idx in range(n_pos):
            # Time check every 27 positions
            if idx % 27 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            # Extract patch
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fft2(psi)
            
            # Fourier magnitude replacement
            amp_Psi = np.abs(Psi)
            phase_Psi = Psi / (amp_Psi + eps)  # unit phase
            Psi_c = damp[j] * phase_Psi
            
            # Inverse propagate
            psi_c = np.fft.ifft2(Psi_c)
            
            # Difference signal
            dpsi = psi_c - psi
            
            # === OBJECT UPDATE (pixel-wise) ===
            O[ry:ry+ps, rx:rx+ps] += beta_obj * np.conj(P) * dpsi / (P_abs2 + eps)
            
            # Amplitude clipping: |O| <= 1.0 (physical reflectance)
            O_patch_new = O[ry:ry+ps, rx:rx+ps]
            amp = np.abs(O_patch_new)
            mask = amp > 1.0
            if np.any(mask):
                O_patch_new[mask] /= amp[mask]
                O[ry:ry+ps, rx:rx+ps] = O_patch_new
            
            # === PROBE UPDATE (pixel-wise, using UPDATED object) ===
            O_patch_updated = O[ry:ry+ps, rx:rx+ps]
            O_abs2 = np.abs(O_patch_updated)**2
            P += beta_probe * np.conj(O_patch_updated) * dpsi / (O_abs2 + eps)
        
        # Update cached P_abs2 for next epoch
        P_abs2 = np.abs(P)**2
        
        # Momentum on object (simple, not Nesterov)
        if use_momentum and epoch >= 1 and O_before is not None:
            delta = O - O_before
            O += beta_mom * delta
            # Re-clip
            amp_O = np.abs(O)
            mask_g = amp_O > 1.0
            if np.any(mask_g):
                O[mask_g] /= amp_O[mask_g]
        
        epoch += 1
    
    return O, P, epoch
```

### Step 7: Post-Processing — Phase Alignment & Output

```python
# Build coverage map
scan_count = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(n_pos):
    ry, rx = positions[j]
    scan_count[ry:ry+ps, rx:rx+ps] += 1.0

illuminated = scan_count > 0
well_illuminated = scan_count >= 4

amp_result = np.abs(O)
phase_result = np.angle(O)

# Phase reference: find the dominant phase value in well-illuminated region
# This should correspond to the substrate (most common material)
valid = well_illuminated & (amp_result > 0.3)

if np.sum(valid) > 100:
    phase_valid = phase_result[valid]
    # Histogram-based mode finding with circular-aware binning
    n_bins = 720
    hist, bin_edges = np.histogram(phase_valid, bins=n_bins, range=(-np.pi, np.pi))
    # Smooth with wide kernel
    kernel_size = 31
    kernel = np.ones(kernel_size) / kernel_size
    hist_smooth = np.convolve(hist, kernel, mode='same')
    mode_idx = np.argmax(hist_smooth)
    phase_ref = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2.0
else:
    phase_ref = np.mean(phase_result[illuminated]) if np.any(illuminated) else 0.0

# Remove global phase offset
O *= np.exp(-1j * phase_ref)

# Zero out non-illuminated regions
O[~illuminated] = 0.0

# Final output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(O).astype(np.float32)
output[1] = np.angle(O).astype(np.float32)
```

## 4. [Loss Function & Optimizer]

### Loss (Fourier magnitude constraint)
$$\mathcal{L} = \sum_{j=1}^{81} \frac{\left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2}{\left\| \sqrt{I_j} \right\|_2^2}$$

### Standard ePIE Update Rules (PIXEL-WISE — Maiden & Rodenburg 2009)

**Object update** at position $j$:
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \beta_O \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{|P(\mathbf{r}-\mathbf{R}_j)|^2 + \epsilon} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$ (using **updated** $O^{(n+1)}$):
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \beta_P \frac{O^{(n+1)*}(\mathbf{r}+\mathbf{R}_j)}{|O^{(n+1)}(\mathbf{r}+\mathbf{R}_j)|^2 + \epsilon} \cdot \Delta\psi_j(\mathbf{r})$$

**Momentum** (simple, applied after each epoch $\geq 1$):
$$O \leftarrow O + 0.2 \cdot (O^{(n+1)} - O^{(n)})$$

### Parameters
- $\beta_O = 0.9$, $\beta_P = 0.5$
- $\epsilon = 10^{-8}$
- $\beta_{mom} = 0.2$
- Probe: Gaussian $\sigma=50$ pixels, energy-matched to data
- Object: uniform amplitude 1.0, phase 0
