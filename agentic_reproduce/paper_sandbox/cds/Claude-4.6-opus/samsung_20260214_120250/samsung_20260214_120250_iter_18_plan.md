## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function (amplitude encodes reflectivity, phase encodes topography via $\Delta\Phi = 4\pi h/\lambda$)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function (Airy or speckle)
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|_2^2$$

## 2. [Proposed Strategy]

**Algorithm**: **rPIE (regularized PIE) with max-norm denominators, followed by rpPIE reflectivity prior enforcement.**

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 14 | 2.51 | Wrong FFT convention; scalar max normalization but other bugs |
| 15 | 7.10 | alpha=0.1 too conservative; probe re-normalization every epoch |
| 16 | 5.29 | Global max normalization but bad probe init (ifft2 of real amplitude); object init 0.7 too low |
| 17 | 7.36 | **Pixel-wise division by |P|² amplified noise in low-probe regions; amplitude clipping to 1.0 too restrictive; momentum destabilized early epochs** |

### Critical Fixes for Iteration 18

1. **MAX-NORM scalar denominators** — The canonical ePIE uses $\max(|P|^2)$ as scalar denominator, NOT pixel-wise $|P(\mathbf{r})|^2$. This was the primary failure in iter 17.
2. **rPIE regularized denominator** — Use $(1-\alpha)|P_j|^2 + \alpha \max|P|^2$ as per Maiden et al. 2017, with $\alpha = 0.5$. This interpolates between pixel-wise and max-norm for stability.
3. **Remove amplitude clipping entirely** — The clipping to 1.0 was too restrictive. Let the algorithm find the correct amplitudes.
4. **Remove momentum entirely** — Until basic reconstruction works, no momentum. Simpler is better.
5. **Robust FFT convention detection** — Run 1 full epoch (81 positions) each way, pick lower residual. Spend up to 4s on this.
6. **Probe init**: Flat-top disk with soft Gaussian edge (radius ~80 pixels, edge roll-off ~20 pixels). More physical than pure Gaussian.
7. **Object init**: Uniform amplitude = 0.9 (average reflectivity), phase = 0.
8. **Probe update uses dpsi from BEFORE object update** — This is actually standard ePIE (simultaneous update). But ensure the formulas are correct.

### Step 3: FFT Convention Detection (Robust — Run 1 Full Epoch Each Way)

```python
def compute_residual_one_epoch(damp, O_init, P_init, positions, beta_O=0.9, alpha_rPIE=0.5):
    """Run 1 epoch of rPIE, return normalized residual."""
    O = O_init.copy()
    P = P_init.copy()
    n_pos = len(positions)
    
    for j in range(n_pos):
        ry, rx = positions[j]
        O_patch = O[ry:ry+ps, rx:rx+ps]
        psi = P * O_patch
        Psi = np.fft.fft2(psi)
        amp_Psi = np.abs(Psi)
        Psi_c = damp[j] * Psi / (amp_Psi + eps)
        psi_c = np.fft.ifft2(Psi_c)
        dpsi = psi_c - psi
        
        P_abs2 = np.abs(P)**2
        P_max2 = np.max(P_abs2)
        denom_O = (1 - alpha_rPIE) * P_abs2 + alpha_rPIE * P_max2 + eps
        O[ry:ry+ps, rx:rx+ps] += beta_O * np.conj(P) * dpsi / denom_O
    
    # Compute residual over all positions
    res = 0.0
    norm = 0.0
    for j in range(n_pos):
        ry, rx = positions[j]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - damp[j])**2)
        norm += np.sum(damp[j]**2)
    return res / (norm + eps)

# Initialize probe and object for testing
def make_probe(damp_test):
    P = np.zeros((ps, ps), dtype=np.complex128)
    yy, xx = np.mgrid[:ps, :ps]
    cy, cx = ps / 2.0, ps / 2.0
    r = np.sqrt((yy - cy)**2 + (xx - cx)**2)
    # Flat-top disk with soft edge
    radius = 80.0
    edge_width = 20.0
    P = (0.5 * (1 + np.cos(np.pi * np.clip((r - radius) / edge_width, 0, 1)))).astype(np.complex128)
    P[r > radius + edge_width] = 0.0
    # Energy scale
    mean_energy = np.mean(np.sum(damp_test**2, axis=(-2, -1)))
    current_energy = np.sum(np.abs(P)**2)
    P *= np.sqrt(mean_energy / (current_energy + eps))
    return P

O_init = 0.9 * np.ones((obj_size, obj_size), dtype=np.complex128)

# Test both conventions
damp_raw = diff_amp.copy()
damp_shifted = np.fft.ifftshift(diff_amp, axes=(-2, -1))

P_test_shifted = make_probe(damp_shifted)
```

### Step 5: Core rPIE Engine (MAX-NORM / Regularized Denominators)

The **canonical rPIE** update (Maiden et al. 2017):

**Object update:**
$$O(\mathbf{r}) \mathrel{+}= \beta_O \frac{P^*(\mathbf{r})}{(1-\alpha)|P(\mathbf{r})|^2 + \alpha \max|P|^2} \cdot \Delta\psi(\mathbf{r})$$

**Probe update:**
$$P(\mathbf{r}) \mathrel{+}= \beta_P \frac{O^*(\mathbf{r})}{(1-\alpha)|O(\mathbf{r})|^2 + \alpha \max|O_{\text{patch}}|^2} \cdot \Delta\psi(\mathbf{r})$$

where $\alpha = 0.5$ provides a balanced regularization.

```python
def run_rPIE(O, P, damp, positions, time_budget,
             beta_O=0.9, beta_P=0.5, alpha=0.5):
    """
    rPIE with regularized denominators.
    
    Object: O += beta_O * conj(P) * dpsi / ((1-alpha)*|P|^2 + alpha*max|P|^2)
    Probe:  P += beta_P * conj(O) * dpsi / ((1-alpha)*|O|^2 + alpha*max|O|^2)
    
    NO amplitude clipping. NO momentum.
    """
    t0 = time.time()
    n_pos = len(positions)
    epoch = 0
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        # Randomize scan order
        order = np.random.permutation(n_pos)
        
        # Pre-compute probe max for this epoch
        P_abs2 = np.abs(P)**2
        P_max2 = np.max(P_abs2)
        
        for idx in range(n_pos):
            if idx % 27 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            # Extract patch
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fft2(psi)
            
            # Fourier magnitude replacement
            amp_Psi = np.abs(Psi)
            Psi_c = damp[j] * Psi / (amp_Psi + eps)
            
            # Inverse propagate
            psi_c = np.fft.ifft2(Psi_c)
            
            # Difference signal
            dpsi = psi_c - psi
            
            # === OBJECT UPDATE (rPIE regularized denominator) ===
            denom_O = (1 - alpha) * P_abs2 + alpha * P_max2 + eps
            O[ry:ry+ps, rx:rx+ps] += beta_O * np.conj(P) * dpsi / denom_O
            
            # === PROBE UPDATE (rPIE regularized denominator) ===
            O_patch_new = O[ry:ry+ps, rx:rx+ps]
            O_abs2 = np.abs(O_patch_new)**2
            O_max2 = np.max(O_abs2)
            denom_P = (1 - alpha) * O_abs2 + alpha * O_max2 + eps
            P += beta_P * np.conj(O_patch_new) * dpsi / denom_P
        
        # Update cached values
        P_abs2 = np.abs(P)**2
        P_max2 = np.max(P_abs2)
        
        epoch += 1
    
    return O, P, epoch
```

```python
# Build coverage map
scan_count = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(n_pos):
    ry, rx = positions[j]
    scan_count[ry:ry+ps, rx:rx+ps] += 1.0

phase_result = np.angle(O)

# Phase reference: histogram mode in well-illuminated region
well_illuminated = scan_count >= 4
valid = well_illuminated & (amp_result > 0.1)

if np.sum(valid) > 100:
    phase_valid = phase_result[valid]
    n_bins = 720
    hist, bin_edges = np.histogram(phase_valid, bins=n_bins, range=(-np.pi, np.pi))
    kernel_size = 31
    kernel = np.ones(kernel_size) / kernel_size
    hist_smooth = np.convolve(hist, kernel, mode='same')
    mode_idx = np.argmax(hist_smooth)
    phase_ref = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2.0

    phase_ref = 0.0

# Remove global phase offset
O *= np.exp(-1j * phase_ref)

O[~illuminated] = 0.0

output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
```

## 4. [Loss Function & Optimizer]

### Loss (Fourier magnitude constraint)
$$\mathcal{L} = \sum_{j=1}^{81} \frac{\left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2}{\left\| \sqrt{I_j} \right\|_2^2}$$

### rPIE Update Rules (Maiden et al. 2017)

**Object update** at position $j$:
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \beta_O \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{(1-\alpha)|P(\mathbf{r}-\mathbf{R}_j)|^2 + \alpha\max_{\mathbf{r}}|P|^2} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$:
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \beta_P \frac{O^{(n+1)*}(\mathbf{r}+\mathbf{R}_j)}{(1-\alpha)|O^{(n+1)}(\mathbf{r}+\mathbf{R}_j)|^2 + \alpha\max_{\mathbf{r}}|O_{\text{patch}}|^2} \cdot \Delta\psi_j(\mathbf{r})$$

### Parameters
| Parameter | Value | Rationale |
|-----------|-------|-----------|
| $\beta_O$ | 0.9 | Standard ePIE range |
| $\beta_P$ | 0.5 | Conservative probe update |
| $\alpha$ | 0.5 | Balanced rPIE regularization |
| $\epsilon$ | $10^{-12}$ | Numerical safety only |
| Probe init | Flat-top disk, $r=80$, edge=20 | Physical Airy-like |
| Object init | amplitude=0.9, phase=0 | Reflective substrate |
| Amp clipping | **NONE** | Removed — was too restrictive |
| Momentum | **NONE** | Removed — simplify first |

### Key Differences from Iteration 17

| Property | Iter 17 (PSNR=7.36) | Iter 18 (This) |
|----------|----------------------|-----------------|
| **Denominator** | Pixel-wise $\|P\|^2 + \epsilon$ | **rPIE: $(1-\alpha)\|P\|^2 + \alpha\max\|P\|^2$** |
| **Amplitude clipping** | Clip to 1.0 | **NONE** |
| **Momentum** | 0.2 from epoch 1 | **NONE** |
| **Probe init** | Gaussian $\sigma=50$ | **Flat-top disk $r=80$, soft edge** |
| **Object init** | amplitude=1.0 | **amplitude=0.9** |
| **FFT convention** | DC heuristic + 20-iter test | **Full 1-epoch test both ways** |
| **$\epsilon$** | $10^{-8}$ | **$10^{-12}$ (safety only, not regularization)** |
