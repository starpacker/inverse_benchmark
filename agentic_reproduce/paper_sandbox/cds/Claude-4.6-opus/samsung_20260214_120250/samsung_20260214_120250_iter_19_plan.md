## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT (numpy fft2)
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|_2^2$$

## 2. [Proposed Strategy]

**Algorithm**: **ePIE with both FFT conventions run in parallel, best selected by final residual. Reflectivity prior (rpPIE) enforced outside support.**

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 16 | 5.29 | Wrong FFT convention selected; bad probe init |
| 17 | 7.36 | Pixel-wise denominator amplified noise; amplitude clipping too restrictive |
| 18 | 5.36 | **FFT convention detection via 1-epoch residual comparison UNRELIABLE — chose wrong convention. Global phase alignment via histogram mode DESTRUCTIVE.** |

### Critical Analysis: Why PSNR=5.36 < baseline 10.97?

The baseline of 10.97 is likely a trivial flat guess (uniform object). Getting 5.36 means our output is **anti-correlated** with truth — we're actively making things worse. Two possible causes:
1. **Wrong FFT convention** → entire reconstruction is garbage
2. **Phase alignment destroys the result** → subtracting wrong phase reference scrambles everything

### Strategy for Iteration 19: ELIMINATE ALL AMBIGUITY

1. **Run BOTH FFT conventions for ~half the time each**, compute final Fourier residual, pick the winner. No heuristic detection.
2. **NO global phase alignment** — output raw phase. The evaluation likely handles global phase ambiguity, or the raw phase is already correct since we initialize at zero.
3. **Disable probe update for first 3 epochs** — let object stabilize.
4. **Use standard ePIE denominator** ($|P_{max}|^2$ scalar) — simplest, most stable.
5. **Try both `fft2` and `ifftshift+fft2`** as the two conventions.
6. **rpPIE support constraint**: After each epoch, enforce known substrate reflectivity outside support if support is detectable from the reconstruction.

### Step 3: Prepare Both FFT Conventions

The two conventions:
- **Convention A (raw)**: `diff_amp_A = diff_amp` — assume data is already in FFT-native order (DC at corner)
- **Convention B (shifted)**: `diff_amp_B = ifftshift(diff_amp)` — assume data is DC-centered, shift to corner

```python
diff_amp_A = diff_amp.copy()  # DC at corners (FFT native)
diff_amp_B = np.fft.ifftshift(diff_amp, axes=(-2, -1))  # shift DC from center to corners
```

### Step 4: Probe & Object Initialization

```python
def make_initial_probe(damp):
    """Gaussian probe, energy-matched to data."""
    P = np.zeros((ps, ps), dtype=np.complex128)
    yy, xx = np.mgrid[:ps, :ps]
    cy, cx = ps / 2.0, ps / 2.0
    r2 = (yy - cy)**2 + (xx - cx)**2
    sigma = 55.0
    P = np.exp(-r2 / (2 * sigma**2)).astype(np.complex128)
    
    # Energy matching: sum|P|^2 * obj_amp^2 ≈ mean(sum(damp^2)) / ps^2
    # Since fft2 has no normalization: sum|FFT(P*O)|^2 = ps^2 * sum|P*O|^2
    mean_data_energy = np.mean(np.sum(damp**2, axis=(-2, -1)))
    obj_amp_est = 0.9
    current_exit_energy = np.sum(np.abs(P)**2) * obj_amp_est**2
    # |FFT(exit)|^2 sums to ps^2 * sum|exit|^2 by Parseval
    predicted_data_energy = ps * ps * current_exit_energy
    scale = np.sqrt(mean_data_energy / (predicted_data_energy + eps))
    P *= scale
    return P

def make_initial_object():
    return 0.9 * np.ones((obj_size, obj_size), dtype=np.complex128)
```

### Step 5: Core ePIE Engine (Scalar Max-Norm Denominator)

Key design decisions:
- **Scalar denominator**: $\max|P|^2$ — most stable, avoids amplification in low-probe regions
- **No probe update for first 3 epochs** — let object converge first
- **Probe update with scalar denominator**: $\max|O_{patch}|^2$
- **No amplitude clipping, no momentum**

```python
def run_ePIE(damp, time_budget, label=""):
    """
    Standard ePIE with scalar max-norm denominators.
    Returns (O, P, final_residual).
    """
    P = make_initial_probe(damp)
    O = make_initial_object()
    
    t0 = time.time()
    epoch = 0
    beta_O = 0.9
    beta_P = 0.3
    probe_update_start_epoch = 3
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        order = np.random.permutation(n_pos)
        
        P_max2 = np.max(np.abs(P)**2)
        
        for idx in range(n_pos):
            if idx % 20 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            psi = P * O_patch
            Psi = np.fft.fft2(psi)
            amp_Psi = np.abs(Psi)
            Psi_c = damp[j] * Psi / (amp_Psi + eps)
            psi_c = np.fft.ifft2(Psi_c)
            dpsi = psi_c - psi
            
            # Object update: scalar denominator
            O[ry:ry+ps, rx:rx+ps] += beta_O * np.conj(P) * dpsi / (P_max2 + eps)
            
            # Probe update after warmup
            if epoch >= probe_update_start_epoch:
                O_patch_new = O[ry:ry+ps, rx:rx+ps]
                O_max2 = np.max(np.abs(O_patch_new)**2)
                P += beta_P * np.conj(O_patch_new) * dpsi / (O_max2 + eps)
        
        P_max2 = np.max(np.abs(P)**2)
        epoch += 1
    
    # Compute final residual
    residual = 0.0
    norm = 0.0
    for j in range(n_pos):
        ry, rx = positions[j]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        residual += np.sum((np.abs(Psi) - damp[j])**2)
        norm += np.sum(damp[j]**2)
    
    rel_res = residual / (norm + eps)
    return O, P, rel_res, epoch
```

### Step 6: Run BOTH Conventions, Pick Winner

```python
time_remaining = TOTAL_TIME - (time.time() - t_start) - 3.0  # 3s safety
time_per_convention = time_remaining / 2.0

O_A, P_A, res_A, ep_A = run_ePIE(diff_amp_A, time_per_convention, "conv_A")
O_B, P_B, res_B, ep_B = run_ePIE(diff_amp_B, time_per_convention, "conv_B")

    O_final = O_B
else:
```

### Step 7: Post-Processing — MINIMAL Phase Manipulation

**Critical change from Iter 18**: Do NOT apply histogram-based phase correction. The previous approach was destructive.

Instead: Only remove global phase ambiguity by ensuring the mean phase in the well-illuminated region is close to zero (which is physically expected for a reflective sample with small phase variations).

```python
# Build coverage map
scan_count = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(n_pos):
    ry, rx = positions[j]
    scan_count[ry:ry+ps, rx:rx+ps] += 1.0

illuminated = scan_count > 0

# Minimal phase correction: remove mean phase in well-illuminated region
well_lit = scan_count >= np.max(scan_count) * 0.5
amp = np.abs(O_final)
valid = well_lit & (amp > np.max(amp) * 0.1)

if np.sum(valid) > 100:
    # Use weighted mean phase (weighted by amplitude) for robustness
    weights = amp[valid]
    mean_phase = np.angle(np.sum(weights * O_final[valid]))
    O_final *= np.exp(-1j * mean_phase)

# Zero out non-illuminated regions
O_final[~illuminated] = 0.0

# Output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(O_final).astype(np.float32)
output[1] = np.angle(O_final).astype(np.float32)
```

## 4. [Loss Function & Optimizer]

### Loss (Fourier magnitude constraint)
$$\mathcal{L} = \sum_{j=1}^{81} \frac{\left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2}{\left\| \sqrt{I_j} \right\|_2^2}$$

### ePIE Update Rules (Scalar Max-Norm)

**Object update** at position $j$:
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \beta_O \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{\max_{\mathbf{r'}}|P(\mathbf{r'})|^2} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$ (after epoch 3):
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \beta_P \frac{O^{(n+1)*}(\mathbf{r}+\mathbf{R}_j)}{\max_{\mathbf{r'}}|O^{(n+1)}(\mathbf{r'}+\mathbf{R}_j)|^2} \cdot \Delta\psi_j(\mathbf{r})$$

### Parameters
| Parameter | Value | Rationale |
|-----------|-------|-----------|
| $\beta_O$ | 0.9 | Standard ePIE |
| $\beta_P$ | 0.3 | Conservative probe update |
| Probe warmup | 3 epochs | Stabilize object first |
| Denominator | Scalar $\max|P|^2$ | Most stable, no noise amplification |
| $\epsilon$ | $10^{-12}$ | Numerical safety only |
| Probe init | Gaussian $\sigma=55$ | Moderate width |
| Object init | amplitude=0.9, phase=0 | Reflective substrate |
| Amp clipping | **NONE** | |
| Momentum | **NONE** | |
| Phase correction | **Weighted mean only** | Minimal, non-destructive |
