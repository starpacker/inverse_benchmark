

# Problem Formulation, Strategy, and Plan for Periodic Structure Ptychographic Phase Retrieval

---

## 1. [Problem Formulation]

### Forward Model

We seek to recover a complex-valued 2D reflection function $O(\mathbf{r})$, where $\mathbf{r} = (\xi, \eta)$, from far-field intensity-only measurements.

**Exit wave formation:**
$$\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})$$

where $P(\mathbf{r})$ is the known probe (illumination) function and $\mathbf{R}_j$ is the $j$-th scan position.

**Fraunhofer propagation (forward operator $\mathcal{A}$):**
$$\hat{\Psi}_j(\mathbf{k}) = \mathcal{F}\{\Psi_j(\mathbf{r})\}$$

with detector-plane spatial frequencies:
$$f_x = \frac{x}{\lambda z}, \quad f_y = \frac{y}{\lambda z}$$

valid when $z \gg \frac{2d^2}{\lambda}$ (Eq. 2).

**Measurement model:**
$$I_j(\mathbf{k}) = \left|\hat{\Psi}_j(\mathbf{k})\right|^2 + n_j(\mathbf{k})$$

where $n_j(\mathbf{k}) \sim \mathcal{N}(0, \sigma^2)$ is additive Gaussian noise.

### Object Model

The reflection function encodes both amplitude (material reflectivity) and phase (topography):

$$O(\mathbf{r}) = \begin{cases} R_{\text{Cu}}(\mathbf{r}) \cdot \exp\!\left(i \frac{4\pi h(\mathbf{r})}{\lambda}\right), & \mathbf{r} \in S_{\text{pad}} \\ R_p, & \mathbf{r} \notin S_{\text{pad}} \end{cases}$$

where:
- $R_{\text{Cu}}$ is the complex reflectivity of Cu pads computed from optical constants $(n, k)$ via the transmission matrix method through the SiCN/SiO₂/Si layer stack
- $R_p$ is the known substrate (inter-pad) reflectivity
- $h(\mathbf{r}) \in [1, 3]$ nm is the recession depth (constant per pad, random across pads)
- $\Delta\Phi = \frac{4\pi h}{\lambda} \leq \frac{4\pi \cdot 3}{13.5} \approx 2.8$ rad (no phase wrapping)
- $S = \bigcup_{m,n} S_{m,n}$ is the binary support mask defining the 15×15 periodic pad array
- $\lambda = 13.5$ nm

### Inverse Problem Statement

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{J} \left\| \sqrt{I_j(\mathbf{k})} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|^2$$

subject to:
- $O(\mathbf{r}) = R_p$ for $\mathbf{r} \notin S$ (reflectivity prior)
- $|O(\mathbf{r})|$ consistent with material reflectivities for $\mathbf{r} \in S$

## 2. [Proposed Strategy]

**Primary algorithms:** rpIO and rpPIE — reflectivity-prior-augmented variants of Hybrid Input-Output (HIO) and extended Ptychographic Iterative Engine (ePIE).

### Why these algorithms?

1. **Standard HIO/PIE fail** for periodic structures because diffraction orders are spatially separated at the detector, creating a sparse Fourier-space sampling problem that traps iterative projection methods in local minima.

2. **rpIO/rpPIE inject physics** via a known substrate reflectivity prior $R_p$ outside the pad support $S$, converting the unconstrained region from a free variable into a hard/soft constraint. This dramatically reduces the solution space and provides a "reference wave" anchoring the phase.

3. **Stability:** Both are projection-based iterative methods (well-understood convergence), not experimental deep learning approaches. ePIE is the gold standard for ptychography.

4. **Two-regime testing:**
   - **rpIO** (single-shot CDI): Tests the reflectivity prior without scanning.
   - **rpPIE** (ptychographic): Tests with scanning diversity + reflectivity prior.

### Architecture Summary

```
rpIO:  Fourier magnitude projection  ←→  Real-space support + reflectivity prior
rpPIE: [ePIE object update per scan position] → [Reflectivity prior enforcement outside S]
```

## 3. [Step-by-Step Plan]

### Step 1: Data Preprocessing & Simulation Setup

#### 1.1 Object Construction

1. **Define grid:** Total object size = 15 pads × pitch_pixels = 15 × 30 = 450 pixels (or 15 × 37 = 555 pixels). Add border for probe extent. Total grid: $N \times N$ where $N \geq 450 + 2 \times \text{probe\_radius}$.

2. **Generate support mask $S$:**
   ```
   S[m·pitch : m·pitch + pad_size, n·pitch : n·pitch + pad_size] = 1
   for m, n in range(15)
   ```

3. **Assign recession depths:** For each of the 225 pads, draw $h_{m,n} \sim \mathcal{U}[1, 3]$ nm independently. Store as ground truth.

4. **Compute reflectivities via Transmission Matrix Method (TMM):**
   - **Substrate (outside pads):** Model as SiCN/SiO₂/Si multilayer. Compute $R_p = r_{\text{substrate}}$ (complex Fresnel reflection coefficient at normal incidence, $\theta = 0$).
   - **Pad regions:** Model as Cu/SiCN/SiO₂/Si stack. Compute $R_{\text{Cu}}$ (complex reflection coefficient).
   - TMM for normal incidence through $M$ layers:
     $$M_j = \begin{pmatrix} \cos\delta_j & -i\sin\delta_j / \tilde{n}_j \\ -i\tilde{n}_j\sin\delta_j & \cos\delta_j \end{pmatrix}, \quad \delta_j = \frac{2\pi \tilde{n}_j d_j}{\lambda}$$
     where $d_j$ is layer thickness. The total transfer matrix $\mathbf{M} = \prod_j M_j$ yields:
     $$r = \frac{(\mathbf{M}_{11} + \mathbf{M}_{12}\tilde{n}_s)\tilde{n}_0 - (\mathbf{M}_{21} + \mathbf{M}_{22}\tilde{n}_s)}{(\mathbf{M}_{11} + \mathbf{M}_{12}\tilde{n}_s)\tilde{n}_0 + (\mathbf{M}_{21} + \mathbf{M}_{22}\tilde{n}_s)}$$
   - **Note:** Layer thicknesses for SiCN, SiO₂ are not explicitly given in the problem statement. I will treat them as configurable parameters (e.g., SiCN cap ~2-5 nm, SiO₂ ~few nm, Si substrate semi-infinite). If the paper uses specific thicknesses, match them. Otherwise, use typical EUV metrology stack thicknesses.

5. **Construct ground truth object:**
   $$O_{\text{true}}(\mathbf{r}) = S(\mathbf{r}) \cdot R_{\text{Cu}} \cdot e^{i \cdot 4\pi h(\mathbf{r})/\lambda} + (1 - S(\mathbf{r})) \cdot R_p$$

#### 1.2 Probe Construction

**Airy spot:**
$$P(\mathbf{r}) = \left[\frac{2 J_1(\pi r / r_0)}{\pi r / r_0}\right]^2 \quad \text{(intensity)}$$
or in complex amplitude form with appropriate normalization. The probe diameter should be chosen such that it illuminates multiple pads (for ptychographic diversity) — likely covering ~4-6 pads across, consistent with 40-pixel step and 75% overlap meaning probe diameter ≈ 160 pixels.

**Speckle field:**
Generate by propagating a random phase screen through an aperture:
$$P_{\text{speckle}}(\mathbf{r}) = \mathcal{F}^{-1}\{A(\mathbf{k}) \cdot e^{i\phi_{\text{rand}}(\mathbf{k})}\}$$
where $A(\mathbf{k})$ is a circular aperture and $\phi_{\text{rand}} \sim \mathcal{U}[0, 2\pi)$.

#### 1.3 Scan Positions

Generate raster scan positions $\{\mathbf{R}_j\}_{j=1}^{J}$ with step size = 40 pixels in both $\xi$ and $\eta$. The number of positions depends on the object extent and probe size. For ~75% overlap with probe diameter $D$: step = $D/4 = 40$ px → $D = 160$ px. Number of scan positions along one axis: $\lceil(N_{\text{obj}} - D) / 40\rceil + 1$.

#### 1.4 Diffraction Pattern Simulation

For each scan position $j$:
1. Extract exit wave: $\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O_{\text{true}}(\mathbf{r})$
2. Propagate: $\hat{\Psi}_j(\mathbf{k}) = \mathcal{F}\{\Psi_j(\mathbf{r})\}$
3. Compute intensity: $I_j(\mathbf{k}) = |\hat{\Psi}_j(\mathbf{k})|^2$
4. Add noise: $I_j^{\text{noisy}}(\mathbf{k}) = I_j(\mathbf{k}) + \sigma \cdot \epsilon$, $\epsilon \sim \mathcal{N}(0,1)$
5. Enforce non-negativity: $I_j^{\text{noisy}} = \max(I_j^{\text{noisy}}, 0)$

**Noise level $\sigma$:** Should be specified as SNR or percentage. If not given, sweep reasonable values (e.g., SNR = 20, 30, 40 dB).

### Step 2: Forward Operator Implementation

#### 2.1 Fraunhofer Propagation

$$\mathcal{A}_j: O \mapsto \mathcal{F}\{P(\cdot - \mathbf{R}_j) \cdot O(\cdot)\}$$

Implementation via 2D FFT with proper centering (`fftshift`/`ifftshift`):

```python
def forward(O, P, R_j):
    """Compute far-field diffraction amplitude."""
    psi = P * extract_region(O, R_j, probe_shape)
    Psi_hat = fftshift(fft2(ifftshift(psi)))
    return Psi_hat

def fourier_magnitude_projection(Psi_hat, I_measured):
    """Replace amplitude, keep phase."""
    amp_measured = np.sqrt(np.maximum(I_measured, 0))
    phase = np.angle(Psi_hat)
    Psi_corrected = amp_measured * np.exp(1j * phase)
    return Psi_corrected
```

### Step 3: Solver Implementation

#### 3.1 Algorithm 1: rpIO (Reflectivity-Prior Input-Output)

**Single-shot CDI** (no scanning, single diffraction pattern from full illumination).

**Initialization:** $O_0(\mathbf{r}) = R_p$ everywhere (or random phase inside $S$ with amplitude $|R_{\text{Cu}}|$).

**Iteration $n$:**

1. **Forward propagate:**
   $$\hat{\Psi}_n(\mathbf{k}) = \mathcal{F}\{O_n(\mathbf{r})\}$$

2. **Fourier magnitude constraint:**
   $$\hat{\Psi}'_n(\mathbf{k}) = \sqrt{I(\mathbf{k})} \cdot \frac{\hat{\Psi}_n(\mathbf{k})}{|\hat{\Psi}_n(\mathbf{k})| + \epsilon}$$

3. **Inverse propagate:**
   $$O'_n(\mathbf{r}) = \mathcal{F}^{-1}\{\hat{\Psi}'_n(\mathbf{k})\}$$

4. **Real-space constraint with reflectivity prior (Eq. 5):**
   $$O_{n+1}(\mathbf{r}) = \begin{cases} O'_n(\mathbf{r}), & \mathbf{r} \in S \\ \beta \, O_n(\mathbf{r}) + (1 - \beta) \, R_p, & \mathbf{r} \notin S \end{cases}$$

   where $\beta \in (0, 1)$ is the HIO feedback parameter (typically $\beta = 0.9$).

**Key difference from standard HIO:** Outside support, standard HIO uses $O_{n+1} = O_n - \beta O'_n$. rpIO instead drives the outside-support region toward the known reflectivity $R_p$ via convex relaxation.

**Convergence criterion:**
$$\mathcal{E}_n = \frac{\sum_{\mathbf{k}} \left(\sqrt{I(\mathbf{k})} - |\hat{\Psi}_n(\mathbf{k})|\right)^2}{\sum_{\mathbf{k}} I(\mathbf{k})}$$

#### 3.2 Algorithm 2: rpPIE (Reflectivity-Prior Ptychographic Iterative Engine)

**Initialization:** $O_0(\mathbf{r}) = R_p$ everywhere (uniform substrate reflectivity).

**Per-epoch iteration** (one pass through all $J$ scan positions in random order):

For each scan position $j$ (randomly shuffled):

**Step A: ePIE-style object update (Eq. 6, line 1)**

1. Form exit wave:
   $$\Psi_n(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O_n(\mathbf{r})$$

2. Forward propagate:
   $$\hat{\Psi}_n(\mathbf{k}) = \mathcal{F}\{\Psi_n(\mathbf{r})\}$$

3. Fourier magnitude replacement:
   $$\hat{\Psi}_{c,n}(\mathbf{k}) = \sqrt{I_j(\mathbf{k})} \cdot \frac{\hat{\Psi}_n(\mathbf{k})}{|\hat{\Psi}_n(\mathbf{k})| + \epsilon}$$

4. Inverse propagate corrected exit wave:
   $$\Psi_{c,n}(\mathbf{r}) = \mathcal{F}^{-1}\{\hat{\Psi}_{c,n}(\mathbf{k})\}$$

5. Update object in the region illuminated by probe at position $j$:
   $$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \frac{|P(\mathbf{r} - \mathbf{R}_j)|}{|P_{\max}(\mathbf{r} - \mathbf{R}_j)|} \cdot \frac{P^*(\mathbf{r} - \mathbf{R}_j)}{|P(\mathbf{r} - \mathbf{R}_j)|^2 + \alpha} \cdot \beta \left(\Psi_{c,n}(\mathbf{r}) - \Psi_n(\mathbf{r})\right)$$

   where:
   - $P^*$ is the complex conjugate of the probe
   - $|P_{\max}|$ is the maximum modulus of the probe (scalar normalizer)
   - $\alpha$ is a Wiener-type regularization parameter (e.g., $\alpha = 10^{-3} \cdot |P_{\max}|^2$)
   - $\beta$ is the step size / feedback parameter

**Step B: Reflectivity prior enforcement (Eq. 6, line 2)**

$$O_{n+1}(\mathbf{r}) = \begin{cases} O_{n+1}(\mathbf{r}), & \mathbf{r} \in S \\ \beta \, O_{n+1}(\mathbf{r}) + (1 - \beta) \, R_p, & \mathbf{r} \notin S \end{cases}$$

**Note:** Step B is applied **after every scan position update**, not just at the end of an epoch. This is the "two-step nested structure."

#### 3.3 Baseline Algorithms (for comparison)

1. **Standard HIO:** Same as rpIO but outside support: $O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) - \beta O'_n(\mathbf{r})$ for $\mathbf{r} \notin S$.

2. **Standard PIE (ePIE):** Same as rpPIE Step A only, without Step B (no reflectivity prior enforcement).

#### 3.4 Hyperparameters

| Parameter | Symbol | Typical Value | Notes |
|-----------|--------|---------------|-------|
| Feedback/step size | $\beta$ | 0.9 (HIO), 1.0 (PIE) | Tune in [0.5, 1.0] |
| Regularization | $\alpha$ | $10^{-3}|P_{\max}|^2$ | Prevents division by zero |
| Division stabilizer | $\epsilon$ | $10^{-8}$ | In Fourier magnitude replacement |
| Max iterations | — | 200–500 (IO), 50–200 epochs (PIE) | Until convergence |

### Step 4: Loss Function & Evaluation Protocol

#### 4.1 Loss / Error Metric (during reconstruction)

**Fourier-space error (R-factor):**
$$\mathcal{E}_{\text{Fourier}} = \frac{\sum_j \sum_{\mathbf{k}} \left|\sqrt{I_j(\mathbf{k})} - |\hat{\Psi}_j(\mathbf{k})|\right|^2}{\sum_j \sum_{\mathbf{k}} I_j(\mathbf{k})}$$

This is the primary convergence monitor, plotted on a log scale. rpPIE should achieve ~2 orders of magnitude lower error than standard PIE.

#### 4.2 Evaluation Metrics (post-reconstruction)

1. **Phase RMSE:**
   $$\text{RMSE}_\phi = \sqrt{\frac{1}{|S|}\sum_{\mathbf{r} \in S} \left(\phi_{\text{recon}}(\mathbf{r}) - \phi_{\text{true}}(\mathbf{r})\right)^2}$$
   after removing global phase ambiguity via:
   $$\hat{\phi}_{\text{offset}} = \arg\min_{\phi_0} \sum_{\mathbf{r} \in S} |\phi_{\text{recon}}(\mathbf{r}) - \phi_0 - \phi_{\text{true}}(\mathbf{r})|^2$$

2. **Recession depth error:**
   $$\Delta h = \frac{\lambda}{4\pi} \cdot \text{RMSE}_\phi$$

3. **Cross-sectional phase profiles:** Extract 1D phase cuts through pad rows and compare with ground truth.

4. **PSNR (if applicable):**
   $$\text{PSNR} = 10 \log_{10}\left(\frac{\max(\phi_{\text{true}})^2}{\text{MSE}_\phi}\right)$$
   Target: match within ±0.3 dB.

5. **Convergence curves:** Plot $\log_{10}(\mathcal{E}_{\text{Fourier}})$ vs. iteration for all four algorithms.

### Critical Implementation Details

1. **FFT convention:** Use `numpy.fft.fft2` with consistent `fftshift`/`ifftshift` pairing. The exit wave should be centered before FFT.

2. **Probe windowing for PIE:** When extracting/updating object regions, use array slicing with proper boundary handling. The probe defines the active region.

3. **Random scan order:** Shuffle scan positions each epoch for PIE/rpPIE to avoid systematic bias.

4. **Phase ambiguity:** All phase retrieval has a global phase ambiguity. Remove it before comparison by aligning to ground truth.

5. **Periodic structure challenge:** The key insight is that periodic objects produce discrete diffraction orders. Between orders, the intensity is near-zero, providing no phase information. The reflectivity prior fills this gap by constraining the inter-pad regions, effectively providing a reference amplitude and phase that links the diffraction orders coherently.

6. **TMM implementation:** For normal incidence ($\theta = 0$), the transfer matrix simplifies. Each layer $j$ with thickness $d_j$ and complex refractive index $\tilde{n}_j$:
   $$M_j = \begin{pmatrix} \cos(k_j d_j) & -\frac{i}{k_j}\sin(k_j d_j) \\ -ik_j\sin(k_j d_j) & \cos(k_j d_j) \end{pmatrix}$$
   where $k_j = 2\pi\tilde{n}_j/\lambda$. Compute $r = \frac{M_{11}n_0 + M_{12}n_0 n_s - M_{21} - M_{22}n_s}{M_{11}n_0 + M_{12}n_0 n_s + M_{21} + M_{22}n_s}$ (adjusting for s/p polarization at normal incidence where they coincide).
