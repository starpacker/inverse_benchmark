## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n_j(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{N \times N}$: complex reflection function (amplitude + phase encoding topography)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function (unknown, must be jointly retrieved)
- $\mathbf{R}_j$: scan position for the $j$-th diffraction pattern
- $\mathcal{F}$: 2D FFT (Fraunhofer propagation)
- $n_j \sim \mathcal{N}(0, \sigma^2)$: Gaussian noise
- $I_j \in \mathbb{R}^{256 \times 256}$: measured intensity at detector

### Critical Dimensional Analysis

**Input:** 81 diffraction patterns of shape $(256, 256)$ → tensor $(81, 256, 256)$
- 81 = $9 \times 9$ raster scan grid
- Each pattern is $256 \times 256$ pixels

**Output:** $(2, 600, 600)$ → **2-channel real tensor**: channel 0 = amplitude $|O|$, channel 1 = phase $\angle O$, both on a $600 \times 600$ pixel grid.

### Scan Geometry Inference

With 81 positions on a $9 \times 9$ grid, scan step of 40 pixels, probe size $256 \times 256$:
- Object extent along one axis: $256 + 8 \times 40 = 256 + 320 = 576$ pixels
- With small border → output grid $600 \times 600$ (matches required output!)
- Overlap: $1 - 40/256 = 84.4\%$ (consistent with "~75% overlap" — close enough)

### Object Phase Model

$$O(\mathbf{r}) = \begin{cases} R_{\text{Cu}} \cdot e^{i 4\pi h(\mathbf{r})/\lambda}, & \mathbf{r} \in S \\ R_p, & \mathbf{r} \notin S \end{cases}$$

where $h(\mathbf{r}) \in [1, 3]$ nm per pad, $\lambda = 13.5$ nm, $\Delta\Phi_{\max} \approx 2.8$ rad.

**Algorithm:** Joint Object-Probe rpPIE with robust convergence strategy

## 3. [Step-by-Step Plan]

### Step 1: Data Loading & Geometry Inference

1. **Load input tensor** $(81, 256, 256)$ — these are the 81 measured diffraction intensity patterns $\{I_j\}$.

2. **Infer scan grid:**
   - $N_{\text{scan}} = 81 = 9 \times 9$
   - Scan step = 40 pixels (from problem statement)
   - Generate positions: $\mathbf{R}_j = (r_x, r_y)$ where $r_x = i \times 40$, $r_y = j \times 40$ for $i, j \in \{0, 1, \ldots, 8\}$
   - Positions are in object-space pixel coordinates, defining the top-left corner of where the probe window sits on the object

3. **Determine object size:**
   - $N_{\text{obj}} = 256 + 8 \times 40 + \text{padding} = 600$
   - This exactly matches the required output $(2, 600, 600)$
   - If $256 + 320 = 576 < 600$, center the reconstruction in a $600 \times 600$ canvas with offset = $(600 - 576) / 2 = 12$ pixels on each side. Scan positions shifted by offset: $\mathbf{R}_j = (12 + i \times 40, 12 + j \times 40)$.

4. **Preprocessing diffraction patterns:**
   - Ensure non-negativity: $I_j = \max(I_j, 0)$
   - Compute amplitude: $A_j = \sqrt{I_j}$
   - Compute total intensity per pattern for normalization diagnostics

### Step 2: Initialization

#### 2.1 Object Initialization

**Strategy:** Initialize with uniform complex constant (not random — avoids trapping).

$$O_0(\mathbf{r}) = \bar{A} \cdot e^{i \cdot 0} \quad \forall \mathbf{r}$$

where $\bar{A}$ is estimated from data:
$$\bar{A} = \sqrt{\frac{\sum_j \sum_{\mathbf{k}} I_j(\mathbf{k})}{N_{\text{scan}} \cdot N_p^2 \cdot \|P_0\|^2}}$$

Or simply $O_0 = 1.0 + 0j$ (unit reflectivity, zero phase). This is a safe starting point.

#### 2.2 Probe Initialization

**Critical insight from failure:** The probe must be estimated, not assumed.

**Initial guess:** Use the average diffraction pattern to seed the probe:
$$P_0(\mathbf{r}) = \mathcal{F}^{-1}\left\{\sqrt{\frac{1}{J}\sum_{j=1}^{J} I_j(\mathbf{k})}\right\}$$

This gives a reasonable starting probe that is consistent with the data's Fourier magnitudes. The phase of $P_0$ will be refined during joint reconstruction.

**Alternative Airy initialization (backup):**
$$P_0(\mathbf{r}) = \text{circ}(r/r_0) \quad \text{(circular aperture in Fourier space)}$$
with $r_0$ chosen so the probe has appropriate spatial extent.

#### 2.3 Support Mask Construction

Generate the binary support mask $S$ for the $600 \times 600$ grid:
- Pad size: 22 pixels
- Pitch: 30 pixels (try also 37 if 30 fails)
- Array: $15 \times 15$ pads
- Total array extent: $15 \times 30 = 450$ pixels (or $15 \times 37 = 555$)
- Center in the $600 \times 600$ grid
- Offset: $(600 - 450) / 2 = 75$ pixels (or $(600 - 555)/2 = 22.5$ → 23 pixels)

```python
S = np.zeros((600, 600), dtype=bool)
for m in range(15):
    for n in range(15):
        y0 = offset + m * pitch
        x0 = offset + n * pitch
        S[y0:y0+pad_size, x0:x0+pad_size] = True
```

**If support is provided as input:** Check if any of the 81 input channels or supplementary data contains the support mask. If not, construct it from parameters.

#### 2.4 Substrate Reflectivity

Compute $R_p$ using TMM at normal incidence for the substrate stack (SiCN/SiO₂/Si):

For normal incidence, Fresnel reflection at each interface between media $a$ and $b$:
$$r_{ab} = \frac{\tilde{n}_a - \tilde{n}_b}{\tilde{n}_a + \tilde{n}_b}$$

For a simple estimate (if layer thicknesses are unknown), use the vacuum-SiCN interface:
$$R_p \approx \frac{1 - \tilde{n}_{\text{SiCN}}}{1 + \tilde{n}_{\text{SiCN}}} = \frac{1 - (0.9791 + 0.0059i)}{1 + (0.9791 + 0.0059i)}$$

**Pragmatic approach:** If exact $R_p$ is uncertain, treat it as a free parameter initialized from the reconstruction itself — after a few initial PIE epochs, estimate $R_p$ as the median complex value of $O$ outside support $S$.

### Step 3: Solver — Joint rpPIE with Probe Update

#### 3.1 Main Loop (per epoch)

```
for epoch in range(N_epochs):
    shuffle scan positions
    for j in shuffled_positions:
        # --- Step A: Forward model ---
        psi = P * O[Rj : Rj+256, ...]           # exit wave
        Psi_hat = FFT(psi)                        # far-field
        
        # --- Step B: Fourier magnitude constraint ---
        Psi_c = sqrt(I_j) * Psi_hat / (|Psi_hat| + eps)
        
        # --- Step C: Inverse propagate ---
        psi_c = IFFT(Psi_c)
        delta_psi = psi_c - psi                   # exit wave difference
        
        # --- Step D: Object update (ePIE) ---
        O[Rj:Rj+256] += (alpha_O * P* / (|P_max| * (|P|^2 + reg_O))) * delta_psi
        
        # --- Step E: Probe update (joint) ---
        if epoch >= probe_start_epoch:
            O_patch = O[Rj:Rj+256]
            P += (alpha_P * O_patch* / (|O_max| * (|O_patch|^2 + reg_P))) * delta_psi
        
        # --- Step F: Reflectivity prior (rpPIE) ---
        if epoch >= prior_start_epoch:
            gamma = min(1.0, (epoch - prior_start_epoch) / ramp_epochs)
            outside_S = ~S[Rj:Rj+256]  # support mask for this patch
            O_patch = O[Rj:Rj+256]
            O_patch[outside_S] = (1 - gamma*(1-beta)) * O_patch[outside_S] + gamma*(1-beta) * R_p
```

#### 3.2 Update Equations (Detailed)

**Object update:**
$$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \alpha_O \cdot \frac{P^*(\mathbf{r} - \mathbf{R}_j)}{|P_{\max}| \cdot \left(|P(\mathbf{r} - \mathbf{R}_j)|^2 + \alpha_{\text{reg}}\right)} \cdot \Delta\Psi(\mathbf{r})$$

**Probe update:**
$$P_{n+1}(\mathbf{r}) = P_n(\mathbf{r}) + \alpha_P \cdot \frac{O^*_n(\mathbf{r} + \mathbf{R}_j)}{|O_{\max}| \cdot \left(|O_n(\mathbf{r} + \mathbf{R}_j)|^2 + \alpha_{\text{reg},P}\right)} \cdot \Delta\Psi(\mathbf{r} + \mathbf{R}_j)$$

**Reflectivity prior (with ramp-up):**
$$O_{n+1}(\mathbf{r}) = \begin{cases} O_{n+1}(\mathbf{r}), & \mathbf{r} \in S \\ (1 - \gamma_n(1-\beta)) \, O_{n+1}(\mathbf{r}) + \gamma_n(1-\beta) \, R_p, & \mathbf{r} \notin S \end{cases}$$

where $\gamma_n = \min(1, n / N_{\text{ramp}})$ gradually activates the prior.

#### 3.3 Convergence Strategy & Hyperparameters

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| $N_{\text{epochs}}$ | **500** (minimum) | Previous 30 was catastrophically insufficient |
| $\alpha_O$ | 1.0 | Standard ePIE step size |
| $\alpha_P$ | 1.0 | Standard probe update step |
| $\beta$ | 0.9 → anneal to 0.5 | Start aggressive, reduce for fine convergence |
| $\alpha_{\text{reg}}$ | $0.1 \cdot |P_{\max}|^2$ | Larger than before to stabilize early iterations |
| $\alpha_{\text{reg},P}$ | $0.1 \cdot |O_{\max}|^2$ | Same philosophy |
| `probe_start_epoch` | 5 | Let object settle slightly before updating probe |
| `prior_start_epoch` | 20 | Let reconstruction form rough structure first |
| `ramp_epochs` | 50 | Gradually introduce reflectivity prior |
| $\epsilon$ (division guard) | $10^{-8}$ | Numerical stability |

**Annealing schedule for $\beta$:**
$$\beta(n) = \max(0.5, \, 0.9 - 0.4 \cdot n / N_{\text{epochs}})$$

**Convergence monitoring:**
$$\mathcal{E}_n = \frac{1}{J} \sum_{j=1}^{J} \frac{\left\|\sqrt{I_j} - |\mathcal{F}\{P \cdot O_{\mathbf{R}_j}\}|\right\|_2^2}{\left\|\sqrt{I_j}\right\|_2^2}$$

Compute every 10 epochs. If $\mathcal{E}_n$ increases for 20 consecutive checks, reduce $\alpha_O$ and $\alpha_P$ by factor 0.5.

#### 3.4 Adaptive $R_p$ Estimation

After epoch 50, periodically re-estimate $R_p$:
$$\hat{R}_p = \text{median}\{O(\mathbf{r}) : \mathbf{r} \notin S\}$$

This makes the algorithm robust to incorrect initial $R_p$ values. Use complex median (median of real and imaginary parts separately, or median of amplitude and phase).

### Step 4: Output Formatting

**The output must be $(2, 600, 600)$:**

```python
O_final = O  # complex, shape (600, 600)

# Remove global phase ambiguity
# Align phase to substrate: set phase of R_p region to reference
phase_ref = np.angle(np.mean(O_final[~S]))
O_final *= np.exp(-1j * phase_ref)

# Construct output
output = np.zeros((2, 600, 600), dtype=np.float32)
output[0] = np.abs(O_final)        # amplitude
output[1] = np.angle(O_final)      # phase (in radians)
```

**Alternative output interpretations to test if evaluation fails:**
- Channel 0: real part, Channel 1: imaginary part
- Channel 0: phase, Channel 1: amplitude
- Normalized to [0, 1] range
- Phase unwrapped and scaled to depth in nm

### Step 5: Loss Function & Evaluation

#### 5.1 Internal Loss (Reconstruction)

$$\mathcal{L}_{\text{recon}} = \sum_{j=1}^{81} \left\|\sqrt{I_j(\mathbf{k})} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right|\right\|_2^2$$

#### 5.2 External Evaluation (Expected)

The evaluator likely computes:
- **PSNR** between reconstructed and ground truth (on amplitude, phase, or both channels)
- **SSIM** on phase map (topographic fidelity)

Target: PSNR within ±0.3 dB of reference.

### Step 6: Robustness Measures

1. **If support mask is wrong:** Run first 100 epochs without support constraint (pure ePIE with probe update), then use the reconstruction to auto-detect periodic structure via autocorrelation, and derive support mask from that.

2. **If scan step ≠ 40:** Try step values {38, 40, 42, 44} and pick the one with lowest $\mathcal{E}$.

3. **If output format is wrong:** Save multiple formats and check which one the evaluator accepts:
   - `(2, 600, 600)` with `[amplitude, phase]`
   - `(2, 600, 600)` with `[real, imag]`
   - `(2, 600, 600)` with `[phase, amplitude]`

4. **Position refinement (optional):** After initial convergence, refine scan positions using cross-correlation of overlapping exit waves — this corrects for any systematic position error.

### Implementation Pseudocode (Complete)

```python
def solve(input_data):  # input_data: (81, 256, 256)
    # --- Geometry ---
    n_patterns = 81
    n_side = 9  # 9x9 grid
    probe_size = 256
    scan_step = 40
    obj_size = 600
    offset = (obj_size - (probe_size + (n_side-1)*scan_step)) // 2  # centering
    
    # --- Scan positions ---
    positions = []
    for i in range(n_side):
        for j in range(n_side):
            positions.append((offset + i*scan_step, offset + j*scan_step))
    
    # --- Diffraction amplitudes ---
    diff_amp = np.sqrt(np.maximum(input_data, 0))  # (81, 256, 256)
    
    # --- Initialize object ---
    O = np.ones((obj_size, obj_size), dtype=complex)
    
    # --- Initialize probe from mean diffraction pattern ---
    mean_amp = np.mean(diff_amp, axis=0)
    P = np.fft.ifftshift(np.fft.ifft2(np.fft.fftshift(mean_amp)))
    P = P.astype(complex)
    
    # --- Support mask ---
    S = build_support_mask(obj_size, pad_size=22, pitch=30, n_pads=15)
    
    # --- Substrate reflectivity (initial estimate) ---
    R_p = 1.0 + 0j  # will be refined
    
    # --- Main reconstruction loop ---
    for epoch in range(500):
        order = np.random.permutation(n_patterns)
        beta = max(0.5, 0.9 - 0.4 * epoch / 500)
        
        for idx in order:
            ry, rx = positions[idx]
            
            # Extract object patch
            O_patch = O[ry:ry+256, rx:rx+256]
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(psi)))
            
            # Magnitude replacement
            Psi_amp = np.abs(Psi)
            Psi_c = diff_amp[idx] * Psi / (Psi_amp + 1e-8)
            
            # Inverse propagate
            psi_c = np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(Psi_c)))
            
            # Difference
            dpsi = psi_c - psi
            
            # Object update
            P_conj = np.conj(P)
            P_abs2 = np.abs(P)**2
            P_max = np.max(np.abs(P))
            reg_O = 0.1 * P_max**2
            O_update = P_conj / (P_max * (P_abs2 + reg_O)) * dpsi
            O[ry:ry+256, rx:rx+256] += O_update
            
            # Probe update (after epoch 5)
            if epoch >= 5:
                O_patch_new = O[ry:ry+256, rx:rx+256]
                O_conj = np.conj(O_patch_new)
                O_abs2 = np.abs(O_patch_new)**2
                O_max = np.max(np.abs(O_patch_new))
                reg_P = 0.1 * O_max**2
                P += O_conj / (O_max * (O_abs2 + reg_P)) * dpsi
            
            # Reflectivity prior (after epoch 20, with ramp)
            if epoch >= 20:
                gamma = min(1.0, (epoch - 20) / 50)
                S_patch = S[ry:ry+256, rx:rx+256]
                outside = ~S_patch
                blend = gamma * (1 - beta)
                O[ry:ry+256, rx:rx+256][outside] = \
                    (1 - blend) * O[ry:ry+256, rx:rx+256][outside] + blend * R_p
        
        # Periodically re-estimate R_p
        if epoch == 60:
            R_p = np.median(np.real(O[~S])) + 1j * np.median(np.imag(O[~S]))
    
    # --- Format output ---
    phase_ref = np.angle(np.mean(O[~S]))
    O *= np.exp(-1j * phase_ref)
    
    output = np.zeros((2, 600, 600), dtype=np.float32)
    output[0] = np.abs(O)
    output[1] = np.angle(O)
    return output
```

### Failure Mode Contingencies

| Failure Mode | Detection | Mitigation |
|---|---|---|
| Wrong scan step | High residual error | Grid search over {36, 38, 40, 42, 44} |
| Wrong support | Phase doesn't show pads | Auto-detect from reconstruction autocorrelation |
| Wrong output format | Eval returns None | Try [real,imag], [phase,amp], normalized variants |
| Slow convergence | Error plateaus | Reduce regularization, increase epochs to 1000 |
| Probe diverges | Probe energy explodes | Clip probe amplitude, reduce $\alpha_P$ |
| Position errors | Artifacts at patch boundaries | Add position refinement via cross-correlation |
