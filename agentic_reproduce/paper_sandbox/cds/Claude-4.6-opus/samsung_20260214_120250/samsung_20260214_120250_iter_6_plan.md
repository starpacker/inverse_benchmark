## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function with $|O(\mathbf{r})| \leq 1$ (physical reflectivity)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function
- $\mathbf{R}_j$: scan position $j \in \{1, \ldots, 81\}$ on a $9 \times 9$ grid with step 40 pixels
- $\mathcal{F}$: 2D FFT (numpy convention: unnormalized forward, $1/N^2$ inverse)
- $n \sim \mathcal{N}(0, \sigma^2)$: additive Gaussian noise

### Key Physical Constraints

- **Reflectivity bound**: $|O(\mathbf{r})| \leq 1.0$ for all $\mathbf{r}$ (EUV reflection, no gain)
- **Phase model on pads**: $\angle O = 4\pi h / \lambda$, $h \in [1,3]$ nm, $\lambda = 13.5$ nm → $\Delta\Phi \in [0.93, 2.79]$ rad
- **Outside pads**: $O(\mathbf{r}) = R_p$ (known substrate complex reflectivity)

### Dimensional Analysis

- **Input**: $(81, 256, 256)$ — 81 measured diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: amplitude $|O|$, channel 1: phase $\angle O$
- **Object grid**: $256 + 8 \times 40 = 576$; centered in 600 → offset = 12 pixels each side

### Numpy FFT Scaling Convention (CRITICAL FIX)

For numpy's `fft2` (unnormalized forward):
$$\text{fft2}(x)[k] = \sum_n x[n] e^{-2\pi i nk/N}$$

Parseval's theorem: $\sum_k |\hat{x}[k]|^2 = N^2 \sum_n |x[n]|^2$

So: $\text{total\_detector\_intensity} = \sum_k I_j(k) = \sum_k |\hat{\psi}|^2 = N^2 \sum_r |P \cdot O_{\text{patch}}|^2$

This means: $\sum_r |P|^2 \approx \frac{\text{mean\_total\_intensity}}{N^2 \cdot \overline{|O|^2}}$

With $|O| \approx 1$ (reflectivity near unity at EUV for these materials): $\sum_r |P|^2 \approx \frac{\text{mean\_total\_intensity}}{N^2}$

where $N = 256$.

### Step 3: Multilayer Reflectivity Computation (TMM)

Compute complex reflectivity for the substrate stack and copper pads using transfer matrix method at normal incidence, $\lambda = 13.5$ nm.

**Complex refractive indices** (from CXRO):
- Vacuum: $\tilde{n}_0 = 1.0$
- SiCN: $\tilde{n}_1 = 0.9791 + 0.0059i$ → $\delta_1 = 0.0209$, $\beta_1 = 0.0059$
- SiO₂: $\tilde{n}_2 = 0.9780 + 0.0108i$
- Si: $\tilde{n}_3 = 0.9991 + 0.0018i$ (substrate, semi-infinite)
- Cu: $\tilde{n}_{\text{Cu}} = 0.9625 + 0.0613i$

**Note on EUV convention**: $\tilde{n} = 1 - \delta + i\beta$ where $\delta, \beta > 0$. The complex form for Fresnel equations uses $\tilde{n} = 1 - \delta + i\beta$ directly.

**TMM for normal incidence** through layers:

For interface between medium $a$ (refractive index $\tilde{n}_a$) and medium $b$ ($\tilde{n}_b$):
$$r_{ab} = \frac{\tilde{n}_a - \tilde{n}_b}{\tilde{n}_a + \tilde{n}_b}, \quad t_{ab} = \frac{2\tilde{n}_a}{\tilde{n}_a + \tilde{n}_b}$$

Propagation through layer of thickness $d$ and index $\tilde{n}$:
$$\phi = \frac{2\pi \tilde{n} d}{\lambda}$$

Transfer matrix for each layer:
$$M_{\ell} = \frac{1}{t_{\ell,\ell+1}} \begin{pmatrix} e^{-i\phi_\ell} & r_{\ell,\ell+1} e^{-i\phi_\ell} \\ r_{\ell,\ell+1} e^{i\phi_\ell} & e^{i\phi_\ell} \end{pmatrix}$$

**Pragmatic simplification**: Since exact layer thicknesses are not specified in the problem, compute single-interface reflectivities as approximations:

$$R_{\text{substrate}} = r_{01} = \frac{1 - \tilde{n}_{\text{SiCN}}}{1 + \tilde{n}_{\text{SiCN}}} = \frac{0.0209 - 0.0059i}{1.9791 + 0.0059i}$$

$$R_{\text{Cu}} = r_{0,\text{Cu}} = \frac{1 - \tilde{n}_{\text{Cu}}}{1 + \tilde{n}_{\text{Cu}}} = \frac{0.0375 - 0.0613i}{1.9625 + 0.0613i}$$

Compute numerically:
- $R_p \approx 0.01055 - 0.00298i$ → $|R_p| \approx 0.01096$
- $R_{\text{Cu}} \approx 0.01910 - 0.03125i$ → $|R_{\text{Cu}}| \approx 0.03663$

**Key insight**: These are very small reflectivities (< 4%). The object amplitude is on the order of 0.01–0.04. This is critical for:
1. Probe energy scaling
2. Understanding why the reconstruction is hard (very weak signal)

**REVISED Parseval scaling**: With $|O| \approx 0.03$:
$$\sum_r |P|^2 \approx \frac{\text{mean\_total\_intensity}}{N^2 \cdot |O|^2} \approx \frac{\text{mean\_total\_intensity}}{N^2 \cdot 0.001}$$

This means the probe is much stronger than naively assumed. We should **NOT** pre-assume $|O| \approx 1$.

**Better approach**: Initialize $|O|$ from data-driven estimate. Since we don't know $|O|$ a priori, use:
$$\|P\|^2 \cdot \|O_{\text{patch}}\|^2 \propto \text{mean\_total\_intensity} / N^2$$

Set probe total energy and object amplitude jointly. Initialize $O = R_p$ everywhere (uniform substrate reflectivity), then:
$$\sum_r |P_0|^2 = \frac{\text{mean\_total\_intensity}}{N^2 \cdot |R_p|^2}$$

**Alternative pragmatic approach**: Don't try to get the absolute scaling right. Instead:
1. Initialize $O = 1$ (arbitrary units)
2. Initialize probe from mean diffraction pattern via IFT
3. Let the joint update find the correct relative scaling
4. At the end, rescale $O$ so that the substrate region has amplitude $|R_p|$

### Step 4: Initialization

#### 4.1 Object Initialization
$$O_0(\mathbf{r}) = 1.0 + 0i \quad \forall \mathbf{r}$$

Simple uniform initialization. The algorithm will converge to the correct values.

**After reconstruction, rescale**:
$$O_{\text{final}} = O \cdot \frac{|R_p|}{\text{median}(|O[{\sim}S]|)}$$

#### 4.2 Probe Initialization (CRITICAL FIX)

**Method**: Use the mean diffraction amplitude as the Fourier magnitude of the probe, with random phase:

$$P_0 = \mathcal{F}^{-1}\left\{\sqrt{\frac{1}{J}\sum_j I_j} \cdot e^{i \cdot 0}\right\}$$

This is equivalent to assuming the object is flat (which it roughly is for initialization). Using zero phase in Fourier space gives a centered, symmetric probe.

With numpy:
```python
mean_diff = np.mean(I, axis=0)  # (256, 256)
P_fourier = np.sqrt(mean_diff)  # amplitude
# Use ifft2 to get probe (assuming object ≈ 1)
P0 = np.fft.ifft2(P_fourier)  # (256, 256) complex
# Note: no fftshift needed if we keep consistent conventions
```

**BUT**: This probe includes the object's contribution. A better initialization:
```python
P0 = np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(np.sqrt(mean_diff))))
```

The key insight: with numpy's unnormalized fft2 and normalized ifft2 ($1/N^2$), the energy is automatically handled correctly by ifft2.

### Step 5: Solver — Corrected ePIE/rpPIE

#### 5.1 Standard ePIE Object Update (CRITICAL FIX)

The **standard ePIE** update (Maiden & Rodenburg 2009) is:

$$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \alpha \cdot \frac{P^*(\mathbf{r} - \mathbf{R}_j)}{\max_{\mathbf{r}'} |P(\mathbf{r}' - \mathbf{R}_j)|^2} \cdot \Delta\Psi(\mathbf{r})$$

**Key**: The denominator is a **scalar** — the maximum of $|P|^2$ over the probe window. This is NOT pointwise division.

Similarly for probe update:
$$P_{n+1}(\mathbf{r}) = P_n(\mathbf{r}) + \beta_P \cdot \frac{O^*(\mathbf{r} + \mathbf{R}_j)}{\max_{\mathbf{r}'} |O(\mathbf{r}' + \mathbf{R}_j)|^2} \cdot \Delta\Psi(\mathbf{r} + \mathbf{R}_j)$$

**However**, the paper's rpPIE (Eq. 6) uses a slightly different form:

$$O_{n+1} = O_n + \frac{|P|}{|P_{\max}|} \cdot \frac{P^*}{|P|^2 + \alpha} \cdot \beta \cdot \Delta\Psi$$

This combines a weighting factor $|P|/|P_{\max}|$ with a regularized inverse $P^*/(|P|^2 + \alpha)$. The net effect is:

$$O_{n+1} = O_n + \beta \cdot \frac{|P| \cdot P^*}{|P_{\max}| \cdot (|P|^2 + \alpha)}$$

Since $|P| \cdot P^* = |P|^2 \cdot e^{-i\angle P} \cdot e^{i\angle P} / |P| = |P| \cdot P^* $... Let me simplify:

$\frac{|P| \cdot P^*}{|P_{\max}| \cdot (|P|^2 + \alpha)} = \frac{P^*}{|P_{\max}| \cdot (|P| + \alpha/|P|)}$

**Pragmatic decision**: Use the **standard ePIE** update (scalar max denominator) since it's well-established and the previous regularized form caused issues:

```python
# Object update
P_conj = np.conj(P)
P_max_sq = np.max(np.abs(P)**2)  # SCALAR
O[ry:ry+ps, rx:rx+ps] += alpha_O * P_conj / P_max_sq * dpsi

# Probe update  
O_patch = O[ry:ry+ps, rx:rx+ps]
O_conj = np.conj(O_patch)
O_max_sq = np.max(np.abs(O_patch)**2)  # SCALAR
P += alpha_P * O_conj / O_max_sq * dpsi
```

#### 5.2 Fourier Constraint (Magnitude Replacement)

```python
psi = P * O_patch                              # exit wave
Psi = fft2(psi)                                # far-field (unnormalized)
Psi_amp = np.abs(Psi)
Psi_corrected = diff_amp[j] * Psi / (Psi_amp + eps)  # replace magnitude
psi_corrected = ifft2(Psi_corrected)           # back-propagate
dpsi = psi_corrected - psi                     # exit wave update
```

**FFT convention**: Use `np.fft.fft2` and `np.fft.ifft2` **without** fftshift, keeping everything in the natural FFT order. The diffraction patterns in the input are likely already in natural FFT order (DC at corner) OR in shifted order (DC at center). 

**CRITICAL**: Determine if input diffraction patterns have DC at center or corner:
- If DC at center (typical for experimental data display): need `ifftshift` before `fft2` operations, or equivalently, `fftshift` the measured amplitudes to match FFT output order.
- If DC at corner (natural FFT order): use directly.

**Strategy**: Try both. The correct one will converge; the wrong one won't.

**Default assumption**: Input has DC at center (common convention). So:
```python
# Convert measured amplitudes to FFT order (DC at corner)
diff_amp_fft = np.fft.ifftshift(diff_amp[j])

# Forward
Psi = np.fft.fft2(psi)  # DC at corner
Psi_amp = np.abs(Psi)
Psi_corrected = diff_amp_fft * Psi / (Psi_amp + eps)
psi_corrected = np.fft.ifft2(Psi_corrected)
```

**Alternative**: If input has DC at corner already:
```python
diff_amp_fft = diff_amp[j]  # use directly
```

**Implement both and check convergence error to decide.**

#### 5.5 Complete Algorithm

```python
def solve(input_data):
    I = np.maximum(input_data, 0)
    diff_amp = np.sqrt(I)  # (81, 256, 256)
    
    # Geometry
    ps = 256  # probe size
    obj_size = 600
    n_side = 9
    step = 40
    offset = (obj_size - (ps + (n_side-1)*step)) // 2  # 12
    
    positions = [(offset + i*step, offset + j*step) 
                 for i in range(n_side) for j in range(n_side)]
    
    # Support mask
    pad_sz, pitch, n_pads = 22, 30, 15
    arr_ext = (n_pads-1)*pitch + pad_sz  # 442
    arr_off = (obj_size - arr_ext) // 2   # 79
    S = np.zeros((obj_size, obj_size), dtype=bool)
    for m in range(n_pads):
        for n in range(n_pads):
            S[arr_off+m*pitch:arr_off+m*pitch+pad_sz,
              arr_off+n*pitch:arr_off+n*pitch+pad_sz] = True
    
    # Reflectivity
    n_SiCN = complex(0.9791, 0.0059)
    n_Cu = complex(0.9625, 0.0613)
    R_sub = (1 - n_SiCN) / (1 + n_SiCN)  # substrate
    R_cu = (1 - n_Cu) / (1 + n_Cu)        # copper
    R_p = R_sub  # prior for outside support
    
    # Object initialization: uniform
    O = np.ones((obj_size, obj_size), dtype=np.complex128)
    
    # Probe initialization from mean diffraction
    mean_I = np.mean(I, axis=0)
    mean_amp = np.sqrt(mean_I)
    # Try DC-at-center convention first
    P = np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(mean_amp)))
    P = P.astype(np.complex128)
    
    # Hyperparameters
    n_epochs = 800
    alpha_O = 1.0
    alpha_P = 1.0
    eps = 1e-8
    prior_start = 3
    prior_ramp = 20
    probe_start = 3
    
    # Determine FFT convention by trying both
    # (implemented as: try ifftshift on measured data)
    use_shift = True  # assume DC at center in input
    
    for epoch in range(n_epochs):
        order = np.random.permutation(81)
        beta_prior = 0.7
        
        for idx in order:
            ry, rx = positions[idx]
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fft2(psi)
            Psi_amp = np.abs(Psi)
            
            # Measured amplitude in FFT order
            if use_shift:
                meas = np.fft.ifftshift(diff_amp[idx])
            else:
                meas = diff_amp[idx]
            
            # Fourier magnitude replacement
            Psi_c = meas * Psi / (Psi_amp + eps)
            
            # Back-propagate
            psi_c = np.fft.ifft2(Psi_c)
            
            # Exit wave difference
            dpsi = psi_c - psi
            
            # Object update (standard ePIE: scalar max denominator)
            P_max_sq = np.max(np.abs(P)**2)
            O[ry:ry+ps, rx:rx+ps] += alpha_O * np.conj(P) * dpsi / (P_max_sq + eps)
            
            # Probe update
            if epoch >= probe_start:
                O_patch_new = O[ry:ry+ps, rx:rx+ps]
                O_max_sq = np.max(np.abs(O_patch_new)**2)
                P += alpha_P * np.conj(O_patch_new) * dpsi / (O_max_sq + eps)
        
        # --- End of epoch: apply constraints ---
        
        # Amplitude clamping to [0, 1]
        amp = np.abs(O)
        phase = np.angle(O)
        amp = np.clip(amp, 0, 1.0)
        O = amp * np.exp(1j * phase)
        
        # Reflectivity prior outside support
        if epoch >= prior_start:
            gamma = min(1.0, (epoch - prior_start) / prior_ramp)
            blend = gamma * (1 - beta_prior)
            O[~S] = (1 - blend) * O[~S] + blend * R_p
        
        # Convergence monitoring (every 50 epochs)
        if epoch % 50 == 0:
            err = 0
            for idx in range(81):
                ry, rx = positions[idx]
                psi_check = P * O[ry:ry+ps, rx:rx+ps]
                Psi_check = np.fft.fft2(psi_check)
                if use_shift:
                    meas_check = np.fft.ifftshift(diff_amp[idx])
                else:
                    meas_check = diff_amp[idx]
                err += np.sum((np.abs(Psi_check) - meas_check)**2) / np.sum(meas_check**2 + eps)
            err /= 81
            # If error is very high after 100 epochs, try flipping FFT convention
            if epoch == 100 and err > 0.5:
                use_shift = not use_shift
    
    # --- Output formatting ---
    # Remove global phase ambiguity: set substrate phase to reference
    substrate_mean = np.mean(O[~S])
    phase_ref = np.angle(substrate_mean)
    O *= np.exp(-1j * phase_ref)
    
    output = np.zeros((2, 600, 600), dtype=np.float32)
    output[0] = np.abs(O).astype(np.float32)
    output[1] = np.angle(O).astype(np.float32)
    return output
```
