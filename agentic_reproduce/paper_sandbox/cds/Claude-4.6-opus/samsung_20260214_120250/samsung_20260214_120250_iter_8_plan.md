

# Revised Plan: Periodic Structure Ptychographic Phase Retrieval (Iteration 8)

## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function (amplitude + phase encoding pad topography)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function (Airy or speckle)
- $\mathbf{R}_j$: scan position $j \in \{1, \ldots, 81\}$ on a $9 \times 9$ grid, step = 40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0, \sigma^2)$: additive Gaussian noise
- **Input**: $(81, 256, 256)$ real-valued diffraction intensities $I_j$
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Phase-Topography Relation
$$\Delta\Phi(\xi,\eta) = \frac{4\pi h(\xi,\eta)}{\lambda}, \quad h \in [1,3]\text{ nm}, \quad \lambda = 13.5\text{ nm}$$

### rpPIE Update (Eq. 6 from paper)
**Step 1** (object update per position $j$):
$$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \frac{|P(\mathbf{r}-\mathbf{R}_j)|}{|P_{\max}(\mathbf{r}-\mathbf{R}_j)|} \cdot \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{|P(\mathbf{r}-\mathbf{R}_j)|^2 + \alpha} \cdot \beta(\Psi_{c,n} - \Psi_n)$$

**Step 2** (reflectivity prior outside support $S$):
$$O_{n+1}(\mathbf{r}) = \beta_p \, O_{n+1}(\mathbf{r}) + (1-\beta_p) \, R_p, \quad \mathbf{r} \notin S$$

## 2. [Proposed Strategy]

**Algorithm**: **Sequential position-by-position ePIE/rpPIE** with the exact paper update rules, running **200 epochs** with optimized per-position operations.

### Key Changes from Failed Iteration 7

| Issue | Iter 7 (PSNR=6.59) | This Iteration (Fixed) |
|-------|---------------------|----------------------|
| **Update mode** | Mini-batch (9×9) with scatter-add averaging | **Fully sequential** — update O after each position |
| **ePIE denominator** | Global `max(|P|²)` scalar | **Per-pixel** `|P|² + α` (spatially-varying, paper Eq. 6) |
| **ePIE numerator** | `conj(P) * dpsi / (max_P² + eps)` | **`|P|/|P_max| * conj(P) / (|P|² + α) * β * dpsi`** (exact Eq. 6) |
| **Epochs** | 60 (insufficient) | **200** (with time-managed early exit) |
| **Reflectivity prior** | Weak: `blend = γ * 0.3` | **Strong**: `β_p = 0.3` → `O = 0.3*O + 0.7*R_p` outside S |
| **FFT convention** | Single test, could fail | **Try both, pick lower residual after 5 epochs** |
| **Probe update** | Mean over batch | **Standard ePIE per-position** with same denominator style |
| **Time budget** | 7.5s (too conservative) | Target ~45s (200 epochs × 81 pos × ~2.5ms/pos) |

### Time Budget Analysis (Sequential)

- Per position: extract patch (~0.01ms) + element multiply (~0.1ms) + fft2(256×256) (~0.8ms) + mag replace (~0.1ms) + ifft2 (~0.8ms) + update (~0.2ms) = **~2ms**
- Per epoch: 81 × 2ms = **~162ms**
- 200 epochs: **~32s** → safe within 60s
- Overhead (init, output, probe update): ~3s
- **Total: ~35s** — comfortable margin

## 3. [Step-by-Step Plan]

### Step 1: Data Loading & Preprocessing

```python
import numpy as np
import time

I = np.maximum(input_data, 0).astype(np.float64)  # (81, 256, 256)
diff_amp = np.sqrt(I)  # measured amplitudes
```

### Step 2: Geometry Setup

```python
ps = 256       # probe size
obj_size = 600
n_side = 9
step = 40

# Center the scan grid on the object
total_scan = ps + (n_side - 1) * step  # 256 + 320 = 576
offset = (obj_size - total_scan) // 2   # 12

positions = []
for i in range(n_side):
    for j in range(n_side):
        positions.append((offset + i * step, offset + j * step))
positions = np.array(positions)  # (81, 2)
```

### Step 3: Support Mask & Reflectivity

```python
pad_sz = 22
pitch = 30
n_pads = 15
arr_ext = (n_pads - 1) * pitch + pad_sz  # 442
arr_off = (obj_size - arr_ext) // 2       # 79

S = np.zeros((obj_size, obj_size), dtype=bool)
for m in range(n_pads):
    for n_idx in range(n_pads):
        r0 = arr_off + m * pitch
        c0 = arr_off + n_idx * pitch
        S[r0:r0+pad_sz, c0:c0+pad_sz] = True

# Substrate (SiCN) reflectivity at normal incidence
n_SiCN = complex(0.9791, -0.0059)  # n - ik convention
R_p = (1.0 - n_SiCN) / (1.0 + n_SiCN)  # Fresnel at normal incidence

# Cu pad reflectivity (for initialization)
n_Cu = complex(0.9625, -0.0613)
R_Cu = (1.0 - n_Cu) / (1.0 + n_Cu)
```

### Step 4: FFT Convention Handling — Dual Initialization

**Critical insight**: Rather than guessing the FFT convention once, we prepare both versions and test after a few iterations.

```python
# Check DC location
center_energy = np.sum(I[:, 112:144, 112:144])
corner_energy = np.sum(I[:, :32, :32]) + np.sum(I[:, :32, -32:]) + \
                np.sum(I[:, -32:, :32]) + np.sum(I[:, -32:, -32:])
dc_at_center = (center_energy > corner_energy)

if dc_at_center:
    diff_amp_fft = np.fft.ifftshift(diff_amp, axes=(-2, -1))
else:
    diff_amp_fft = diff_amp.copy()
```

### Step 5: Initialization

```python
# Object: uniform substrate reflectivity, Cu reflectivity on pads
O = np.full((obj_size, obj_size), R_p, dtype=np.complex128)
O[S] = R_Cu  # pads start at Cu reflectivity

# Probe: from mean diffraction pattern
mean_amp = np.mean(diff_amp_fft, axis=0)  # (256, 256)
P = np.fft.ifft2(mean_amp).astype(np.complex128)
# Normalize probe so max|P|=1
P /= np.max(np.abs(P))
```

### Step 6: Sequential rpPIE Solver (Core Algorithm)

This is the **exact** implementation of Eq. 6 from the paper.

```python
n_epochs = 200
beta = 1.0          # ePIE feedback parameter (object)
beta_P = 0.5        # probe feedback parameter  
alpha = 1e-3        # regularization in denominator
beta_prior = 0.3    # rpPIE: O_outside = beta_prior*O + (1-beta_prior)*R_p
eps = 1e-12

t_start = time.time()

for epoch in range(n_epochs):
    # Time check: leave 5s for output formatting
    if time.time() - t_start > 50.0:
        break
    
    # Random scan order (important for convergence)
    order = np.random.permutation(81)
    
    # Precompute probe quantities (constant within epoch before probe update)
    P_abs = np.abs(P)                    # (256, 256)
    P_max = np.max(P_abs)               # scalar
    P_conj = np.conj(P)                 # (256, 256)
    P_abs_sq = P_abs ** 2               # (256, 256)
    # Eq. 6 weight: |P|/|P_max| * P* / (|P|^2 + alpha)
    obj_weight = (P_abs / (P_max + eps)) * P_conj / (P_abs_sq + alpha)  # (256, 256)
    
    for idx in order:
        ry, rx = positions[idx]
        
        # Extract object patch
        O_patch = O[ry:ry+ps, rx:rx+ps]  # view into O
        
        # Exit wave
        psi = P * O_patch  # (256, 256)
        
        # Forward propagation
        Psi = np.fft.fft2(psi)
        
        # Fourier magnitude replacement
        Psi_amp = np.abs(Psi)
        Psi_c = diff_amp_fft[idx] * Psi / (Psi_amp + eps)
        
        # Inverse propagation
        psi_c = np.fft.ifft2(Psi_c)
        
        # Exit wave difference
        dpsi = psi_c - psi
        
        # --- Object update (Eq. 6, Step 1) ---
        O[ry:ry+ps, rx:rx+ps] += beta * obj_weight * dpsi
        
        # --- Probe update (standard ePIE) ---
        if epoch >= 2:
            O_patch_new = O[ry:ry+ps, rx:rx+ps]  # updated patch
            O_abs = np.abs(O_patch)
            O_max = np.max(O_abs)
            O_abs_sq = O_abs ** 2
            probe_weight = (O_abs / (O_max + eps)) * np.conj(O_patch) / (O_abs_sq + alpha)
            P += beta_P * probe_weight * dpsi
    
    # --- End-of-epoch constraints ---
    
    # 1. Amplitude constraint: reflection coefficient <= 1
    amp = np.abs(O)
    mask_too_large = amp > 1.0
    if np.any(mask_too_large):
        O[mask_too_large] = O[mask_too_large] / amp[mask_too_large]
    
    # 2. rpPIE reflectivity prior (Eq. 6, Step 2) — outside support
    # Ramp in over first 10 epochs for stability
    if epoch >= 1:
        gamma = min(1.0, epoch / 10.0)
        bp = beta_prior  # 0.3
        # O_outside = bp * O + (1-bp) * R_p, blended with gamma
        O[~S] = (1.0 - gamma * (1.0 - bp)) * O[~S] + gamma * (1.0 - bp) * R_p
    
    # 3. Probe normalization: prevent energy blow-up
    P_energy = np.sum(np.abs(P)**2)
    if epoch == 0:
        P_energy_init = P_energy
    elif P_energy > 4.0 * P_energy_init:
        P *= np.sqrt(P_energy_init / P_energy)
```

### Step 7: Output Formatting

```python
# Remove global phase ambiguity using substrate as reference
substrate_vals = O[~S]
# Use median for robustness
phase_ref = np.angle(np.median(np.real(substrate_vals)) + 1j * np.median(np.imag(substrate_vals)))
O *= np.exp(-1j * phase_ref)

output = np.zeros((2, 600, 600), dtype=np.float32)
output[0] = np.abs(O).astype(np.float32)
output[1] = np.angle(O).astype(np.float32)
return output
```

### Step 8: Fallback — FFT Convention Verification

If after 5 epochs the residual error is extremely high, flip the convention:

```python
# Inside the epoch loop, at epoch == 5:
if epoch == 5:
    # Compute residual for a few positions
    residual = 0.0
    for idx in range(0, 81, 20):
        ry, rx = positions[idx]
        psi_test = P * O[ry:ry+ps, rx:rx+ps]
        Psi_test = np.fft.fft2(psi_test)
        residual += np.sum((np.abs(Psi_test) - diff_amp_fft[idx])**2)
    
    # Also test flipped convention
    diff_amp_alt = np.fft.fftshift(diff_amp_fft, axes=(-2, -1)) if not dc_at_center else diff_amp_fft
    # Actually: flip means use the OTHER convention
    if dc_at_center:
        diff_amp_alt = diff_amp  # no shift
    else:
        diff_amp_alt = np.fft.ifftshift(diff_amp, axes=(-2, -1))
    
    residual_alt = 0.0
    for idx in range(0, 81, 20):
        ry, rx = positions[idx]
        psi_test = P * O[ry:ry+ps, rx:rx+ps]
        Psi_test = np.fft.fft2(psi_test)
        residual_alt += np.sum((np.abs(Psi_test) - diff_amp_alt[idx])**2)
    
    if residual_alt < 0.5 * residual:
        diff_amp_fft = diff_amp_alt
        # Reset object and re-initialize
        O = np.full((obj_size, obj_size), R_p, dtype=np.complex128)
        O[S] = R_Cu
```

### Step 9: Loss Function (Conceptual)

$$\mathcal{E} = \frac{1}{J}\sum_{j=1}^{81} \frac{\left\|\left|\mathcal{F}\{P \cdot O_j\}\right| - \sqrt{I_j}\right\|_2^2}{\left\|\sqrt{I_j}\right\|_2^2}$$

Not computed during reconstruction to save time. The ePIE updates implicitly minimize this.

---

## Summary of Critical Fixes

1. **Sequential updates** (not batched): Object is modified after EVERY scan position, which is fundamental to ePIE convergence.
2. **Exact Eq. 6 denominator**: Per-pixel `|P|/|P_max| * P* / (|P|² + α)` instead of global `P* / max(|P|²)`.
3. **200 epochs** with time-managed early exit at 50s.
4. **Strong reflectivity prior**: `β_p = 0.3` means 70% of R_p is enforced outside support (after ramp-in).
5. **FFT convention verification** at epoch 5 with automatic correction.
6. **Physically meaningful initialization**: O initialized to substrate/Cu reflectivities, not uniform ones.
