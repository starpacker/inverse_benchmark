

# Revised Plan: Periodic Structure Ptychographic Phase Retrieval (Iteration 9)

---

## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT; $n \sim \mathcal{N}(0,\sigma^2)$
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Phase-Topography
$$\Delta\Phi(\xi,\eta) = \frac{4\pi h(\xi,\eta)}{\lambda}, \quad h \in [1,3]\text{ nm}, \quad \lambda = 13.5\text{ nm}$$

---

## 2. [Proposed Strategy]

**Algorithm**: **Momentum-accelerated rpPIE (mPIE)** with RAAR-style Fourier constraint, aggressive prior enforcement from epoch 0, GPU-free optimized NumPy with precomputed weights.

| **Fourier constraint** | Simple magnitude replacement | **RAAR**: $\hat{\Psi} = \frac{1}{2}\beta_R(\hat{R}_S\hat{R}_M + I)\Psi + (1-\beta_R)\hat{R}_M\Psi$ simplified to weighted blend |

| **Regularization** | None on object | **TV-lite smoothing** every 20 epochs on phase |
| **Probe update** | Start at epoch 2 | **Start at epoch 5**, gentler update rate β_P=0.3 |
| **Object update weight** | Full Eq. 6 | Eq. 6 with **damped β=0.8** to prevent oscillation |
| **Time management** | 50s hard limit | **55s limit**, ~35s for 150 epochs |

### Step 1: Data Loading & FFT Convention Detection (Epoch 0)

```python
import numpy as np
import time

t_start = time.time()

I = np.maximum(input_data, 0).astype(np.float64)  # (81, 256, 256)
diff_amp = np.sqrt(I)

# FFT convention: check DC location
ps = 256
center_region = slice(ps//2 - 32, ps//2 + 32)
center_energy = np.mean(I[:, center_region, center_region])
corner_energy = (np.mean(I[:, :32, :32]) + np.mean(I[:, :32, -32:]) +
                 np.mean(I[:, -32:, :32]) + np.mean(I[:, -32:, -32:])) / 4
dc_centered = (center_energy > corner_energy)

if dc_centered:
    diff_amp_work = np.fft.ifftshift(diff_amp, axes=(-2, -1))
else:
    diff_amp_work = diff_amp.copy()
```

### Step 3: Material Reflectivities (Fresnel at Normal Incidence)

```python
# Fresnel reflection: r = (1 - n_complex) / (1 + n_complex) where n_complex = n - ik
# CXRO convention: n_complex = 1 - delta - i*beta, so n_cxro = 1-delta, k_cxro = beta
# Given: SiCN (n=0.9791, k=0.0059) → n_complex = 0.9791 - 0.0059j
n_SiCN = 0.9791 - 0.0059j
n_Cu   = 0.9625 - 0.0613j

R_sub = (1.0 - n_SiCN) / (1.0 + n_SiCN)  # substrate reflectivity (complex)
R_Cu  = (1.0 - n_Cu) / (1.0 + n_Cu)       # Cu pad reflectivity (complex)

R_p = R_sub  # reflectivity prior for outside support
```

### Step 4: Probe Initialization (Gaussian Airy Model)

```python
# Better than ifft2(mean_amp): use a Gaussian approximation to Airy
# Airy first zero at 1.22*lambda*f/D. For our setup, estimate from data.
# Use the autocorrelation width of mean diffraction pattern to estimate probe size.

# Approach: Start with Gaussian probe, let probe update refine it
yy, xx = np.mgrid[:ps, :ps] - ps/2
sigma_probe = 40.0  # pixels, ~1 micron beam — reasonable for EUV
P = np.exp(-(xx**2 + yy**2) / (2 * sigma_probe**2)).astype(np.complex128)
P /= np.max(np.abs(P))

# Also prepare the data-driven probe for comparison
mean_amp = np.mean(diff_amp_work, axis=0)
P_data = np.fft.ifft2(mean_amp)
P_data /= np.max(np.abs(P_data))
```

### Step 5: FFT Convention Verification (Before Main Loop)

```python
def compute_residual(O_test, P_test, diff_amp_test, positions, n_test=20):
    """Compute Fourier residual for a subset of positions."""
    res = 0.0
    indices = np.linspace(0, 80, n_test, dtype=int)
    for idx in indices:
        ry, rx = positions[idx]
        psi = P_test * O_test[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - diff_amp_test[idx])**2)
    return res

# Test both conventions and both probe inits
O_init = np.full((obj_size, obj_size), R_p, dtype=np.complex128)
O_init[S] = R_Cu

# Convention A: current diff_amp_work
res_A_gauss = compute_residual(O_init, P, diff_amp_work, positions)
res_A_data  = compute_residual(O_init, P_data, diff_amp_work, positions)

# Convention B: flipped
if dc_centered:
    diff_amp_alt = diff_amp.copy()
else:
    diff_amp_alt = np.fft.ifftshift(diff_amp, axes=(-2, -1))

res_B_gauss = compute_residual(O_init, P, diff_amp_alt, positions)
res_B_data  = compute_residual(O_init, P_data, diff_amp_alt, positions)

# Pick best combination
best = np.argmin([res_A_gauss, res_A_data, res_B_gauss, res_B_data])
if best in [2, 3]:
    diff_amp_work = diff_amp_alt
if best in [1, 3]:
    P = P_data.copy()

# Initialize object
O = O_init.copy()
O_prev = O.copy()  # for momentum
```

### Step 6: Momentum-Accelerated rpPIE (mPIE) with RAAR

**Core algorithm — the critical piece that must converge fast.**

```python
n_epochs = 250
beta_obj = 0.8       # damped object update (< 1.0 prevents oscillation)
beta_probe = 0.3     # probe update strength
alpha = 1e-4         # regularization
beta_prior = 0.1     # STRONG prior: 90% R_p outside support from epoch 0
beta_raar = 0.9      # RAAR relaxation parameter
momentum = 0.3       # Nesterov momentum coefficient
eps = 1e-12

# Precompute outside-support mask indices for fast operations
outside_S = ~S

for epoch in range(n_epochs):
    if time.time() - t_start > 55.0:
        break
    
    # --- Nesterov momentum: extrapolate object ---
    if epoch >= 2:
        gamma_m = min(momentum, 0.5 * epoch / n_epochs)  # ramp up
        O_extrap = O + gamma_m * (O - O_prev)
        O_prev = O.copy()
        O = O_extrap
    else:
        O_prev = O.copy()
    
    # Precompute probe-derived weights (constant within epoch)
    P_abs = np.abs(P)
    P_max_val = np.max(P_abs) + eps
    P_conj = np.conj(P)
    P_abs_sq = P_abs ** 2
    # Eq. 6 weight
    obj_weight = (P_abs / P_max_val) * P_conj / (P_abs_sq + alpha)
    
    # Random scan order
    order = np.random.permutation(81)
    
    for idx in order:
        ry, rx = positions[idx]
        
        O_patch = O[ry:ry+ps, rx:rx+ps].copy()
        
        # Exit wave
        psi = P * O_patch
        
        # Forward: FFT
        Psi = np.fft.fft2(psi)
        
        # RAAR Fourier constraint (more robust than simple replacement)
        Psi_amp = np.abs(Psi) + eps
        ratio = diff_amp_work[idx] / Psi_amp
        
        # RAAR: Psi_c = beta_raar * (2*P_M - I)Psi + (1-beta_raar)*P_M*Psi
        # where P_M replaces magnitude: P_M(Psi) = measured_amp * Psi/|Psi|
        Psi_M = ratio * Psi  # magnitude-replaced
        Psi_c = beta_raar * (2 * Psi_M - Psi) + (1 - beta_raar) * Psi_M
        # Simplifies to: Psi_c = (2*beta_raar + (1-beta_raar)) * Psi_M - beta_raar * Psi
        # = (1 + beta_raar) * Psi_M - beta_raar * Psi
        
        # Backward: IFFT
        psi_c = np.fft.ifft2(Psi_c)
        
        dpsi = psi_c - psi
        
        # Object update (Eq. 6 Step 1 with damping)
        O[ry:ry+ps, rx:rx+ps] = O_patch + beta_obj * obj_weight * dpsi
        
        # Probe update (after epoch 5 for stability)
        if epoch >= 5:
            O_patch_abs = np.abs(O_patch)
            O_max_val = np.max(O_patch_abs) + eps
            O_conj = np.conj(O_patch)
            O_abs_sq = O_patch_abs ** 2
            probe_weight = (O_patch_abs / O_max_val) * O_conj / (O_abs_sq + alpha)
            P += beta_probe * probe_weight * dpsi
    
    # --- End-of-epoch constraints ---
    
    # 1. rpPIE reflectivity prior (Eq. 6 Step 2) — STRONG, from epoch 0
    bp = beta_prior  # 0.1
    O[outside_S] = bp * O[outside_S] + (1.0 - bp) * R_p
    
    # 2. Amplitude clipping: |O| <= 1
    amp = np.abs(O)
    too_large = amp > 1.0
    if np.any(too_large):
        O[too_large] *= 1.0 / amp[too_large]
    
    # 3. TV-lite smoothing on phase every 25 epochs (mild regularization)
    if epoch > 0 and epoch % 25 == 0 and epoch < 200:
        phase = np.angle(O)
        amp_o = np.abs(O)
        # Simple 3x3 average on phase (only inside support to preserve edges)
        from scipy.ndimage import uniform_filter
        phase_smooth = uniform_filter(phase, size=3)
        # Blend: 80% original + 20% smoothed, only on pads
        blend = 0.8
        phase_new = phase.copy()
        phase_new[S] = blend * phase[S] + (1 - blend) * phase_smooth[S]
        O_new = amp_o * np.exp(1j * phase_new)
        O[S] = O_new[S]
    
    # 4. Probe normalization
    P_en = np.sum(np.abs(P)**2)
    if epoch == 0:
        P_en_ref = P_en
    elif P_en > 3.0 * P_en_ref:
        P *= np.sqrt(P_en_ref / P_en)
```

### Step 7: Post-Processing & Output

```python
# Remove global phase ambiguity: align substrate phase to expected R_p phase
substrate_phase = np.angle(O[outside_S])
expected_phase = np.angle(R_p)
phase_offset = np.median(substrate_phase) - expected_phase
O *= np.exp(-1j * phase_offset)

# Final output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(O).astype(np.float32)
output[1] = np.angle(O).astype(np.float32)
return output
```

## Summary of Critical Improvements

1. **RAAR Fourier constraint** instead of simple magnitude replacement → better convergence for noisy periodic structures
2. **Nesterov momentum** (γ=0.3, ramped) → 2-3× faster convergence
3. **Strong prior from epoch 0**: β_prior=0.1 → 90% substrate reflectivity enforced outside support every epoch
4. **Damped object update** β=0.8 → prevents oscillation that β=1.0 caused
5. **Gaussian probe initialization** → more physical than ifft2(mean_amp)
6. **FFT convention tested at epoch 0** with both probe options → no wasted iterations
7. **TV-lite phase smoothing** every 25 epochs → regularizes noisy phase
8. **250 epoch budget** with 55s time limit → ~40s actual runtime, plenty of margin
9. **Probe update delayed to epoch 5** with gentle β_P=0.3 → object converges first before probe refinement
