

# Revised Plan: Coherent Diffractive Imaging Phase Retrieval (Iteration 11)

## [Problem Formulation]

### Hard Constraints
- **Input**: `(81, 256, 256)` — 81 ptychographic diffraction intensity patterns, each 256×256
- **Output**: `(2, 600, 600)` — channel 0 = magnitude, channel 1 = phase of complex object

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n, \quad j = 1, \ldots, 81$$

### Root Cause Analysis of ALL Past Failures

| Iter | PSNR | SSIM | Root Cause |
|------|------|------|------------|
| 7 | 8.73 | 0.28 | Wrong probe scaling/normalization |
| 8 | 7.37 | 0.13 | Wrong probe energy, safety-net blending |
| 9 | — | — | TIMEOUT (300K+ FFTs) |
| 10 | 4.39 | 0.30 | Insufficient iterations (150), narrow probe search (3 candidates), fragile phase referencing, no seed |
| **Baseline** | **10.97** | **0.71** | Constant predictor — we must beat this |

**The baseline (constant predictor) gets PSNR=10.97, SSIM=0.71.** This means the ground truth is relatively smooth/uniform, and our reconstructions have been introducing MORE error than a flat guess. The fundamental problem is that our ptychographic reconstruction is diverging or producing artifacts that are worse than no reconstruction at all.

### Critical Insight: Why Baseline Beats Us

A constant predictor achieving PSNR=10.97 and SSIM=0.71 means the object is mostly uniform with small variations. The copper pad array has:
- Substrate reflectivity as the dominant feature (~64% area)
- Pad reflectivity slightly different
- Phase variations of only 0.93–2.8 radians (small compared to 2π)

Our reconstructions have been introducing large-scale artifacts (wrong probe, wrong FFT convention, insufficient convergence) that create more error than a flat image. **We need to ensure our reconstruction at minimum converges to something reasonable, and if it doesn't, we should fall back gracefully.**

## [Proposed Strategy]

### Algorithm: ePIE (Extended Ptychographic Iterative Engine) with:
1. **Fixed random seed** for reproducibility
2. **Spectral initialization** — back-projection estimate instead of all-ones
3. **Robust probe search** — 7 candidates × 30 quick iterations
4. **Conservative ePIE** — simpler than rPIE, more robust
5. **Probe power constraint** — prevent probe from absorbing object features
6. **Graceful fallback** — if Fourier error doesn't decrease, output the best partial result
7. **Minimal post-processing** — no TV denoising (it was destroying signal), simple median-based phase referencing
8. **Budget-aware iteration count** — 300 main iterations with patience 80

### Budget Calculation
- Probe search: 7 × 30 × 81 × 2 = 34,020 FFTs (~70s)
- Main: 300 × 81 × 2 = 48,600 FFTs (~100s)
- Total: ~82,620 FFTs (~170s) — should fit in ~3-4 min

### Step 1: Data Loading, Seed, Analytical DC Detection

```python
import numpy as np
np.random.seed(42)  # CRITICAL: reproducibility

data = np.load("input.npy")  # (81, 256, 256)
amplitudes = np.sqrt(np.maximum(data, 0.0))

# Analytical DC detection
avg_pattern = np.mean(data, axis=0)
peak_idx = np.unravel_index(np.argmax(avg_pattern), avg_pattern.shape)

# Robust detection: check if peak is near center or corner
center_region = avg_pattern[112:144, 112:144].sum()
corner_region = (avg_pattern[:16, :16].sum() + avg_pattern[:16, -16:].sum() +
                 avg_pattern[-16:, :16].sum() + avg_pattern[-16:, -16:].sum())

use_fftshift = center_region > corner_region

mean_total_intensity = np.mean(np.sum(data, axis=(1, 2)))
```

### Step 2: FFT Convention

```python
def forward_prop(x):
    if use_fftshift:
        return np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(x)))
    else:
        return np.fft.fft2(x)

def backward_prop(X):
    if use_fftshift:
        return np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(X)))
    else:
        return np.fft.ifft2(X)
```

### Step 3: Scan Positions (9×9 grid, centered)

```python
N_scan = 9
probe_size = 256
obj_size = 600

# Compute step to cover the object uniformly
scan_step = (obj_size - probe_size) / (N_scan - 1)  # = 43.0

positions = []
for iy in range(N_scan):
    for ix in range(N_scan):
        ry = int(round(iy * scan_step))
        rx = int(round(ix * scan_step))
        ry = min(ry, obj_size - probe_size)
        rx = min(rx, obj_size - probe_size)
        positions.append((ry, rx))
```

### Step 4: Probe Construction with Power Constraint

```python
Y, X = np.mgrid[-128:128, -128:128]
R2 = (X**2 + Y**2).astype(np.float64)

N_pix = probe_size ** 2

def make_probe(sigma):
    P = np.exp(-R2 / (2.0 * sigma**2)).astype(np.complex128)
    # Normalize probe so that |F{P*O}|^2 sums match measured intensity
    # With O ≈ 1, we need sum(|P|^2) * N_pix ≈ mean_total_intensity (Parseval)
    # Actually: sum(|F{P}|^2) = N_pix * sum(|P|^2) by Parseval for unnormalized FFT
    # And sum(I_j) ≈ sum(|F{P*O_patch}|^2) ≈ N_pix * sum(|P|^2 * |O_patch|^2)
    # With |O| ≈ 1: target sum(|P|^2) = mean_total_intensity / N_pix
    current_energy = np.sum(np.abs(P)**2)
    target_energy = mean_total_intensity / N_pix
    P *= np.sqrt(target_energy / current_energy)
    return P

def constrain_probe_power(P, target_energy):
    """Soft probe power constraint to prevent probe from absorbing object features."""
    current_energy = np.sum(np.abs(P)**2)
    if current_energy > 0:
        ratio = np.sqrt(target_energy / current_energy)
        # Soft constraint: blend toward target
        P *= (0.9 + 0.1 * ratio)
    return P

probe_target_energy = mean_total_intensity / N_pix
```

### Step 5: Spectral Initialization (Back-Projection)

```python
def compute_backprojection_init():
    """Initialize object from averaged back-projections instead of all-ones."""
    O_num = np.zeros((obj_size, obj_size), dtype=np.complex128)
    O_den = np.zeros((obj_size, obj_size), dtype=np.float64)
    
    P_init = make_probe(40)  # Use moderate probe for initialization
    
    for j in range(81):
        ry, rx = positions[j]
        # Back-project: use amplitude as rough estimate
        # Inverse FT of amplitude (with zero phase) gives rough real-space estimate
        rough_exit = backward_prop(amplitudes[j].astype(np.complex128))
        
        P_abs = np.abs(P_init)
        mask = P_abs > 0.01 * np.max(P_abs)
        
        O_patch = np.zeros((probe_size, probe_size), dtype=np.complex128)
        O_patch[mask] = rough_exit[mask] / P_init[mask]
        
        O_num[ry:ry+probe_size, rx:rx+probe_size] += np.conj(P_init) * rough_exit
        O_den[ry:ry+probe_size, rx:rx+probe_size] += np.abs(P_init)**2
    
    O_den_safe = np.maximum(O_den, 1e-12 * np.max(O_den))
    O_init = O_num / O_den_safe
    
    # Normalize: set mean magnitude to ~1
    well_covered = O_den > 0.1 * np.max(O_den)
    if np.sum(well_covered) > 0:
        mean_mag = np.mean(np.abs(O_init[well_covered]))
        if mean_mag > 0:
            O_init /= mean_mag
    
    return O_init
```

### Step 6: Probe Selection (7 candidates × 30 iterations)

```python
def quick_ePIE(P_init, n_iter=30):
    """Quick ePIE reconstruction for probe selection."""
    O = np.ones((obj_size, obj_size), dtype=np.complex128)
    P = P_init.copy()
    beta_obj = 1.0
    beta_probe = 1.0
    
    final_err = np.inf
    for ep in range(n_iter):
        order = np.random.permutation(81)
        epoch_err = 0.0
        
        for idx in range(81):
            j = order[idx]
            ry, rx = positions[j]
            O_patch = O[ry:ry+probe_size, rx:rx+probe_size]
            
            psi = P * O_patch
            Psi = forward_prop(psi)
            mag = np.abs(Psi)
            
            Psi_c = amplitudes[j] * Psi / (mag + 1e-12)
            epoch_err += np.sum((mag - amplitudes[j])**2)
            
            psi_c = backward_prop(Psi_c)
            dpsi = psi_c - psi
            
            # ePIE updates (simpler, more robust than rPIE)
            P_conj = np.conj(P)
            P_abs2_max = np.max(np.abs(P)**2)
            O[ry:ry+probe_size, rx:rx+probe_size] += beta_obj * P_conj * dpsi / (P_abs2_max + 1e-12)
            
            if ep >= 3:
                O_conj = np.conj(O_patch)
                O_abs2_max = np.max(np.abs(O_patch)**2)
                P += beta_probe * O_conj * dpsi / (O_abs2_max + 1e-12)
        
        total_energy = np.sum(amplitudes**2)
        final_err = epoch_err / total_energy
    
    return final_err

sigma_candidates = [10, 20, 30, 40, 55, 75, 100]
best_sigma = 40
best_err = np.inf

for sigma in sigma_candidates:
    P_test = make_probe(sigma)
    err = quick_ePIE(P_test, n_iter=30)
    print(f"  sigma={sigma}, err={err:.6f}")
    if err < best_err:
        best_err = err
        best_sigma = sigma

print(f"Best sigma: {best_sigma}, error: {best_err:.6f}")
```

### Step 7: Main ePIE Reconstruction (300 iterations, patience 80)

```python
np.random.seed(42)  # Reset seed for reproducibility

P = make_probe(best_sigma)
# Try spectral initialization
O = compute_backprojection_init()
# Fallback if back-projection produces nonsense
if np.any(np.isnan(O)) or np.any(np.isinf(O)):
    O = np.ones((obj_size, obj_size), dtype=np.complex128)

best_O = O.copy()
best_P = P.copy()
best_fourier_error = np.inf
best_epoch = 0

N_iter = 300
beta_obj = 1.0
beta_probe = 1.0
probe_update_start = 5
patience = 80

error_history = []

for epoch in range(N_iter):
    order = np.random.permutation(81)
    epoch_err = 0.0
    
    for idx in range(81):
        j = order[idx]
        ry, rx = positions[j]
        O_patch = O[ry:ry+probe_size, rx:rx+probe_size].copy()
        
        psi = P * O_patch
        Psi = forward_prop(psi)
        mag = np.abs(Psi)
        
        # Modulus replacement
        Psi_c = amplitudes[j] * Psi / (mag + 1e-12)
        epoch_err += np.sum((mag - amplitudes[j])**2)
        
        psi_c = backward_prop(Psi_c)
        dpsi = psi_c - psi
        
        # ePIE object update: O += beta * P* * dpsi / max(|P|^2)
        P_conj = np.conj(P)
        P_abs2_max = np.max(np.abs(P)**2)
        O[ry:ry+probe_size, rx:rx+probe_size] += beta_obj * P_conj * dpsi / (P_abs2_max + 1e-12)
        
        # ePIE probe update
        if epoch >= probe_update_start:
            O_conj = np.conj(O_patch)
            O_abs2_max = np.max(np.abs(O_patch)**2)
            P += beta_probe * O_conj * dpsi / (O_abs2_max + 1e-12)
    
    # Apply probe power constraint every 10 epochs
    if epoch % 10 == 0 and epoch >= probe_update_start:
        P = constrain_probe_power(P, probe_target_energy)
    
    total_energy = np.sum(amplitudes**2)
    rel_err = epoch_err / total_energy
    error_history.append(rel_err)
    
    if rel_err < best_fourier_error:
        best_fourier_error = rel_err
        best_O = O.copy()
        best_P = P.copy()
        best_epoch = epoch
    
    # Early stopping
    if epoch - best_epoch > patience and epoch > 50:
        print(f"Early stopping at epoch {epoch}, best was {best_epoch}")
        break

O_final = best_O.copy()
print(f"Best Fourier error: {best_fourier_error:.6f} at epoch {best_epoch}")
```

### Step 8: Quality Check and Fallback Strategy

# Build illumination coverage map
illum_weight = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(81):
    ry, rx = positions[j]
    illum_weight[ry:ry+probe_size, rx:rx+probe_size] += np.abs(best_P)**2

# Mask for well-illuminated region
illum_thresh = 0.1 * np.max(illum_weight) if np.max(illum_weight) > 0 else 0
well_lit = illum_weight > illum_thresh

# Quality check: if reconstruction looks unreasonable, fall back
mag_final = np.abs(O_final)
phase_final = np.angle(O_final)

# Check if magnitude has reasonable statistics in well-lit region
if np.sum(well_lit) > 0:
    mag_well = mag_final[well_lit]
    mag_std = np.std(mag_well)
    mag_mean = np.mean(mag_well)
    coeff_var = mag_std / (mag_mean + 1e-12)
    
    # If coefficient of variation is extremely high, reconstruction may have diverged
    if coeff_var > 5.0 or mag_mean < 1e-10:
        print(f"WARNING: Reconstruction may have diverged (CV={coeff_var:.2f}). Using fallback.")
        # Fallback: use a gentler normalization
        mag_final = mag_final / (np.max(mag_final) + 1e-12)

### Step 9: Phase Referencing — Simple Median Subtraction

**Key change from Iter 10**: Use simple, robust median-based phase referencing instead of fragile Otsu + histogram + sign-flip logic.

```python
if np.sum(well_lit) > 0:
    phase_well = phase_final[well_lit]
    
    # The substrate occupies ~64% of the area (duty cycle 0.6 for pads)
    # The median phase in the well-lit region should correspond to substrate
    substrate_phase = np.median(phase_well)
    phase_final = phase_final - substrate_phase
    
    # Wrap to [-pi, pi]
    phase_final = np.arctan2(np.sin(phase_final), np.cos(phase_final))
    
    # Phase sign convention: ΔΦ = 4πh/λ > 0 for recessed pads
    # Pads have DIFFERENT reflectivity than substrate
    # We check: the minority population (pads, ~36% area) should have positive phase
    # Use the 75th percentile vs 25th percentile
    p25 = np.percentile(phase_well - substrate_phase, 25)
    p75 = np.percentile(phase_well - substrate_phase, 75)
    
    # The distribution should be bimodal: substrate near 0, pads at positive values
    # If the tail is more negative than positive, flip
    if abs(p25) > abs(p75) and p25 < -0.1:
        phase_final = -phase_final

# Zero out poorly illuminated regions
phase_final[~well_lit] = 0.0
mag_final[~well_lit] = 0.0
```
