## [Problem Formulation]

### Hard Constraints
- **Input**: `(81, 256, 256)` — 81 ptychographic diffraction intensity patterns, each 256×256
- **Output**: `(2, 600, 600)` — complex object as 2 channels (real/imaginary or magnitude/phase)

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n, \quad j = 1, \ldots, 81$$

where $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$, $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$, and $\mathcal{F}$ is the 2D DFT.

### Root Cause Analysis of ALL Previous Failures

| Iter | PSNR | Key Failure |
|------|------|-------------|
| 3 | 6.54 | Wrong output format, no convergence monitoring |
| 4 | 1.67 | Anti-correlated output — completely wrong format/normalization |
| 5 | 4.79 | Probe scaling wrong (Parseval factor), algorithm diverged (error went from 0.205 back to 0.54), used FINAL state instead of BEST state |

**Critical fixes identified from feedback:**

1. **Parseval scaling**: `sum(|fft2(x)|^2) = N * sum(|x|^2)` where `N = 256*256 = 65536`. Previous code was off by factor of √N = 256.
2. **Must save and restore BEST reconstruction** (at minimum Fourier error), not the final one.
3. **Alpha schedule too aggressive** — keep α ≥ 0.2 to prevent instability.
4. **Output format/normalization still unknown** — must try multiple formats systematically.

---

## [Proposed Strategy]

### Algorithm: ePIE/rPIE with Best-State Tracking, Correct Parseval Scaling, and Multi-Format Output

**Core philosophy**: Get convergence right first, then handle output format.

**Key algorithmic changes from Iter 5:**
1. Fix Parseval scaling: `probe_energy = mean_total_intensity / N_pixels`
2. Track best reconstruction state (minimum Fourier error) and RESTORE it at the end
3. Keep α = 0.3 constant (no annealing — annealing caused divergence)
4. More iterations (800) with early stopping if error plateaus
5. Try BOTH output formats and BOTH normalizations, save the most physically plausible one
6. Add mild support constraint based on object magnitude

## [Step-by-Step Plan]

### Step 1: Data Loading and FFT Convention Detection

```python
import numpy as np

data = np.load("input.npy")  # (81, 256, 256)
amplitudes = np.sqrt(np.maximum(data, 0.0))  # (81, 256, 256)

# Determine DC location
avg_pattern = np.mean(data, axis=0)
center_val = avg_pattern[128, 128]
corner_val = avg_pattern[0, 0]

# Choose FFT convention based on DC location
if center_val > corner_val * 5:
    # DC at center — data is fftshifted
    use_shift = True
else:
    use_shift = False

def forward_prop(x):
    if use_shift:
        return np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(x)))
    else:
        return np.fft.fft2(x)

def backward_prop(X):
    if use_shift:
        return np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(X)))
    else:
        return np.fft.ifft2(X)
```

**Rationale for ifftshift before fft2**: If the data has DC at center, we need the full shift-fft-shift cycle to maintain consistency. Previous iterations may have had an asymmetric shift issue.

### Step 3: Probe Initialization with CORRECT Parseval Scaling

**The critical fix**: For numpy's `fft2`, Parseval's theorem gives:
$$\sum |F\{x\}|^2 = N \cdot \sum |x|^2$$
where $N = 256 \times 256 = 65536$.

The measured intensity for pattern $j$ is:
$$\sum I_j(\mathbf{k}) = \sum |\mathcal{F}\{P \cdot O_j\}|^2 = N \cdot \sum |P \cdot O_j|^2$$

If $|O| \approx 1$ on average, then:
$$\sum I_j \approx N \cdot \sum |P|^2$$

So: $\text{probe\_energy} = \sum |P|^2 \approx \frac{\text{mean\_total\_intensity}}{N}$

```python
N_pixels = probe_size * probe_size  # 65536
mean_total_intensity = np.mean(np.sum(data, axis=(1, 2)))

# Gaussian probe initialization
Y, X = np.mgrid[-128:128, -128:128]
R2 = (X**2 + Y**2).astype(np.float64)

# Try multiple sigma values, pick best after short run
best_probe = None
best_err = np.inf

for sigma in [20, 30, 40, 55]:
    P_test = np.exp(-R2 / (2.0 * sigma**2)).astype(np.complex128)
    
    # Correct Parseval scaling
    current_energy = np.sum(np.abs(P_test)**2)
    target_energy = mean_total_intensity / N_pixels
    scale = np.sqrt(target_energy / current_energy)
    P_test *= scale
    
    # Quick 10-iteration ePIE to test
    O_test = np.ones((600, 600), dtype=np.complex128)
    err = _run_epie(O_test, P_test, 10)  # returns final Fourier error
    
    if err < best_err:
        best_err = err
        best_probe = P_test.copy()
        best_sigma = sigma

print(f"Best probe sigma: {best_sigma}, error: {best_err:.6e}")
```

### Step 4: Main rPIE Reconstruction with Best-State Tracking

**This is the most critical change**: We track the reconstruction state at minimum Fourier error and restore it at the end.

```python
# Initialize
O = np.ones((600, 600), dtype=np.complex128)
P = best_probe.copy()

# Best state tracking
best_O = O.copy()
best_P = P.copy()
best_fourier_error = np.inf
best_epoch = 0

# Hyperparameters
N_iter = 800
alpha = 0.3  # CONSTANT — no annealing (annealing caused divergence in Iter 5)
probe_update_start = 20

errors = []

for epoch in range(N_iter):
    order = np.random.permutation(81)
    epoch_err = 0.0
    
    for idx in range(81):
        j = order[idx]
        ry, rx = positions[j]
        
        # Extract object patch
        O_patch = O[ry:ry+256, rx:rx+256]
        
        # Exit wave
        psi = P * O_patch
        
        # Forward propagate
        Psi = forward_prop(psi)
        
        # Modulus constraint
        mag = np.abs(Psi)
        Psi_c = amplitudes[j] * Psi / (mag + 1e-12)
        
        # Fourier error
        epoch_err += np.sum((mag - amplitudes[j])**2)
        
        # Back-propagate corrected exit wave
        psi_c = backward_prop(Psi_c)
        dpsi = psi_c - psi
        
        # rPIE Object update
        P_abs2 = np.abs(P)**2
        P_max = np.max(P_abs2)
        denom_O = (1.0 - alpha) * P_abs2 + alpha * P_max
        O[ry:ry+256, rx:rx+256] += np.conj(P) * dpsi / (denom_O + 1e-12)
        
        # rPIE Probe update (delayed start)
        if epoch >= probe_update_start:
            O_patch_new = O[ry:ry+256, rx:rx+256]
            O_abs2 = np.abs(O_patch)**2  # Use OLD patch for probe update
            O_max = np.max(O_abs2)
            denom_P = (1.0 - alpha) * O_abs2 + alpha * O_max
            P += np.conj(O_patch) * dpsi / (denom_P + 1e-12)
    
    # Compute relative Fourier error
    total_amp_energy = np.sum(amplitudes**2)
    rel_err = epoch_err / total_amp_energy
    errors.append(rel_err)
    
    # *** BEST STATE TRACKING (CRITICAL FIX) ***
    if rel_err < best_fourier_error:
        best_fourier_error = rel_err
        best_O = O.copy()
        best_P = P.copy()
        best_epoch = epoch
    
    # Logging
    if epoch % 50 == 0:
        print(f"Epoch {epoch}: error={rel_err:.6e}, best={best_fourier_error:.6e} (epoch {best_epoch})")
    
    # Early stopping: if no improvement for 200 epochs, stop
    if epoch - best_epoch > 200:
        print(f"Early stopping at epoch {epoch}, best was at {best_epoch}")
        break

# *** RESTORE BEST STATE ***
O = best_O
P = best_P
print(f"Restored best state from epoch {best_epoch}, error={best_fourier_error:.6e}")
```

### Step 5: Global Phase Removal

Ptychographic reconstruction has a global phase ambiguity. We remove it by setting the mean phase to zero in the high-reflectivity (pad) regions.

```python
mag = np.abs(O)
# Use median as threshold (robust)
threshold = np.median(mag)
mask = mag > threshold

# Compute mean complex value in high-magnitude region
mean_complex = np.mean(O[mask])
global_phase = np.angle(mean_complex)
O *= np.exp(-1j * global_phase)
```

### Step 6: Output Format — Try Multiple and Select Best

**Strategy**: Since we cannot evaluate against ground truth, we generate the output in the format most consistent with the problem physics. The problem states:
- Channel 0 and Channel 1 represent a complex object
- Magnitude = reflectivity, Phase = topography

We try **both** real/imag and mag/phase, **without aggressive normalization** (which destroyed information in previous iterations).

```python
# Format A: Real / Imaginary (common in ML frameworks)
out_ri = np.zeros((2, 600, 600), dtype=np.float64)
out_ri[0] = np.real(O)
out_ri[1] = np.imag(O)

# Format B: Magnitude / Phase

# Heuristic selection:

# (pads vs background with different reflectivities)
# Phase should be relatively flat within each region

# Compute contrast metric for magnitude channel
```
