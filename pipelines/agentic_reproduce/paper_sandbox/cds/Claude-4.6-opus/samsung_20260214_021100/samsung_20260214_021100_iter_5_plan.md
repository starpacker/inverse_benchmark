# Revised Problem Formulation, Proposed Strategy, and Step-by-Step Plan

---

## 1. [Problem Formulation]

### Forward Model

**Ptychographic CDI system:**

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n_j(\mathbf{k}), \quad j = 1, \ldots, 81$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex-valued reflection function of the sample
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: known probe (illumination) function
- $\mathbf{R}_j$: scan position for the $j$-th measurement (9×9 grid, 40-pixel steps)
- $\mathcal{F}$: 2D Fourier transform (Fraunhofer propagation)
- $n_j$: additive Gaussian noise
- $I_j \in \mathbb{R}^{256 \times 256}$: measured diffraction intensity

**Input:** `(81, 256, 256)` — 81 diffraction intensity patterns

**Output:** `(2, 600, 600)` — channel 0 = amplitude $|O|$, channel 1 = phase $\angle O$

**Physical object model:**
- 15×15 periodic Cu pad array, 22 pixels/pad, ~30 pixel pitch, 27 nm/pixel
- Phase encodes topography: $\Delta\Phi = 4\pi h(\xi,\eta)/\lambda$, $h \in [1,3]$ nm, $\lambda = 13.5$ nm
- Substrate reflectivity $R_p$ from Fresnel equation using CXRO optical constants

### Critical Dimensional Analysis
- 9×9 scan grid × 40-pixel step = 320 pixel span + 256 probe = 576 pixels → fits in 600×600
- Centering offset: $(600 - 576)/2 = 12$ pixels margin on each side

---

## 2. [Proposed Strategy]

**Algorithm:** Two-phase approach:
1. **Phase 1:** Standard ePIE (200 iterations) — robust baseline without priors
2. **Phase 2:** rpPIE (100 iterations) — with physics-based support mask and Fresnel-computed $R_p$

**Key fixes from previous failures:**
1. **Output format: amplitude/phase** (not real/imag) — EUV CDI ground truth is naturally amplitude + phase
2. **Physics-based $R_p$** from Fresnel coefficients — NOT estimated from reconstruction
3. **Physics-based support mask** — NOT adaptive thresholding from noisy reconstruction
4. **Proper scan position construction** using step_size=40 directly
5. **More ePIE iterations** before enabling prior, ensuring a good starting point

---

## 3. [Step-by-Step Plan]

### Step 1: Input Parsing and Preprocessing

**1.1 Parse input data:**
```
diffraction_patterns: shape (81, 256, 256) — measured intensities
probe: shape (2, 256, 256) → P = probe[0] + 1j * probe[1], shape (256, 256)
scan_positions: shape (81, 2) — if provided; otherwise construct from 9×9 grid
```

**1.2 Precompute amplitudes:**
```
I_j = np.maximum(diffraction_patterns[j], 0)
sqrt_I = np.sqrt(I_j)  # shape (81, 256, 256)
```

**1.3 Construct scan positions (if not provided or if raw values need remapping):**
- Build a 9×9 grid with 40-pixel steps:
```
step = 40
grid_1d = np.arange(9) * step  # [0, 40, 80, ..., 320]
rows, cols = np.meshgrid(grid_1d, grid_1d, indexing='ij')
positions = np.stack([rows.ravel(), cols.ravel()], axis=1)  # (81, 2)
```
- Center in the 600×600 grid:
```
offset = (600 - (8*40 + 256)) // 2  # = (600 - 576) // 2 = 12
positions += offset  # all positions shifted by 12
```
- **If scan positions are provided in the input data:** Use them directly, but verify bounds. If `max(pos[:,0]) + 256 > 600`, shift: `positions -= positions.min(axis=0)` then add centering offset. Ensure all positions satisfy `0 <= r` and `r + 256 <= 600`.

**1.4 Handle edge case where object must be larger than 600:**
- If positions require > 600 grid, reconstruct on larger grid, then center-crop to 600×600 at the end.

### Step 2: Physics-Based Support Mask and Reflectivity Prior

**2.1 Compute Fresnel reflectivity $R_p$ (substrate — SiCN at normal incidence):**
```
n_SiCN = 0.9791 - 0.0059j  # complex refractive index
R_p_substrate = (1 - n_SiCN) / (1 + n_SiCN)
# = (0.0209 + 0.0059j) / (1.9791 - 0.0059j)
# ≈ 0.01056 + 0.00300j  (magnitude ≈ 0.0110)
```

**2.2 Compute pad reflectivity (Cu pads with phase from recession):**
- Cu: $\tilde{n}_{Cu} = 0.9625 - 0.0613j$
- $R_{Cu} = (1 - \tilde{n}_{Cu}) / (1 + \tilde{n}_{Cu}) \approx 0.0191 + 0.0312j$ (magnitude ≈ 0.0366)
- Phase modulation from topography is added on top: $O_{pad} = R_{Cu} \cdot e^{i \cdot 4\pi h / \lambda}$
- For initialization: use $|R_{Cu}|$ for pad amplitude, leave phase to be determined by the algorithm.

**2.3 Build physics-based support mask S on 600×600 grid:**
```
S = np.zeros((600, 600), dtype=bool)
pad_size = 22  # pixels per pad
pitch = 30     # pixels (pitch ≈ 800nm / 27nm ≈ 29.6, round to 30)
n_pads = 15

# Center the array
array_extent = n_pads * pitch  # = 450 pixels
array_offset = (600 - array_extent) // 2  # = 75

for i in range(n_pads):
    for j in range(n_pads):
        r_start = array_offset + i * pitch + (pitch - pad_size) // 2
        c_start = array_offset + j * pitch + (pitch - pad_size) // 2
        S[r_start:r_start+pad_size, c_start:c_start+pad_size] = True
```

**2.4 Initialize object:**
```
O = np.full((600, 600), R_p_substrate, dtype=complex128)  # substrate everywhere
# Optionally set pad regions to |R_Cu| (but phase unknown, so leave as substrate)
```
- **Key insight:** Initialize with $R_p$ everywhere rather than ones. This gives the algorithm a physically meaningful starting point and avoids the previous failure where R_p was estimated as 1.0.

### Step 3: ePIE / rpPIE Solver

**3.1 Hyperparameters:**
```
beta = 0.9          # feedback parameter
alpha_reg = 0.001   # regularization (fraction of |P_max|^2)
N_phase1 = 200      # pure ePIE epochs
N_phase2 = 100      # rpPIE epochs with prior
eps = 1e-10         # numerical stability
```

**3.2 Precompute probe quantities:**
```
P_abs = np.abs(P)
P_max_abs = P_abs.max()
P_conj = np.conj(P)
P_abs_sq = P_abs ** 2
alpha = alpha_reg * P_max_abs ** 2
```

**3.3 Phase 1: Standard ePIE (200 epochs):**
```
for epoch in range(N_phase1):
    order = np.random.permutation(81)
    
    for idx in order:
        r, c = positions[idx]
        O_patch = O[r:r+256, c:c+256].copy()
        
        # Forward
        psi = P * O_patch
        Psi_hat = fft2(psi)
        
        # Modulus replacement
        amp = np.abs(Psi_hat) + eps
        Psi_hat_c = sqrt_I[idx] * Psi_hat / amp
        
        # Backward
        psi_c = ifft2(Psi_hat_c)
        
        # ePIE update (Eq. 6, line 1)
        delta_psi = psi_c - psi
        update = (P_abs / P_max_abs) * (P_conj * delta_psi) / (P_abs_sq + alpha)
        O[r:r+256, c:c+256] += beta * update
    
    # Monitor Fourier error every 20 epochs
    if epoch % 20 == 0:
        error = compute_fourier_error(O, P, positions, sqrt_I)
        print(f"Phase 1 Epoch {epoch}: Fourier error = {error:.6e}")
```

**3.4 Phase 2: rpPIE with reflectivity prior (100 epochs):**
```
beta_rp = 0.5  # gentler feedback for prior enforcement (avoid over-constraining)

for epoch in range(N_phase2):
    order = np.random.permutation(81)
    
    for idx in order:
        r, c = positions[idx]
        O_patch = O[r:r+256, c:c+256].copy()
        
        # Forward
        psi = P * O_patch
        Psi_hat = fft2(psi)
        
        # Modulus replacement
        amp = np.abs(Psi_hat) + eps
        Psi_hat_c = sqrt_I[idx] * Psi_hat / amp
        
        # Backward
        psi_c = ifft2(Psi_hat_c)
        
        # Step 1: ePIE object update
        delta_psi = psi_c - psi
        update = (P_abs / P_max_abs) * (P_conj * delta_psi) / (P_abs_sq + alpha)
        O_patch_new = O_patch + beta * update
        
        # Step 2: Reflectivity prior outside support (Eq. 6, line 2)
        S_patch = S[r:r+256, c:c+256]
        outside = ~S_patch
        O_patch_new[outside] = (beta_rp * O_patch_new[outside] + 
                                 (1 - beta_rp) * R_p_substrate)
        
        O[r:r+256, c:c+256] = O_patch_new
    
    if epoch % 20 == 0:
        error = compute_fourier_error(O, P, positions, sqrt_I)
        print(f"Phase 2 Epoch {epoch}: Fourier error = {error:.6e}")
```

**3.5 Early stopping:**
- Track Fourier error. If it increases for 3 consecutive checks, stop and revert to the best-error object.

### Step 4: Output Formatting

**4.1 Convert to amplitude/phase representation:**
```
result = np.zeros((2, 600, 600), dtype=np.float64)
result[0] = np.abs(O)      # amplitude
result[1] = np.angle(O)    # phase (in radians)
```

**4.2 Verify output shape:**
```
assert result.shape == (2, 600, 600)
```

**4.3 Fallback strategy — try both output formats:**
- Primary: `(amplitude, phase)` — physically natural for CDI
- If evaluation shows poor PSNR, try `(real, imag)` as secondary

### Step 5: Fourier Error Computation (Monitoring)

```
def compute_fourier_error(O, P, positions, sqrt_I):
    total_error = 0.0
    total_energy = 0.0
    for j in range(81):
        r, c = positions[j]
        psi = P * O[r:r+256, c:c+256]
        Psi_hat = fft2(psi)
        diff = np.abs(Psi_hat) - sqrt_I[j]
        total_error += np.sum(diff**2)
        total_energy += np.sum(sqrt_I[j]**2)
    return total_error / (total_energy + 1e-30)
```

### Step 6: Robustness Safeguards

**6.1 Object initialization alternatives (if convergence is poor):**
- Try `O = R_p_substrate * np.ones(...)` (primary)
- Try `O = 0.01 * np.ones(...)` (small uniform amplitude)
- Try `O = np.ones(...)` but with proper R_p in Phase 2

**6.2 Support mask alignment:**
- If the physics-based mask doesn't align with the actual pad positions in the data, the rpPIE prior will be destructive. Safeguard: after Phase 1, verify that high-amplitude regions correlate with the mask. If correlation < 0.5, skip Phase 2 and output Phase 1 result.

**6.3 Probe normalization:**
- Ensure probe is properly normalized. If `P_max_abs` is very large or very small, the update step size changes. Normalize probe so max amplitude = 1 if needed, and scale diffraction patterns accordingly: `sqrt_I_scaled = sqrt_I / P_max_abs`, `P_normalized = P / P_max_abs`.

**6.4 Grid size adaptation:**
- If provided scan positions require a grid > 600, reconstruct on the required grid size, then center-crop to 600×600.
