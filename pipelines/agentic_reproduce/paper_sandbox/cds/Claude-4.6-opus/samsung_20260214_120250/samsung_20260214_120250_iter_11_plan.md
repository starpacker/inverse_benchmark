## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O}, \hat{P} = \arg\min_{O,P} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|^2$$

---

## 2. [Proposed Strategy]

**Algorithm**: **Standard ePIE** (extended Ptychographic Iterative Engine) — the simplest, most robust formulation — with **no priors, no support enforcement, no amplitude clamping** initially.

### Root Cause Analysis of ALL 10 Previous Failures

The fundamental problem across all iterations is that **every added constraint/prior has hurt performance**. The baseline at 10.97 PSNR likely uses a clean ePIE with no domain-specific priors. Our additions have consistently made things worse:

| Failure Pattern | Iterations | Impact |
|----------------|------------|--------|
| Reflectivity prior too strong | 8,9,10,11 | Forces wrong values outside support |
| Wrong support mask / pitch | All | Corrupts prior enforcement regions |
| Material-specific initialization | 8,9,10 | Creates convergence bias |
| Amplitude clamping | 9,10 | Restricts valid solutions |
| RAAR/momentum | 9 | Destabilizes convergence |
| Aggressive β_obj | 10 | Oscillation |

### Radical Simplification Strategy

**Strip everything back to bare ePIE.** No priors. No support. No material constants. No amplitude clamping. Just the core algorithm that provably converges.

### Key Changes from ALL Previous Iterations

| Aspect | Previous Iterations | This Iteration |
|--------|---------------------|----------------|
| **Reflectivity prior** | Various strengths | **NONE** |
| **Support mask** | Used for prior enforcement | **NOT USED** during reconstruction |
| **Object init** | Material-specific R_Cu, R_p | **Uniform ones** |
| **Amplitude clamping** | Clamp to 1.0 | **NONE** |
| **β_obj** | 0.8–1.0 | **0.5** (conservative, stable) |
| **Probe update** | Delayed start | **From epoch 1** |
| **FFT convention** | Complex testing | **Simple DC detection + both tested** |
| **Pitch candidates** | 4 values | **Not needed** (no support enforcement) |
| **Post-processing** | Phase alignment to R_p | **Minimal — just global phase offset removal** |

## 3. [Step-by-Step Plan]

### Step 1: Data Loading & Preprocessing

```python
import numpy as np
import time

t_start = time.time()

I = np.maximum(input_data, 0).astype(np.float64)  # (81, 256, 256)
diff_amp = np.sqrt(I)  # (81, 256, 256)

ps = 256  # probe/patch size
obj_size = 600
n_side = 9
step = 40
n_pos = 81
```

### Step 2: Scan Positions

```python
total_scan = ps + (n_side - 1) * step  # 256 + 320 = 576
offset = (obj_size - total_scan) // 2   # 12

positions = []
for i in range(n_side):
    for j in range(n_side):
        positions.append((offset + i * step, offset + j * step))
positions = np.array(positions)  # (81, 2)
```

### Step 3: FFT Convention Detection

```python
# Detect whether DC is at center or corner
center_region = slice(ps//2 - 32, ps//2 + 32)
center_energy = np.mean(I[:, center_region, center_region])
corner_energy = (np.mean(I[:, :32, :32]) + np.mean(I[:, :32, -32:]) +
                 np.mean(I[:, -32:, :32]) + np.mean(I[:, -32:, -32:])) / 4.0

dc_at_center = center_energy > corner_energy

# If DC is at center, we need ifftshift before fft2 to move DC to corner
# OR equivalently, use fftshift after fft2 and ifftshift before ifft2
# Simplest: pre-shift the measured amplitudes so they match fft2 output convention
if dc_at_center:
    diff_amp_work = np.fft.ifftshift(diff_amp, axes=(-2, -1))
else:
    diff_amp_work = diff_amp.copy()
```

### Step 4: Object & Probe Initialization — Uniform, Unbiased

```python
# Object: uniform complex ones — NO material-specific initialization
O = np.ones((obj_size, obj_size), dtype=np.complex128)

# Probe: Gaussian with reasonable width
yy, xx = np.mgrid[:ps, :ps]
yy = yy - ps / 2.0
xx = xx - ps / 2.0
sigma_probe = 35.0
P = np.exp(-(xx**2 + yy**2) / (2 * sigma_probe**2)).astype(np.complex128)

# Normalize probe so that forward model amplitudes roughly match data
# Estimate: mean measured amplitude ≈ mean |FFT(P * O_patch)|
# With O=1, |FFT(P)| should match data scale
P_fft_amp = np.abs(np.fft.fft2(P))
scale = np.mean(diff_amp_work) / (np.mean(P_fft_amp) + 1e-12)
P *= scale
```

### Step 5: Core ePIE Reconstruction — Bare Minimum

This is **standard ePIE** with the Maiden & Rodenburg (2009) update rules. No modifications.

```python
# Hyperparameters
beta_obj = 0.5      # conservative object update step
beta_probe = 0.5    # probe update step  
alpha_obj = 1e-2    # object update regularization
alpha_probe = 1e-2  # probe update regularization
eps = 1e-12

# Time budget
time_elapsed = time.time() - t_start
time_remaining = 55.0 - time_elapsed
# Estimate ~0.15s per epoch
max_epochs = int(time_remaining / 0.17)
n_epochs = min(max(max_epochs, 100), 400)

# Store initial probe energy for normalization
P_energy_init = np.sum(np.abs(P)**2)

for epoch in range(n_epochs):
    if time.time() - t_start > 54.0:
        break
    
    # Random scan order
    order = np.random.permutation(n_pos)
    
    for idx in order:
        ry, rx = positions[idx]
        
        O_patch = O[ry:ry+ps, rx:rx+ps]
        
        # Exit wave
        psi = P * O_patch
        
        # Forward: FFT
        Psi = np.fft.fft2(psi)
        
        # Fourier magnitude constraint
        Psi_amp = np.abs(Psi) + eps
        Psi_c = diff_amp_work[idx] * Psi / Psi_amp
        
        # Corrected exit wave
        psi_c = np.fft.ifft2(Psi_c)
        
        # Difference
        dpsi = psi_c - psi
        
        # ePIE Object update
        P_conj = np.conj(P)
        P_abs_sq = np.abs(P)**2
        P_max_sq = np.max(P_abs_sq) + eps
        O[ry:ry+ps, rx:rx+ps] = O_patch + beta_obj * P_conj / (P_max_sq + alpha_obj) * dpsi
        
        # ePIE Probe update (from epoch 1)
        if epoch >= 1:
            O_patch_new = O[ry:ry+ps, rx:rx+ps]
            O_conj = np.conj(O_patch_new)
            O_abs_sq = np.abs(O_patch_new)**2
            O_max_sq = np.max(O_abs_sq) + eps
            P = P + beta_probe * O_conj / (O_max_sq + alpha_probe) * dpsi
    
    # Probe energy stabilization (prevent runaway growth/decay)
    P_energy = np.sum(np.abs(P)**2)
    if P_energy > 3.0 * P_energy_init or P_energy < 0.3 * P_energy_init:
        P *= np.sqrt(P_energy_init / P_energy)
```

### Step 6: Try Both FFT Conventions — Pick Better One

Since the FFT convention detection may be wrong, run a short test with the opposite convention and compare residuals.

```python
def compute_residual(O, P, diff_amp_w, positions, ps):
    res = 0.0
    for idx in range(len(positions)):
        ry, rx = positions[idx]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - diff_amp_w[idx])**2)
    return res

# After main loop, check residual
res_main = compute_residual(O, P, diff_amp_work, positions, ps)

# Try opposite convention with short run
time_left = 54.0 - (time.time() - t_start)
if time_left > 8.0:
    if dc_at_center:
        diff_amp_alt = diff_amp.copy()
    else:
        diff_amp_alt = np.fft.ifftshift(diff_amp, axes=(-2, -1))
    
    O_alt = np.ones((obj_size, obj_size), dtype=np.complex128)
    P_alt = np.exp(-(xx**2 + yy**2) / (2 * sigma_probe**2)).astype(np.complex128)
    P_fft_amp_alt = np.abs(np.fft.fft2(P_alt))
    scale_alt = np.mean(diff_amp_alt) / (np.mean(P_fft_amp_alt) + 1e-12)
    P_alt *= scale_alt
    P_energy_init_alt = np.sum(np.abs(P_alt)**2)
    
    alt_epochs = min(int(time_left / 0.20) - 5, n_epochs)
    alt_epochs = max(alt_epochs, 30)
    
    for epoch in range(alt_epochs):
        if time.time() - t_start > 53.5:
            break
        order = np.random.permutation(n_pos)
        for idx in order:
            ry, rx = positions[idx]
            O_patch = O_alt[ry:ry+ps, rx:rx+ps]
            psi = P_alt * O_patch
            Psi = np.fft.fft2(psi)
            Psi_amp = np.abs(Psi) + eps
            Psi_c = diff_amp_alt[idx] * Psi / Psi_amp
            psi_c = np.fft.ifft2(Psi_c)
            dpsi = psi_c - psi
            P_conj = np.conj(P_alt)
            P_abs_sq = np.abs(P_alt)**2
            P_max_sq = np.max(P_abs_sq) + eps
            O_alt[ry:ry+ps, rx:rx+ps] = O_patch + beta_obj * P_conj / (P_max_sq + alpha_obj) * dpsi
            if epoch >= 1:
                O_patch_new = O_alt[ry:ry+ps, rx:rx+ps]
                O_conj = np.conj(O_patch_new)
                O_abs_sq = np.abs(O_patch_new)**2
                O_max_sq = np.max(O_abs_sq) + eps
                P_alt = P_alt + beta_probe * O_conj / (O_max_sq + alpha_probe) * dpsi
        P_energy = np.sum(np.abs(P_alt)**2)
        if P_energy > 3.0 * P_energy_init_alt or P_energy < 0.3 * P_energy_init_alt:
            P_alt *= np.sqrt(P_energy_init_alt / P_energy)
    
    res_alt = compute_residual(O_alt, P_alt, diff_amp_alt, positions, ps)
    
    if res_alt < res_main:
        O = O_alt
        P = P_alt
```

### Step 7: Post-Processing — Minimal

```python
# Remove global phase ambiguity using the most common phase value
# (the substrate/background should be the dominant region)
phase = np.angle(O)

# Use the mode of the phase histogram as reference
hist, bin_edges = np.histogram(phase.ravel(), bins=360)
mode_idx = np.argmax(hist)
phase_ref = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2.0

# Subtract reference phase so background is near zero
O *= np.exp(-1j * phase_ref)

# Output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(O).astype(np.float32)
output[1] = np.angle(O).astype(np.float32)
return output
```

## Summary: Why This Should Work

| Design Choice | Rationale |
|---------------|-----------|
| **No reflectivity prior** | All priors have hurt performance; let ePIE converge freely |
| **No support mask enforcement** | Wrong mask corrupts; no mask lets data speak |
| **Uniform object init** | No bias toward wrong material values |
| **No amplitude clamping** | Removes artificial constraint on valid solutions |
| **β_obj = 0.5, β_probe = 0.5** | Conservative, stable, proven in literature |
| **Standard ePIE update** | Simplest, most robust, well-understood convergence |
| **α = 1e-2** | Larger regularization for stability |
| **Both FFT conventions tested** | Eliminates convention ambiguity |
| **Probe from epoch 1** | Allows joint refinement earlier |
| **Minimal post-processing** | Only global phase removal, no distortion |

The key insight: **we have been over-engineering the solution**. The baseline likely uses a straightforward ePIE. Every domain-specific addition (material priors, support constraints, RAAR, momentum) has introduced errors. This iteration returns to the mathematical core of ptychography.
