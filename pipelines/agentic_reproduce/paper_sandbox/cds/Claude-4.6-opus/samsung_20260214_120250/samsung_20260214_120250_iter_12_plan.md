## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function encoding amplitude (material reflectivity) and phase ($\Delta\Phi = 4\pi h/\lambda$, topographic height)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O}, \hat{P} = \arg\min_{O,P} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|^2$$

---

## 2. [Proposed Strategy]

**Algorithm**: **Standard ePIE with per-pixel normalization** — fixing the critical bug from Iteration 11 where global max normalization was used instead of per-pixel normalization.

### Root Cause Analysis of Iteration 11 Failure (PSNR 2.92)

The catastrophic failure was caused by **three compounding errors**:

| Bug | Impact | Fix |
|-----|--------|-----|
| **Global max normalization** `conj(P)/(max(\|P\|²)+α)` instead of per-pixel `conj(P)/(|P|²+α)` | Effective step size near zero everywhere except probe peak → almost no update in most pixels | Use standard per-pixel normalization |
| **β=0.5 too conservative** combined with global-max denominator | Double suppression of update magnitude | Use β=1.0 (standard ePIE) |
| **α=1e-2 too large** | Further suppresses updates in low-probe regions | Use α = 1e-3 × max(|P|²) |

The combination meant that for a Gaussian probe with peak ~1.0, pixels at σ distance (where |P|²≈0.37) got update weight ≈ 0.5 × 0.37/(1.0 + 0.01) ≈ 0.18, while with per-pixel normalization they'd get ≈ 1.0 × 0.37/(0.37 + small) ≈ 1.0. This is a **5× difference** in effective step size, explaining the near-total failure to converge.

### Key Changes from Iteration 11

| Aspect | Iteration 11 (PSNR 2.92) | This Iteration |
|--------|--------------------------|----------------|
| **Object update denominator** | `max(|P|²) + α` (GLOBAL) | `|P|² + α` (PER-PIXEL) |
| **Probe update denominator** | `max(|O|²) + α` (GLOBAL) | `|O|² + α` (PER-PIXEL) |
| **β_obj** | 0.5 | **1.0** |
| **β_probe** | 0.5 | **1.0** |
| **α regularization** | Fixed 1e-2 | **Adaptive: 1e-3 × max(|P|²)** |
| **Probe init sigma** | 35 | **50** (wider, more coverage) |
| **Both conventions** | Sequential (wastes time) | **Quick residual test first, then full run** |

### Step 4: Run Both Conventions with Quick Validation

Rather than running full reconstruction twice, run 5 epochs with each convention and pick the one with lower residual.

```python
def init_probe_and_object(ps, obj_size, diff_amp_w):
    """Initialize probe and object."""
    O = np.ones((obj_size, obj_size), dtype=np.complex128)
    
    yy, xx = np.mgrid[:ps, :ps]
    yy = yy - ps / 2.0
    xx = xx - ps / 2.0
    
    # Try multiple probe sigmas, pick one
    sigma_probe = 50.0
    P = np.exp(-(xx**2 + yy**2) / (2 * sigma_probe**2)).astype(np.complex128)
    
    # Scale probe so forward model amplitudes match data
    P_fft_amp = np.abs(np.fft.fft2(P))
    scale = np.mean(diff_amp_w) / (np.mean(P_fft_amp) + eps)
    P *= scale
    
    return O, P

def run_epie(O, P, diff_amp_w, positions, n_epochs, t_start, t_limit):
    """Standard ePIE with per-pixel normalization."""
    beta_obj = 1.0
    beta_probe = 1.0
    
    for epoch in range(n_epochs):
        if time.time() - t_start > t_limit:
            break
        
        # Adaptive regularization based on current probe/object
        alpha_obj = 1e-3 * np.max(np.abs(P)**2)
        alpha_probe = 1e-3 * np.max(np.abs(O)**2) if epoch > 0 else 1e-1
        
        order = np.random.permutation(n_pos)
        
        for idx in order:
            ry, rx = positions[idx]
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagation
            Psi = np.fft.fft2(psi)
            
            # Fourier magnitude constraint
            Psi_amp = np.abs(Psi) + eps
            Psi_c = diff_amp_w[idx] * Psi / Psi_amp
            
            # Corrected exit wave
            psi_c = np.fft.ifft2(Psi_c)
            
            # Difference
            dpsi = psi_c - psi
            
            # === STANDARD ePIE OBJECT UPDATE (per-pixel normalization) ===
            P_conj = np.conj(P)
            P_abs_sq = np.abs(P)**2  # per-pixel!
            O[ry:ry+ps, rx:rx+ps] += beta_obj * P_conj / (P_abs_sq + alpha_obj) * dpsi
            
            # === STANDARD ePIE PROBE UPDATE (per-pixel normalization) ===
            if epoch >= 2:
                O_patch_new = O[ry:ry+ps, rx:rx+ps]
                O_conj = np.conj(O_patch_new)
                O_abs_sq = np.abs(O_patch_new)**2  # per-pixel!
                P += beta_probe * O_conj / (O_abs_sq + alpha_probe) * dpsi
    
    return O, P

def compute_residual(O, P, diff_amp_w, positions):
    res = 0.0
    count = 0
    for idx in range(len(positions)):
        ry, rx = positions[idx]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - diff_amp_w[idx])**2)
        count += 1
    return res / count

# Quick test: 5 epochs each convention
O1, P1 = init_probe_and_object(ps, obj_size, diff_amp_shifted)
O1, P1 = run_epie(O1, P1, diff_amp_shifted, positions, 5, t_start, 50.0)
res1 = compute_residual(O1, P1, diff_amp_shifted, positions)

# Opposite convention
if dc_at_center:
    diff_amp_alt = diff_amp.copy()
else:
    diff_amp_alt = np.fft.ifftshift(diff_amp, axes=(-2, -1))

O2, P2 = init_probe_and_object(ps, obj_size, diff_amp_alt)
O2, P2 = run_epie(O2, P2, diff_amp_alt, positions, 5, t_start, 50.0)
res2 = compute_residual(O2, P2, diff_amp_alt, positions)

# Pick better convention
if res1 <= res2:
    diff_amp_use = diff_amp_shifted
    O, P = O1, P1
else:
    diff_amp_use = diff_amp_alt
    O, P = O2, P2
```

### Step 6: Post-Processing

```python
# Remove global phase ambiguity
# The background/substrate region should be the dominant area
# Use phase histogram mode as reference
phase = np.angle(O)

# Weight by amplitude to focus on well-reconstructed regions
amp = np.abs(O)
mask_valid = amp > 0.1 * np.max(amp)

if np.sum(mask_valid) > 100:
    phase_valid = phase[mask_valid]
    hist, bin_edges = np.histogram(phase_valid, bins=360, range=(-np.pi, np.pi))
    mode_idx = np.argmax(hist)
    phase_ref = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2.0
else:
    phase_ref = np.median(phase)

O *= np.exp(-1j * phase_ref)

# Output
output = np.zeros((2, obj_size, obj_size), dtype=np.float32)
output[0] = np.abs(O).astype(np.float32)
output[1] = np.angle(O).astype(np.float32)
return output
```

## 4. [Loss Function & Convergence]

The implicit loss minimized by ePIE is:
$$\mathcal{L} = \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2$$

The per-pixel normalization ensures that the update step size is:
$$\Delta O(\mathbf{r}) = \beta \cdot \frac{P^*(\mathbf{r})}{|P(\mathbf{r})|^2 + \alpha} \cdot \Delta\psi(\mathbf{r})$$

This is equivalent to a **Wiener-filter-like** deconvolution step, where each pixel gets an update proportional to its SNR (probe intensity). This is fundamentally different from the global-max normalization used in Iter 11, which effectively gave all pixels the same (tiny) step size.

---

## Summary: Critical Fixes

| Fix | Mathematical Impact |
|-----|-------------------|
| **Per-pixel |P|² in denominator** | Correct gradient normalization; each pixel updates at rate ∝ local SNR |
| **β=1.0** | Standard ePIE convergence rate (proven in Maiden & Rodenburg 2009) |
| **α = 1e-3 × max(|P|²)** | Regularizes only where probe is negligible, not globally |
| **Probe update from epoch 2** | Allows object to stabilize slightly first |
| **Quick convention test** | Saves time vs full dual reconstruction |
