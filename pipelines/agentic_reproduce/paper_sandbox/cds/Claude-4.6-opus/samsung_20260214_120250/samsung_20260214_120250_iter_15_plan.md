## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function (amplitude ≤ 1 for physical reflectance)
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function (spatially compact)
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Inverse Problem

$$\hat{O} = \arg\min_{O} \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| \right\|_2^2 \quad \text{s.t. } |O(\mathbf{r})| \leq 1$$

---

## 2. [Proposed Strategy]

**Algorithm**: **rPIE (regularized PIE) with rapid FFT convention detection, pixel-wise normalization, amplitude clipping, and probe support constraint.**

### Root Cause Analysis of ALL Previous Failures

| Iteration | PSNR | Root Cause |
|-----------|------|------------|
| 11 | 2.92 | Global-max normalization killed updates |
| 12 | 6.54 | Sequential ePIE, poor probe init, too few epochs |
| 13 | 1.84 | Nesterov momentum caused divergence |
| 14 | 2.51 | **Wrong FFT convention wasted budget; scalar max normalization; no object amplitude clipping; probe spread; reg too weak** |

### Critical Fixes for This Iteration

1. **FFT convention detection FIRST** — Run 2 quick epochs with each convention, pick winner, then commit full budget
2. **Pixel-wise normalization (rPIE)** — Use $(1-\alpha)|P(\mathbf{r})|^2 + \alpha \max|P|^2$ denominator instead of just $\max|P|^2$
3. **Object amplitude clipping** — Clip $|O| \leq 1.0$ after each position update (physical reflectance constraint)
4. **Probe support constraint** — Re-apply Gaussian window to probe every epoch
5. **Stronger regularization** — $\alpha = 0.1$ in rPIE denominator
6. **No momentum** — Proven to cause divergence

### Step 4: Initialization Functions

```python
def init_probe(damp):
    """Initialize probe from average diffraction amplitude."""
    avg_amp = np.mean(damp, axis=0)  # (256, 256)
    P = np.fft.ifft2(avg_amp)  # complex (256, 256)
    # Apply support window
    P *= probe_window_fft
    # Normalize
    P = P / (np.max(np.abs(P)) + eps)
    return P

def init_object():
    """Initialize object as uniform reflector with amplitude 0.5."""
    return np.full((obj_size, obj_size), 0.5 + 0j, dtype=np.complex128)
```

### Step 5: Core rPIE Engine

```python
def run_rPIE(O, P, damp, positions, time_budget, alpha_rPIE=0.1):
    """
    rPIE (regularized Ptychographic Iterative Engine).
    
    Object update (pixel-wise normalization):
      O(r) += conj(P(r)) * dpsi(r) / ((1-α)|P(r)|² + α·max|P|²)
    
    Probe update (pixel-wise normalization):
      P(r) += conj(O_j(r)) * dpsi(r) / ((1-α)|O_j(r)|² + α·max|O_j|²)
    
    Plus: amplitude clipping on O, probe support constraint.
    """
    t0 = time.time()
    n_pos = len(positions)
    epoch = 0
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        # Randomize position ordering
        order = np.random.permutation(n_pos)
        
        for idx in range(n_pos):
            # Time check every 20 positions
            if idx % 20 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            # Extract object patch
            O_patch = O[ry:ry+ps, rx:rx+ps].copy()
            
            # Exit wave
            psi = P * O_patch
            
            # Forward propagate
            Psi = np.fft.fft2(psi)
            
            # Fourier magnitude constraint
            Psi_amp = np.abs(Psi)
            Psi_corrected = damp[j] * np.exp(1j * np.angle(Psi))
            # Handle zero amplitude gracefully
            # mask = Psi_amp > eps
            # More stable: just use angle
            
            # Inverse propagate
            psi_c = np.fft.ifft2(Psi_corrected)
            
            # Difference
            dpsi = psi_c - psi
            
            # rPIE Object update (PIXEL-WISE normalization)
            P_abs2 = np.abs(P)**2
            P_max2 = np.max(P_abs2)
            denom_O = (1.0 - alpha_rPIE) * P_abs2 + alpha_rPIE * P_max2 + eps
            O[ry:ry+ps, rx:rx+ps] += np.conj(P) * dpsi / denom_O
            
            # Object amplitude clipping (physical constraint: reflectance ≤ 1)
            O_patch_new = O[ry:ry+ps, rx:rx+ps]
            amp_patch = np.abs(O_patch_new)
            mask_clip = amp_patch > 1.0
            if np.any(mask_clip):
                O[ry:ry+ps, rx:rx+ps][mask_clip] *= 1.0 / amp_patch[mask_clip]
            
            # rPIE Probe update (PIXEL-WISE normalization)
            O_abs2 = np.abs(O_patch)**2
            O_max2 = np.max(O_abs2)
            denom_P = (1.0 - alpha_rPIE) * O_abs2 + alpha_rPIE * O_max2 + eps
            P += np.conj(O_patch) * dpsi / denom_P
        
        # Probe support constraint: re-apply window every epoch
        P *= probe_window_fft
        # Re-normalize probe
        P_maxval = np.max(np.abs(P))
        if P_maxval > eps:
            P = P / P_maxval
        
        epoch += 1
    
    return O, P, epoch

def compute_residual(O, P, damp, positions, n_sample=15):
    """Compute Fourier residual on a subset of positions."""
    subset = np.random.RandomState(42).choice(len(positions), min(n_sample, len(positions)), replace=False)
    res = 0.0
    for j in subset:
        ry, rx = positions[j]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        res += np.sum((np.abs(Psi) - damp[j])**2)
    return res / len(subset)
```

## 4. [Loss Function & Optimizer]

### Loss (Amplitude-based Fourier constraint)
$$\mathcal{L} = \sum_{j=1}^{81} \left\| \sqrt{I_j} - \left|\mathcal{F}\{P \cdot O_j\}\right| \right\|_2^2$$

### rPIE Update Rules (Maiden & Rodenburg 2009)

**Object update** at position $j$ (pixel-wise):
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{(1-\alpha)|P(\mathbf{r}-\mathbf{R}_j)|^2 + \alpha\max_{\mathbf{r}'}|P(\mathbf{r}')|^2} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$ (pixel-wise):
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \frac{O^{(n)*}(\mathbf{r}+\mathbf{R}_j)}{(1-\alpha)|O^{(n)}(\mathbf{r}+\mathbf{R}_j)|^2 + \alpha\max_{\mathbf{r}'}|O^{(n)}(\mathbf{r}')|^2} \cdot \Delta\psi_j(\mathbf{r})$$

with $\alpha = 0.1$ (regularization parameter).

**Physical constraints applied per update:**
- $|O(\mathbf{r})| \leq 1.0$ (reflectance clipping)
- $P \leftarrow P \cdot W$ every epoch (probe support window)
