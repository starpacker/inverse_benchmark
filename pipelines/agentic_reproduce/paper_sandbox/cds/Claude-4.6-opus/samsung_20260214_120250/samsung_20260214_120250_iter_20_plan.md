## 1. [Problem Formulation]

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n(\mathbf{k})$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$: complex reflection function of periodic Cu pad array
- $P(\mathbf{r}) \in \mathbb{C}^{256 \times 256}$: probe/illumination function (Airy or speckle)
- $\mathbf{R}_j$: scan position $j \in \{1,\ldots,81\}$ on $9\times9$ grid, step=40 pixels
- $\mathcal{F}$: 2D DFT
- $n \sim \mathcal{N}(0,\sigma^2)$: Gaussian noise
- **Input**: $(81, 256, 256)$ real diffraction intensities
- **Output**: $(2, 600, 600)$ — channel 0: $|O|$, channel 1: $\angle O$

### Key Physical Parameters
- Pixel size: 27 nm, pad CD: ~600 nm (22 pixels), pitch: ~800 nm (~30 pixels)
- Recession depth $h \in [1,3]$ nm → $\Delta\Phi = 4\pi h/\lambda \in [0.93, 2.8]$ rad
- Substrate reflectivity $R_p$ computed from CXRO optical constants

## 2. [Proposed Strategy]

**Algorithm**: **rPIE (regularized PIE) with rapid FFT convention detection, full time budget on winner, rpPIE support/reflectivity prior, and careful post-processing.**

### Root Cause Analysis of ALL 19 Previous Failures

The core issue across all iterations: **PSNR consistently below the trivial baseline of 10.97**. This means our reconstructions are *worse than a flat guess*. The SSIM of 0.05-0.17 vs baseline 0.71 confirms structural content is being destroyed.

**Critical insight**: A PSNR of ~5 dB means our output has ~3× the MSE of a flat constant. This suggests:
1. The reconstruction itself may be reasonable but **output format/scaling is wrong**
2. Or the FFT convention is wrong and we're getting garbage
3. Or phase alignment is destructive

**New hypothesis**: The baseline PSNR=10.97, SSIM=0.705 likely comes from a **non-trivial method** (perhaps a simple but correct approach). The fact that SSIM=0.705 for baseline suggests it captures structural content well. Our SSIM=0.05 means we have essentially zero structural correlation.

**Strategy shift**: Instead of trying to be clever, focus on **getting the basics absolutely right**:
1. Quick convention test (2 epochs each), then ALL remaining time on winner
2. Use rPIE with proper regularization parameter
3. **Critically**: Ensure output amplitude and phase are in the correct range/convention
4. Apply rpPIE reflectivity prior outside support
5. Proper phase ramp removal (linear phase gradient from position errors)

### Step 3: Probe Initialization — Use Data-Derived Probe

Instead of guessing Gaussian sigma, **estimate probe from data autocorrelation**:

```python
def init_probe_from_data(damp):
    """Initialize probe via inverse FFT of average diffraction pattern."""
    mean_dp = np.mean(damp**2, axis=0)  # average intensity
    # The autocorrelation of the exit wave = IFFT of intensity
    # For a roughly uniform object, this approximates |P|^2 autocorrelation
    autocorr = np.fft.ifft2(mean_dp)
    autocorr = np.fft.fftshift(autocorr)
    # Use sqrt of autocorrelation magnitude as probe amplitude estimate
    probe_amp = np.sqrt(np.maximum(np.abs(autocorr), 0))
    # Normalize
    probe_amp /= np.max(probe_amp) + eps
    
    # Also try simple Gaussian
    yy, xx = np.mgrid[:ps, :ps]
    cy, cx = ps/2.0, ps/2.0
    r2 = (yy - cy)**2 + (xx - cx)**2
    gauss = np.exp(-r2 / (2 * 50.0**2))
    
    # Use Gaussian - it's more reliable for initialization
    P = gauss.astype(np.complex128)
    
    # Energy matching
    mean_data_energy = np.mean(np.sum(damp**2, axis=(-2,-1)))
    obj_amp_est = 0.85  # expected reflectivity
    current_energy = ps * ps * np.sum(np.abs(P)**2) * obj_amp_est**2
    scale = np.sqrt(mean_data_energy / (current_energy + eps))
    P *= scale
    return P
```

### Step 4: Quick FFT Convention Detection (2 epochs each, ~4s total)

```python
def compute_residual(O, P, damp, positions, n_pos, ps):
    """Compute normalized Fourier residual."""
    residual = 0.0
    norm = 0.0
    for j in range(n_pos):
        ry, rx = positions[j]
        psi = P * O[ry:ry+ps, rx:rx+ps]
        Psi = np.fft.fft2(psi)
        residual += np.sum((np.abs(Psi) - damp[j])**2)
        norm += np.sum(damp[j]**2)
    return residual / (norm + eps)

def run_quick_test(damp, n_epochs=2):
    """Run n_epochs of ePIE, return residual."""
    P = init_probe_from_data(damp)
    O = 0.85 * np.ones((obj_size, obj_size), dtype=np.complex128)
    beta_O = 0.9
    
    for ep in range(n_epochs):
        order = np.random.permutation(n_pos)
        P_max2 = np.max(np.abs(P)**2)
        for idx in range(n_pos):
            j = order[idx]
            ry, rx = positions[j]
            O_patch = O[ry:ry+ps, rx:rx+ps]
            psi = P * O_patch
            Psi = np.fft.fft2(psi)
            Psi_c = damp[j] * Psi / (np.abs(Psi) + eps)
            psi_c = np.fft.ifft2(Psi_c)
            dpsi = psi_c - psi
            O[ry:ry+ps, rx:rx+ps] += beta_O * np.conj(P) * dpsi / (P_max2 + eps)
    
    return compute_residual(O, P, damp, positions, n_pos, ps)
```

### Step 5: Main rPIE Reconstruction (Full Remaining Time)

**Key changes from previous iterations:**
- **rPIE denominator**: $(|P|^2 + \alpha \cdot \max|P|^2)$ with $\alpha=0.1$ — better conditioned than scalar max
- **Probe update from epoch 5** — longer warmup
- **Support detection and rpPIE prior** every epoch after epoch 3
- **Phase ramp removal** every 5 epochs

```python
def build_support_mask(O, positions, ps, obj_size, n_pos):
    """Detect support from amplitude pattern — periodic pad array."""
    # Use illuminated region
    scan_count = np.zeros((obj_size, obj_size))
    for j in range(n_pos):
        ry, rx = positions[j]
        scan_count[ry:ry+ps, rx:rx+ps] += 1.0
    illuminated = scan_count > 0
    
    amp = np.abs(O) * illuminated
    if np.max(amp) < eps:
        return illuminated, None
    
    # Threshold to find pads (high reflectivity regions)
    amp_norm = amp / (np.max(amp) + eps)
    threshold = 0.5
    pad_mask = amp_norm > threshold
    
    return illuminated, pad_mask

def remove_phase_ramp(O, illuminated):
    """Remove linear phase ramp from reconstruction."""
    amp = np.abs(O)
    phase = np.angle(O)
    
    valid = illuminated & (amp > np.max(amp) * 0.2)
    if np.sum(valid) < 100:
        return O
    
    yy, xx = np.mgrid[:obj_size, :obj_size]
    y_valid = yy[valid].astype(np.float64)
    x_valid = xx[valid].astype(np.float64)
    ph_valid = phase[valid]
    w_valid = amp[valid]
    
    # Weighted least squares fit of phase = a*y + b*x + c
    # Using normal equations
    W = w_valid**2
    Sw = np.sum(W)
    Sy = np.sum(W * y_valid)
    Sx = np.sum(W * x_valid)
    Syy = np.sum(W * y_valid**2)
    Sxx = np.sum(W * x_valid**2)
    Sxy = np.sum(W * y_valid * x_valid)
    Sp = np.sum(W * ph_valid)
    Spy = np.sum(W * ph_valid * y_valid)
    Spx = np.sum(W * ph_valid * x_valid)
    
    A = np.array([[Syy, Sxy, Sy],
                  [Sxy, Sxx, Sx],
                  [Sy,  Sx,  Sw]])
    b = np.array([Spy, Spx, Sp])
    
    try:
        coeffs = np.linalg.solve(A, b)
        ramp = coeffs[0] * yy + coeffs[1] * xx + coeffs[2]
        O_corrected = amp * np.exp(1j * (phase - ramp))
        return O_corrected
    except np.linalg.LinAlgError:
        return O

def run_rPIE(damp, time_budget):
    """
    rPIE with rpPIE support constraint and phase ramp removal.
    """
    P = init_probe_from_data(damp)
    O = 0.85 * np.ones((obj_size, obj_size), dtype=np.complex128)
    
    # Known substrate reflectivity (approximate from CXRO data)
    # SiCN at 13.5nm: n=0.9791, k=0.0059
    # Reflectivity R = |(1-n_complex)/(1+n_complex)|^2
    # n_complex = 0.9791 + 0.0059j (for SiCN)
    n_sicn = complex(0.9791, 0.0059)
    R_substrate = abs((1 - n_sicn) / (1 + n_sicn))**2
    Rp = np.sqrt(R_substrate)  # amplitude reflectivity ~0.0105
    # Actually for reflection at normal incidence from vacuum:
    # r = (1 - n_complex)/(1 + n_complex)
    r_sub = (1 - n_sicn) / (1 + n_sicn)
    # This gives very small reflectivity. 
    # For Cu: n=0.9625, k=0.0613
    n_cu = complex(0.9625, 0.0613)
    r_cu = (1 - n_cu) / (1 + n_cu)
    # These are Fresnel coefficients for vacuum/material interface
    # |r_sub| ~ 0.01, |r_cu| ~ 0.04
    # These seem too small. The actual sample is a multilayer.
    # Better: just use the reconstruction to self-calibrate.
    # Skip explicit Rp enforcement for now, just use support constraint.
    
    t0 = time.time()
    epoch = 0
    beta_O = 0.8
    beta_P = 0.5
    alpha = 0.1  # rPIE regularization
    probe_start = 5
    
    best_O = O.copy()
    best_P = P.copy()
    best_res = 1e10
    
    while True:
        elapsed = time.time() - t0
        if elapsed > time_budget:
            break
        
        order = np.random.permutation(n_pos)
        
        for idx in range(n_pos):
            if idx % 27 == 0 and (time.time() - t0) > time_budget:
                break
            
            j = order[idx]
            ry, rx = positions[j]
            
            O_patch = O[ry:ry+ps, rx:rx+ps]
            psi = P * O_patch
            Psi = np.fft.fft2(psi)
            amp_Psi = np.abs(Psi)
            
            # Fourier magnitude replacement
            Psi_c = damp[j] * Psi / (amp_Psi + eps)
            psi_c = np.fft.ifft2(Psi_c)
            dpsi = psi_c - psi
            
            # rPIE object update
            P_abs2 = np.abs(P)**2
            P_max2 = np.max(P_abs2)
            denom_O = P_abs2 + alpha * P_max2
            O[ry:ry+ps, rx:rx+ps] += beta_O * np.conj(P) * dpsi / (denom_O + eps)
            
            # Probe update after warmup
            if epoch >= probe_start:
                O_patch_new = O[ry:ry+ps, rx:rx+ps]
                O_abs2 = np.abs(O_patch_new)**2
                O_max2 = np.max(O_abs2)
                denom_P = O_abs2 + alpha * O_max2
                P += beta_P * np.conj(O_patch_new) * dpsi / (denom_P + eps)
        
        epoch += 1
        
        # Every 5 epochs: phase ramp removal and support constraint
        if epoch % 5 == 0:
            illuminated, _ = build_support_mask(O, positions, ps, obj_size, n_pos)
            O = remove_phase_ramp(O, illuminated)
            # Zero outside illuminated
            O[~illuminated] = 0.0
        
        # Track best by residual every 10 epochs
        if epoch % 10 == 0 or (time.time() - t0) > time_budget * 0.95:
            res = 0.0
            norm = 0.0
            for j in range(min(20, n_pos)):  # subsample for speed
                ry, rx = positions[j]
                psi = P * O[ry:ry+ps, rx:rx+ps]
                Psi = np.fft.fft2(psi)
                res += np.sum((np.abs(Psi) - damp[j])**2)
                norm += np.sum(damp[j]**2)
            rel_res = res / (norm + eps)
            if rel_res < best_res:
                best_res = rel_res
                best_O = O.copy()
                best_P = P.copy()
    
    return best_O, best_P, best_res, epoch
```

## 4. [Loss Function & Optimizer]

### rPIE Update Rules

**Object update** at position $j$:
$$O^{(n+1)}(\mathbf{r}) = O^{(n)}(\mathbf{r}) + \beta_O \frac{P^*(\mathbf{r}-\mathbf{R}_j)}{|P(\mathbf{r}-\mathbf{R}_j)|^2 + \alpha \max_{\mathbf{r'}}|P(\mathbf{r'})|^2} \cdot \Delta\psi_j(\mathbf{r})$$

**Probe update** at position $j$ (after epoch 5):
$$P^{(n+1)}(\mathbf{r}) = P^{(n)}(\mathbf{r}) + \beta_P \frac{O^{(n+1)*}(\mathbf{r}+\mathbf{R}_j)}{|O^{(n+1)}(\mathbf{r}+\mathbf{R}_j)|^2 + \alpha \max_{\mathbf{r'}}|O^{(n+1)}(\mathbf{r'})|^2} \cdot \Delta\psi_j(\mathbf{r})$$

### Parameters
| Parameter | Value | Rationale |
|-----------|-------|-----------|
| $\beta_O$ | 0.8 | Slightly conservative for stability |
| $\beta_P$ | 0.5 | Moderate probe update |
| $\alpha$ | 0.1 | rPIE regularization — prevents division by near-zero |
| Probe warmup | 5 epochs | Let object stabilize well before probe updates |
| Probe init | Gaussian $\sigma=50$, energy-matched | Standard, reliable |
| Object init | amplitude=0.85, phase=0 | Reflective substrate |
| Phase ramp removal | Every 5 epochs | Prevents drift |
| Amp normalization | Percentile-99 to 1.0 | Robust scaling |
| Convention test | 2 epochs each, then full time on winner | Efficient, reliable |
