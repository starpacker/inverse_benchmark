## PROBLEM SETTING
The problem is recovering the complex-valued reflection function O(r) of a 15×15 periodic copper pad array embedded in SiCN/SiO2/Si layers, where r=[ξ,η] is a 2D coordinate vector, from far-field diffraction intensity measurements I(k) collected at a detector. Each pad occupies 22 pixels with 27 nm pixel size, yielding ~600 nm critical dimension and ~800 nm pitch (duty cycle 0.6), at wavelength λ=13.5 nm. The phase component encodes pad topography via ΔΦ=4πh(ξ,η)/λ, where recession depth h ranges 1–3 nm (maximum phase modulation 2.8 rad, no wrapping). Inputs include the measured intensity I(k), a binary support mask S defining pad geometry/periodicity, known substrate reflectivity Rp from tabulated optical constants (n,k) for Cu, SiCN, SiO2, Si, and probe function P(r) (Airy spot or speckle field). Ptychographic data use ~75% overlap scanning in 40-pixel steps. Gaussian noise contaminates diffraction patterns. The challenge is reconstructing nanometer-scale topographic phase variations of periodic structures where diffraction orders are spatially separated at the detector, degrading standard HIO/PIE convergence.

## ALGORITHMIC BLUEPRINT
The paper presents two coupled algorithms—rpIO (Eq. 5) and rpPIE (Eq. 6)—for coherent diffractive imaging of periodic Cu pad arrays in hybrid bonding metrology. The forward model computes far-field diffraction via Fraunhofer propagation (Eq. 3), valid when z ≫ 2d²/λ (Eq. 2), with detector frequencies fx = x/(λz), fy = y/(λz) (Eq. 4). The sample reflection function encodes pad topography through a phase term ΔΦ = 4πh(ξ,η)/λ (Eq. 7), where recession depth h ∈ [1, 3] nm yields phase modulation up to 2.8 rad at λ = 13.5 nm, avoiding wrapping. The rpIO algorithm (Eq. 5) iterates: inside support S, apply inverse Fourier transform of measured amplitude √I(k) with current phase estimate; outside S, enforce reflectivity prior via convex combination βO_n(r) + (1−β)R_p, where R_p is the known substrate reflectivity and β is the feedback constant. The rpPIE (Eq. 6) operates in a two-step nested structure per scan position: Step 1 updates the object via the ePIE-style rule O_{n+1}(r) = O_n(r) + [|P(r−R)|/|P_max(r−R)|] · [P*(r−R)/(|P(r−R)|² + α)] · β(Ψ_{c,n} − Ψ_n), where Ψ = P·O is the exit wave, Ψ_{c,n} is the corrected exit wave after Fourier magnitude replacement, and α is a regularization parameter; Step 2 enforces the reflectivity prior outside S: O_{n+1}(r) = βO_{n+1}(r) + (1−β)R_p. Ptychographic scanning uses ~75% overlap with 40-pixel steps. The object grid uses 27 nm pixel size, 15×15 pad array, 22 pixels per pad, duty cycle 0.6, CD ≈ 600 nm, pitch ≈ 800 nm. Two illumination modes are tested: Airy spot and structured speckle field. Gaussian noise is added to diffraction patterns. Material optical constants (n, k) for SiCN, SiO₂, Si, Cu are taken from the CXRO database at 13.5 nm.

## EXPERIMENTAL PROTOCOL
The simulated object is a 15×15 periodic array of copper pads with ~600 nm critical dimension, ~800 nm pitch (duty cycle 0.6), each pad occupying 22 pixels at 27 nm pixel size, wavelength λ=13.5 nm. Pad recession depths are randomly assigned per pad in [1, 3] nm, introducing phase modulation via ΔΦ=4πh(ξ,η)/λ (max 2.8 rad). Reflection functions are computed using the transmission matrix method with CXRO optical constants: SiCN (n=0.9791, k=0.0059), SiO₂ (n=0.9780, k=0.0108), Si (n=0.9991, k=0.0018), Cu (n=0.9625, k=0.0613). Two illuminations are tested: Airy spot and speckle field, both at normal incidence. Gaussian noise is added to far-field diffraction patterns. Ptychographic scans use 40-pixel steps with ~75% overlap. Four algorithms are compared: standard HIO (β feedback, did not converge), rpIO (Eq. 5, reflectivity prior Rp outside support), standard PIE (Eq. 6 first line only), and rpPIE (Eq. 6, both lines, enforcing known substrate reflectivity outside pad support). Evaluation uses cross-sectional phase profiles and a loss/error metric, with rpPIE achieving up to 2 orders of magnitude error reduction versus PIE.
