

# Revised Plan: Coherent Diffractive Imaging Phase Retrieval

## [Problem Formulation]

### Critical Reanalysis of the Actual Task

The **hard constraints** tell us everything:
- **Input**: `(81, 256, 256)` — 81 diffraction patterns, each 256×256 pixels
- **Output**: `(2, 600, 600)` — 2-channel (magnitude, phase) of the reconstructed object on a 600×600 grid

This is **NOT** a 2048×2048 detector problem. The actual simulation uses a much smaller, more practical grid. The 81 patterns correspond to a **9×9 ptychographic scan grid**.

### Revised Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n, \quad j = 1, \ldots, 81$$

where:
- $O(\mathbf{r}) \in \mathbb{C}^{600 \times 600}$ — complex object reflection function
- $P(\mathbf{r}) \in \mathbb{C}^{M \times M}$ — illumination probe (unknown size $M$ to be determined from data)
- $\mathbf{R}_j$ — scan positions on a 9×9 grid
- $I_j(\mathbf{k}) \in \mathbb{R}_{\geq 0}^{256 \times 256}$ — measured intensity at position $j$
- $\mathcal{F}$ — 2D DFT (Fraunhofer propagation)
- Output: `output[0,:,:] = |O(r)|`, `output[1,:,:] = arg(O(r))`

### Key Dimensional Constraints

The exit wave $\Psi_j(\mathbf{r}) = P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})$ is cropped/windowed to the probe's extent, then FFT'd to produce a 256×256 diffraction pattern. This means the **probe window** is 256×256 pixels (the exit wave region that gets Fourier-transformed is 256×256).

The object is 600×600. With a 9×9 scan grid and probe size 256, the scan step is:
$$\text{step} = \frac{600 - 256}{9 - 1} = \frac{344}{8} = 43 \text{ pixels}$$

Overlap: $1 - 43/256 \approx 83\%$ — consistent with ~75-83% overlap requirement.

### Inverse Problem

Given $\{I_j\}_{j=1}^{81}$ (intensity-only measurements), recover $O(\mathbf{r})$ by solving:
$$\min_{O, P} \sum_{j=1}^{81} \left\| \left|\mathcal{F}\{P(\mathbf{r}-\mathbf{R}_j) \cdot O(\mathbf{r})\}\right| - \sqrt{I_j(\mathbf{k})} \right\|^2$$

subject to physical constraints on $O$.

## [Proposed Strategy]

### Standard ePIE (Extended Ptychographic Iterative Engine) — **WITHOUT** hardcoded reflectivity prior

**Rationale for removing the reflectivity prior:**
1. The previous attempt failed catastrophically (error 12.4 = 1240%) because the TMM-computed reflectivities $R_p$, $R_{\text{Cu}}$ did not match the actual data generation model.
2. We do not know what forward model generated `input.npy` — the optical constants, layer thicknesses, and exact reflectivities are unknown.
3. Standard ePIE is a robust, well-proven algorithm that converges reliably with sufficient overlap (~83% here) **without** needing material-specific priors.
4. After basic ePIE converges, we can optionally apply a soft prior if needed.

### Algorithm: ePIE with blind probe refinement

- **Object update**: Standard ePIE multiplicative update
- **Probe update**: Simultaneous probe recovery (blind ptychography) — since we don't know the exact probe
- **Initialization**: Object = uniform amplitude 1.0, zero phase; Probe = circular aperture or flat
- **No hardcoded physical constants** — let the algorithm discover the object from data alone

### Step 2: Forward Operator Implementation

**2.1 Exit wave formation (for position $j$):**
```python
def form_exit_wave(O, P, pos_j):
    ry, rx = pos_j
    obj_patch = O[ry:ry+256, rx:rx+256]  # (256, 256) complex
    psi = P * obj_patch                    # (256, 256) complex
    return psi
```

**2.2 Forward propagation:**
```python
def forward(psi):
    return np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(psi)))
```

**2.3 Fourier modulus replacement:**
```python
def modulus_replace(Psi_hat, measured_amplitude):
    mag = np.abs(Psi_hat)
    # Careful division with epsilon to avoid NaN
    eps = 1e-12 * np.max(mag)
    Psi_corrected = measured_amplitude * Psi_hat / (mag + eps)
    return Psi_corrected
```

**2.4 Backward propagation:**
```python
def backward(Psi_hat):
    return np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(Psi_hat)))
```

**CRITICAL FFT CONVENTION NOTE:** We must try BOTH conventions:
- Convention A: `fft2` with `fftshift/ifftshift` (centered DC)
- Convention B: `fft2` without any shift (DC at corner)

The correct convention depends on how the input data was generated. We should detect this: if the diffraction patterns have their peak at the center, use Convention A; if at the corner, use Convention B.

```python
# Auto-detect: check where maximum intensity is
avg_pattern = np.mean(data, axis=0)
center = (128, 128)
corner = (0, 0)
if avg_pattern[128, 128] > avg_pattern[0, 0]:
    # DC is centered → data is already fftshifted
    centered_data = True
else:
    centered_data = False
```

### Step 3: ePIE Solver with Blind Probe Recovery

**3.1 Initialization:**
```python
# Object: uniform complex, magnitude 1.0, phase 0
O = np.ones((600, 600), dtype=np.complex128)

# Probe: initial guess — circular aperture with smooth edges
# or flat probe (all ones within circle)
Y, X = np.mgrid[-128:128, -128:128]
R = np.sqrt(X**2 + Y**2)
P = np.ones((256, 256), dtype=np.complex128)
# Smooth circular support (Gaussian-tapered disk)
probe_radius = 100  # pixels, will be refined
P = np.exp(-0.5 * np.maximum(R - probe_radius, 0)**2 / 10**2)
# Normalize probe
P = P / np.sqrt(np.sum(np.abs(P)**2)) * np.sqrt(np.mean(np.sum(data, axis=(1,2))))
```

**3.2 ePIE Main Loop:**

```
Parameters:
  α_obj = 1.0      (object update step size)
  α_probe = 1.0    (probe update step size) 
  reg = 1e-3       (regularization)
  N_iter = 300      (number of epochs)

For epoch = 0, ..., N_iter-1:
  
  # Randomize scan order
  order = np.random.permutation(81)
  
  For j in order:
    pos = positions[j]
    ry, rx = pos
    
    # 3.2a: Extract object patch
    obj_patch = O[ry:ry+256, rx:rx+256]
    
    # 3.2b: Form exit wave
    psi = P * obj_patch
    
    # 3.2c: Forward propagate
    Psi_hat = FFT2(psi)  # with appropriate shift convention
    
    # 3.2d: Fourier modulus constraint
    Psi_hat_c = amplitudes[j] * Psi_hat / (|Psi_hat| + eps)
    
    # 3.2e: Back-propagate
    psi_c = IFFT2(Psi_hat_c)
    
    # 3.2f: Compute update difference
    delta_psi = psi_c - psi
    
    # 3.2g: Object update (ePIE formula)
    P_max = max(|P|)
    O[ry:ry+256, rx:rx+256] += α_obj * conj(P) / (|P|^2_max + reg) * delta_psi
    
    # 3.2h: Probe update (blind ptychography)
    O_max = max(|obj_patch|)
    P += α_probe * conj(obj_patch) / (|obj_patch|^2_max + reg) * delta_psi
  
  # 3.2i: Compute Fourier error for monitoring
  E = (1/81) * Σ_j || |FFT2(P · O_patch_j)| - amplitudes[j] ||^2 / || amplitudes[j] ||^2
  
  # 3.2j: Optional — apply loose constraints every K iterations
  if epoch > 50 and epoch % 10 == 0:
      # Soft positivity on magnitude (physical reflectivity is bounded)
      mag = |O|
      mag = clip(mag, 0, 2.0)  # loose bound
      O = mag * exp(i * arg(O))
```

**3.3 Alternative ePIE update formulation (more standard):**

The standard ePIE update from Maiden & Rodenburg (2009):

$$O_{n+1}(\mathbf{r}) = O_n(\mathbf{r}) + \alpha_O \frac{P^*(\mathbf{r} - \mathbf{R}_j)}{\max_\mathbf{r}|P(\mathbf{r}-\mathbf{R}_j)|^2} \cdot \Delta\Psi(\mathbf{r}, \mathbf{R}_j)$$

$$P_{n+1}(\mathbf{r}) = P_n(\mathbf{r}) + \alpha_P \frac{O^*(\mathbf{r} + \mathbf{R}_j)}{\max_\mathbf{r}|O(\mathbf{r}+\mathbf{R}_j)|^2} \cdot \Delta\Psi(\mathbf{r}, \mathbf{R}_j)$$

where $\Delta\Psi = \Psi_c - \Psi$.

**3.4 Convergence acceleration — momentum (optional, if convergence is slow):**
```
After epoch 100, apply Nesterov-like momentum:
O_accel = O + 0.5 * (O - O_prev)
```

### Step 4: Post-processing and Output Formation

**4.1 Global phase removal:**
The reconstruction has an arbitrary global phase. Remove it by:
```python
# Set the mean phase of the object to zero (or some reference)
global_phase = np.angle(np.mean(O[np.abs(O) > 0.5 * np.max(np.abs(O))]))
O = O * np.exp(-1j * global_phase)
```

**4.2 Form output array:**
```python
output = np.zeros((2, 600, 600), dtype=np.float64)
output[0, :, :] = np.abs(O)       # magnitude channel
output[1, :, :] = np.angle(O)     # phase channel (in radians)
np.save("output.npy", output)
```

**4.3 Verify output shape:** `assert output.shape == (2, 600, 600)`

- Try: random phase probe (for speckle illumination)
- Scale probe amplitude to match total measured intensity:

**5.4 Scan step verification:**
If step=43 doesn't work well, try nearby values:
- step = 40 (paper's stated value): object would be $40 \times 8 + 256 = 576$ — doesn't reach 600
- step = 43: object = $43 \times 8 + 256 = 600$ ✓ **exact match**
- Therefore step = 43 is correct.

**5.5 Handle edge cases:**
- If `|Psi_hat|` has zeros: use `eps = 1e-12 * max(|Psi_hat|)` 
- If object diverges: clip magnitude to [0, 5] range
- If probe diverges: re-normalize probe every 10 epochs

### Step 6: Error Metrics for Monitoring

**6.1 Fourier error (primary convergence metric):**
$$E_F = \frac{1}{J}\sum_{j=1}^{J} \frac{\sum_\mathbf{k}\left(|\hat\Psi_j(\mathbf{k})| - \sqrt{I_j(\mathbf{k})}\right)^2}{\sum_\mathbf{k} I_j(\mathbf{k})}$$

**6.2 Object change (stationarity check):**
$$\Delta O = \frac{\|O^{(n+1)} - O^{(n)}\|_2}{\|O^{(n)}\|_2}$$

Stop when $\Delta O < 10^{-6}$ or after $N_{\text{iter}} = 300$ epochs.
