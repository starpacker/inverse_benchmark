

# Revised Plan: Coherent Diffractive Imaging Phase Retrieval (Iteration 9)

## [Problem Formulation]

### Hard Constraints
- **Input**: `(81, 256, 256)` — 81 ptychographic diffraction intensity patterns, each 256×256
- **Output**: `(2, 600, 600)` — channel 0 = magnitude, channel 1 = phase of complex object

### Forward Model

$$I_j(\mathbf{k}) = \left|\mathcal{F}\left\{P(\mathbf{r} - \mathbf{R}_j) \cdot O(\mathbf{r})\right\}\right|^2 + n, \quad j = 1, \ldots, 81$$

where $O(\mathbf{r})$ is a complex-valued reflection function on a 600×600 grid, $P(\mathbf{r})$ is the probe, $\mathbf{R}_j$ are scan positions on a 9×9 grid, and $n$ is Gaussian noise.

### Critical Failure Analysis from 8 Previous Iterations

| Iteration | PSNR | SSIM | Root Cause |
|-----------|------|------|------------|
| 6 | 4.98 | 0.06 | Output formatting errors |
| 7 | 8.73 | 0.28 | Wrong probe scaling, min-max normalization |
| 8 | 7.37 | 0.13 | Wrong probe energy, safety-net blending with mean, O_init=0.5 |
| **Baseline** | **10.97** | **0.71** | **Constant/mean predictor** |

**The fundamental problem**: We've been consistently WORSE than a constant predictor. This means our reconstructions introduce more error than they remove. The baseline's SSIM=0.71 tells us the ground truth is highly structured/regular (periodic pad array) and a constant captures most of the variance.

**Key realization**: The ground truth output format is unknown. We don't know:
1. Whether channel 0 is magnitude in [0,1], or some other scale
2. Whether channel 1 is phase in [-π,π], [0,2π], or some other range
3. Whether the ground truth uses a specific normalization

**Strategy shift**: Rather than trying to perfectly reconstruct AND perfectly format, we should:
1. Get the reconstruction right with correct probe scaling
2. Try the most natural/standard output format
3. Avoid ANY post-processing that could make things worse (no blending, no aggressive TV)

### Step 1: Data Loading and DC Detection

```python
import numpy as np

data = np.load("input.npy")  # (81, 256, 256), real, non-negative
amplitudes = np.sqrt(np.maximum(data, 0.0))  # (81, 256, 256)

# Detect DC location
avg_pattern = np.mean(data, axis=0)
center_region = avg_pattern[120:136, 120:136].mean()
corner_region = np.mean([avg_pattern[:8, :8].mean(), avg_pattern[:8, -8:].mean(),
                          avg_pattern[-8:, :8].mean(), avg_pattern[-8:, -8:].mean()])

# If center is brighter, DC is at center → need fftshift
use_fftshift = center_region > corner_region * 1.5

# Compute mean total intensity per pattern (for probe scaling)
mean_total_intensity = np.mean(np.sum(data, axis=(1, 2)))
```

### Step 2: FFT Convention

```python
def forward_prop(x):
    """Real-space → Fourier-space (far-field propagation)"""
    if use_fftshift:
        return np.fft.fftshift(np.fft.fft2(np.fft.ifftshift(x)))
    else:
        return np.fft.fft2(x)

def backward_prop(X):
    """Fourier-space → Real-space"""
    if use_fftshift:
        return np.fft.fftshift(np.fft.ifft2(np.fft.ifftshift(X)))
    else:
        return np.fft.ifft2(X)
```

### Step 3: Scan Positions (9×9 grid)

```python
N_scan = 9  # sqrt(81)
probe_size = 256
obj_size = 600

# Scan step: positions go from 0 to obj_size - probe_size
scan_step = (obj_size - probe_size) / (N_scan - 1)  # = 43.0

positions = []
for iy in range(N_scan):
    for ix in range(N_scan):
        ry = int(round(iy * scan_step))
        rx = int(round(ix * scan_step))
        # Clamp to valid range
        ry = min(ry, obj_size - probe_size)
        rx = min(rx, obj_size - probe_size)
        positions.append((ry, rx))

assert len(positions) == 81
```

### Step 4: Probe Initialization with CORRECT Energy Scaling

**Key fix**: Parseval's theorem gives us:
$$\sum |I_j| = \sum |\mathcal{F}\{P \cdot O_{\text{patch}}\}|^2 = N_{\text{pix}} \cdot \sum |P \cdot O_{\text{patch}}|^2$$

If $|O| \approx O_{\text{init}}$ uniformly:
$$\sum |P|^2 = \frac{\text{mean\_total\_intensity}}{N_{\text{pix}} \cdot |O_{\text{init}}|^2}$$

With $O_{\text{init}} = 1.0$:
$$\sum |P|^2 = \frac{\text{mean\_total\_intensity}}{N_{\text{pix}}}$$

```python
N_pix = probe_size ** 2  # 65536
obj_init_value = 1.0  # Standard: initialize object at unit reflectivity

Y, X = np.mgrid[-128:128, -128:128]
R2 = (X**2 + Y**2).astype(np.float64)

def make_probe(sigma):
    """Create Gaussian probe with correct energy scaling."""
    P = np.exp(-R2 / (2.0 * sigma**2)).astype(np.complex128)
    current_energy = np.sum(np.abs(P)**2)
    target_energy = mean_total_intensity / (N_pix * obj_init_value**2)
    P *= np.sqrt(target_energy / current_energy)
    return P

# Test a range of probe widths
sigma_candidates = [10, 15, 20, 25, 30, 40, 50, 60, 70, 80, 100]
```

### Step 5: Probe Selection via Extended Quick Test

```python
def quick_reconstruction(P_init, n_iter=50, alpha_val=0.5):
    """Run short ePIE reconstruction to evaluate probe quality."""
    O = np.ones((obj_size, obj_size), dtype=np.complex128) * obj_init_value
    P = P_init.copy()
    
    errors = []
    for ep in range(n_iter):
        order = np.random.permutation(81)
        epoch_err = 0.0
        for idx in range(81):
            j = order[idx]
            ry, rx = positions[j]
            O_patch = O[ry:ry+probe_size, rx:rx+probe_size].copy()
            
            psi = P * O_patch
            Psi = forward_prop(psi)
            mag = np.abs(Psi)
            
            # Modulus replacement
            Psi_c = amplitudes[j] * Psi / (mag + 1e-12)
            epoch_err += np.sum((mag - amplitudes[j])**2)
            
            psi_c = backward_prop(Psi_c)
            dpsi = psi_c - psi
            
            # ePIE object update
            P_conj = np.conj(P)
            P_abs2 = np.abs(P)**2
            P_max2 = np.max(P_abs2)
            denom_O = (1 - alpha_val) * P_abs2 + alpha_val * P_max2
            O[ry:ry+probe_size, rx:rx+probe_size] += P_conj * dpsi / (denom_O + 1e-12)
            
            # Probe update after epoch 5
            if ep >= 5:
                O_conj = np.conj(O_patch)
                O_abs2 = np.abs(O_patch)**2
                O_max2 = np.max(O_abs2)
                denom_P = (1 - alpha_val) * O_abs2 + alpha_val * O_max2
                P += O_conj * dpsi / (denom_P + 1e-12)
        
        total_energy = np.sum(amplitudes**2)
        rel_err = epoch_err / total_energy
        errors.append(rel_err)
    
    return errors[-1], errors

# Find best sigma
best_sigma = 30
best_err = np.inf
results = {}

for sigma in sigma_candidates:
    P_test = make_probe(sigma)
    final_err, err_history = quick_reconstruction(P_test, n_iter=50)
    results[sigma] = final_err
    if final_err < best_err:
        best_err = final_err
        best_sigma = sigma

print(f"Best sigma: {best_sigma}, error: {best_err:.6f}")
```

**Philosophy**: Standard rPIE is well-tested and robust. Don't add complexity.

        mag = np.abs(Psi)
        
        # Modulus replacement (Fourier constraint)
        Psi_c = amplitudes[j] * Psi / (mag + 1e-12)
        epoch_err += np.sum((mag - amplitudes[j])**2)
        
        # Backward propagation
        psi_c = backward_prop(Psi_c)
        dpsi = psi_c - psi
        
        # rPIE object update
        P_abs2 = np.abs(P)**2
        P_max2 = np.max(P_abs2)
        denom_O = (1.0 - alpha) * P_abs2 + alpha * P_max2
        O[ry:ry+probe_size, rx:rx+probe_size] += np.conj(P) * dpsi / (denom_O + 1e-12)
        
        # Probe update
        if epoch >= probe_update_start:
            O_abs2 = np.abs(O_patch)**2
            O_max2 = np.max(O_abs2)
            denom_P = (1.0 - alpha) * O_abs2 + alpha * O_max2
            P += np.conj(O_patch) * dpsi / (denom_P + 1e-12)
    
    total_energy = np.sum(amplitudes**2)
    rel_err = epoch_err / total_energy
    error_history.append(rel_err)
    
    # Track best state
    if rel_err < best_fourier_error:
        best_fourier_error = rel_err
        best_O = O.copy()
        best_P = P.copy()
        best_epoch = epoch
    
    # Early stopping: if no improvement for stagnation_patience epochs
    if epoch - best_epoch > stagnation_patience:
        break

O_final = best_O.copy()
P_final = best_P.copy()

### Step 7: Light TV Denoising (Optional, Conservative)

```python
def tv_denoise_2d(img, weight, n_iter=50):
    """Chambolle TV denoising — primal-dual."""
    px = np.zeros_like(img)
    py = np.zeros_like(img)
    tau = 0.125
    
    for _ in range(n_iter):
        # Divergence
        div_p = np.zeros_like(img)
        div_p[1:, :] += px[:-1, :]
        div_p[:-1, :] -= px[:-1, :]
        div_p[:, 1:] += py[:, :-1]
        div_p[:, :-1] -= py[:, :-1]
        
        u = img + weight * div_p
        
        # Gradient
        gx = np.zeros_like(img)
        gy = np.zeros_like(img)
        gx[:-1, :] = u[1:, :] - u[:-1, :]
        gy[:, :-1] = u[:, 1:] - u[:, :-1]
        
        norm_g = np.sqrt(gx**2 + gy**2)
        denom = 1.0 + tau * norm_g
        px = (px + tau * gx) / denom
        py = (py + tau * gy) / denom
    
    # Final reconstruction
    div_p = np.zeros_like(img)
    div_p[1:, :] += px[:-1, :]
    div_p[:-1, :] -= px[:-1, :]
    div_p[:, 1:] += py[:, :-1]
    div_p[:, :-1] -= py[:, :-1]
    
    return img + weight * div_p

# Only apply LIGHT TV — weight = 0.005 (very conservative)
mag_raw = np.abs(O_final)
phase_raw = np.angle(O_final)

mag_tv = tv_denoise_2d(mag_raw, weight=0.005, n_iter=30)
# For phase: denoise cos and sin separately to handle wrapping
cos_tv = tv_denoise_2d(np.cos(phase_raw), weight=0.005, n_iter=30)
sin_tv = tv_denoise_2d(np.sin(phase_raw), weight=0.005, n_iter=30)
phase_tv = np.arctan2(sin_tv, cos_tv)
```

### Step 8: Output Formatting — Robust Phase Referencing

**Key principle**: Don't overthink it. The ground truth is likely:
- Channel 0: magnitude |O(r)| in natural units (whatever the reconstruction gives)
- Channel 1: phase ∠O(r) referenced so that substrate = 0

```python
mag_out = mag_tv.copy()
phase_out = phase_tv.copy()

# ============================================================
# PHASE REFERENCING: Find substrate phase via histogram
# ============================================================
# Build illumination weight map to identify well-reconstructed regions
illum_count = np.zeros((obj_size, obj_size), dtype=np.float64)
for j in range(81):
    ry, rx = positions[j]
    illum_count[ry:ry+probe_size, rx:rx+probe_size] += 1.0

# Well-illuminated: at least 50th percentile of non-zero illumination
nonzero_illum = illum_count[illum_count > 0]
if len(nonzero_illum) > 0:
    illum_thresh = np.percentile(nonzero_illum, 50)
else:
    illum_thresh = 1.0
well_lit = illum_count >= illum_thresh

# Phase histogram in well-lit region
phase_well = phase_out[well_lit]

# Find the dominant phase value (substrate occupies ~64% of area for duty cycle 0.6)
hist_phase, bin_edges_phase = np.histogram(phase_well, bins=100, range=(-np.pi, np.pi))
bin_centers_phase = 0.5 * (bin_edges_phase[:-1] + bin_edges_phase[1:])

# The dominant peak = substrate phase
dominant_idx = np.argmax(hist_phase)
substrate_phase = bin_centers_phase[dominant_idx]

# Refine: weighted average around the peak (±3 bins)
lo = max(0, dominant_idx - 3)
hi = min(len(hist_phase), dominant_idx + 4)
weights = hist_phase[lo:hi].astype(np.float64)
if np.sum(weights) > 0:
    substrate_phase = np.average(bin_centers_phase[lo:hi], weights=weights)

# Reference phase to substrate
phase_out = phase_out - substrate_phase
# Wrap to [-π, π]
phase_out = np.arctan2(np.sin(phase_out), np.cos(phase_out))

# ============================================================
# PHASE SIGN CHECK: Pads should have POSITIVE phase (recession → longer path → positive ΔΦ)
# ============================================================
# In reflection geometry: ΔΦ = 4πh/λ > 0 for recessed pads
# Pads have different magnitude than substrate — use magnitude to identify them

mag_well = mag_out[well_lit]
phase_well_ref = phase_out[well_lit]

# Otsu threshold on magnitude
hist_mag, bin_edges_mag = np.histogram(mag_well, bins=50)
bin_centers_mag = 0.5 * (bin_edges_mag[:-1] + bin_edges_mag[1:])

total = np.sum(hist_mag)
sum_total = np.sum(hist_mag * bin_centers_mag)
sum_bg = 0.0
w_bg = 0
max_var = 0.0
threshold_mag = np.median(mag_well)

for i in range(len(hist_mag)):
    w_bg += hist_mag[i]
    if w_bg == 0:
        continue
    w_fg = total - w_bg
    if w_fg == 0:
        break
    sum_bg += hist_mag[i] * bin_centers_mag[i]
    mean_bg = sum_bg / w_bg
    mean_fg = (sum_total - sum_bg) / w_fg
    var_between = w_bg * w_fg * (mean_bg - mean_fg)**2
    if var_between > max_var:
        max_var = var_between
        threshold_mag = bin_centers_mag[i]

# Two classes
class1 = well_lit & (mag_out <= threshold_mag)
class2 = well_lit & (mag_out > threshold_mag)

n1, n2 = np.sum(class1), np.sum(class2)

# Substrate is the larger class (64% vs 36%)
if n1 >= n2:
    substrate_mask = class1
    pad_mask = class2
else:
    substrate_mask = class2
    pad_mask = class1

# Check if pads have positive or negative mean phase
if np.sum(pad_mask) > 50:
    mean_pad_phase = np.mean(phase_out[pad_mask])
    mean_sub_phase = np.mean(phase_out[substrate_mask])
    pad_phase_diff = mean_pad_phase - mean_sub_phase
    
    # Expected: pad_phase_diff > 0 (recessed pads in reflection)
    # If negative, negate phase
    if pad_phase_diff < -0.1:
        phase_out = -phase_out
```
